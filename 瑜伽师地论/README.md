# 瑜伽师地论

## 简介

《瑜伽师地论》，又称《十七地论》，是弥勒菩萨所说，无著菩萨记录，玄奘法师翻译的大乘佛教重要论典。本论共一百卷，分为五分：本地分、摄决择分、摄释分、摄异门分、摄事分。其中本地分是根本，详细阐述瑜伽师修行所经历的十七地，从凡夫到成佛的完整修行次第。

《瑜伽师地论》是唯识宗的根本论典，系统阐述了唯识学的理论体系，包括识的建立、三性三无性、转识成智等核心教义。本论不仅是理论著作，更是实践修行的指南，详细说明了从凡夫到成佛的完整修行道路。

## 主要教义

### 核心思想

- **唯识无境**：一切法唯识所现，外境非有
- **三性三无性**：遍计所执性、依他起性、圆成实性；相无性、生无性、胜义无性
- **转识成智**：转八识成四智，成就佛果
- **十七地**：从五识身相应地到无余依涅槃地，完整的修行次第

### 重要概念

- **五识**：眼识、耳识、鼻识、舌识、身识
- **意识**：第六识
- **末那识**：第七识，我执识
- **阿赖耶识**：第八识，藏识
- **三性**：遍计所执性、依他起性、圆成实性
- **十七地**：瑜伽师修行所经历的十七个阶段

1. **信解受持**：对论典生起信心，理解其义，并依教奉行
2. **系统学习**：按照十七地的次第，系统学习修行方法

4. **转识成智**：通过修行，转八识成四智
5. **求善知识**：求正知见人，依善知识修行
