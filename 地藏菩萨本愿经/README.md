# 地藏菩萨本愿经

## 简介

《地藏菩萨本愿经》，又称《地藏经》，是佛教重要经典之一。本经主要记载了地藏菩萨在释迦牟尼佛入灭后、弥勒佛未出世前，发愿救度一切众生的事迹。全经共分十三品，详细阐述了地藏菩萨的大愿、地狱的种种状况、因果报应的法则，以及供养地藏菩萨的功德利益。

### 正文（共十三品）

1. **忉利天宫神通品第一** - 佛陀在忉利天宫为母说法，十方诸佛菩萨集会，赞叹释迦牟尼佛，地藏菩萨因地发愿的故事

2. **分身集会品第二** - 十方地狱处分身地藏菩萨与被救众生前来见佛，佛陀嘱托地藏菩萨在弥勒佛出世前救度众生

3. **观众生业缘品第三** - 摩耶夫人询问众生业报，地藏菩萨简述五无间地狱的状况

4. **阎浮众生业感品第四** - 定自在王菩萨请问地藏菩萨往因，佛陀讲述国王及光目女的故事，说明因果报应法则

5. **地狱名号品第五** - 普贤菩萨询问地狱名号等情况，地藏菩萨详细回答以救拔苦难众生

6. **如来赞叹品第六** - 佛陀放光赞叹地藏菩萨，普广菩萨请问供养地藏菩萨的利益

7. **利益存亡品第七** - 地藏菩萨劝导众生断恶修善，说明超度亡灵的功德

8. **阎罗王众赞叹品第八** - 鬼王与阎罗天子前来忉利天宫，询问众生不依善道的原因

9. **称佛名号品第九** - 地藏菩萨为利益众生，演说听闻及念诵过去诸佛名号的功德

10. **校量布施功德缘品第十** - 地藏菩萨请问佛陀布施功德的大小，佛陀举例详细回答

11. **地神护法品第十一** - 佛陀说明供养地藏菩萨塑像、画像的十种广大利益

12. **见闻利益品第十二** - 佛陀放光称赞地藏菩萨，观世音菩萨请问不思议事，佛陀讲述供像持名等应获的功德

13. **嘱累人天品第十三** - 佛陀再次摩地藏菩萨顶，将众生托付给地藏菩萨，希望其救度一切受苦众生

## 主要教义

- **地藏菩萨的大愿**：众生度尽，方证菩提；地狱不空，誓不成佛
- **因果报应**：详细说明各种恶业所感的地狱果报
- **救度方法**：供养地藏菩萨、读诵本经、称念地藏名号等功德利益
- **超度亡灵**：说明如何为亡者修福，使其脱离恶道

## 功德利益

根据经文记载，读诵、供养、瞻礼地藏菩萨及本经，可获得以下利益：

- 消除罪业，不堕恶道
- 增长智慧，获得安乐
- 家宅平安，眷属和乐
- 所求遂意，衣食丰足
- 最终成就佛道
