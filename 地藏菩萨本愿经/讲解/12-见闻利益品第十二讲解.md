# 见闻利益品第十二讲解

## 章节概述

本品通过观世音菩萨的提问和佛陀的回答，详细说明了见闻地藏菩萨的各种利益。包括天人见闻的利益、临命终时见闻的利益、为亡者见闻的利益、发愿修行者的利益、读诵经典者的利益、生活困苦者的利益、出行者的利益等。说明了地藏菩萨的不可思议功德，以及如何通过见闻地藏菩萨来获得各种现世和来世的利益。

## 主要内容讲解

### 一、佛陀的放光

**经文要点：**
- 尔时世尊从顶门上放百千万亿大毫相光
- 所谓白毫相光、大白毫相光、瑞毫相光、大瑞毫相光、玉毫相光、大玉毫相光、紫毫相光、大紫毫相光、青毫相光、大青毫相光、碧毫相光、大碧毫相光、红毫相光、大红毫相光、绿毫相光、大绿毫相光、金毫相光、大金毫相光、庆云毫相光、大庆云毫相光、千轮毫光、大千轮毫光、宝轮毫光、大宝轮毫光、日轮毫光、大日轮毫光、月轮毫光、大月轮毫光、宫殿毫光、大宫殿毫光、海云毫光、大海云毫光
- 于顶门上放如是等毫相光已，出微妙音，告诸大众，天龙八部、人非人等：听吾今日于忉利天宫，称扬赞叹地藏菩萨于人天中，利益等事、不思议事、超圣因事、证十地事、毕竟不退阿耨多罗三藐三菩提事

**讲解：**
1. **放光的意义**：
   - 从顶门上放百千万亿大毫相光
   - 顶门是最高处，放光表示最高的赞叹和认可
   - 百千万亿表示数量极多，无法计算
   - 显示佛陀对地藏菩萨的赞叹和认可
   - 显示地藏菩萨的功德不可思议
   - 体现了地藏菩萨在诸佛菩萨中的重要地位

2. **各种毫相光**：
   - 白毫相光、瑞毫相光、玉毫相光、紫毫相光、青毫相光、碧毫相光、红毫相光、绿毫相光、金毫相光、庆云毫相光、千轮毫光、宝轮毫光、日轮毫光、月轮毫光、宫殿毫光、海云毫光等
   - 每种光都有大小两种，表示功德的大小
   - 不同的光代表不同的功德和利益
   - 说明地藏菩萨的功德遍及各个方面
   - 体现了地藏菩萨功德的全面性和不可思议

3. **微妙音的意义**：
   - 出微妙音，告诸大众
   - 微妙音表示佛法的殊胜和美妙
   - 告诸大众，包括天龙八部、人非人等
   - 说明这次法会的殊胜和重要
   - 体现了佛陀对地藏菩萨的重视

4. **称扬赞叹的内容**：
   - 利益等事：地藏菩萨给众生带来的各种利益
   - 不思议事：地藏菩萨的不可思议功德
   - 超圣因事：超越圣人的因行
   - 证十地事：证得十地果位
   - 毕竟不退阿耨多罗三藐三菩提事：毕竟不退转于无上正等正觉
   - 说明地藏菩萨的功德包括现世利益和究竟成佛
   - 体现了地藏菩萨功德的全面性和殊胜性

### 二、观世音菩萨的提问

**经文要点：**
- 会中有一菩萨摩诃萨，名观世音，从座而起，胡跪合掌白佛言：世尊，是地藏菩萨摩诃萨具大慈悲，怜愍罪苦众生，于千万亿世界，化千万亿身
- 所有功德及不思议威神之力。我闻世尊，与十方无量诸佛，异口同音，赞叹地藏菩萨云：正使过去现在未来诸佛，说其功德，犹不能尽
- 向者，又蒙世尊，普告大众：欲称扬地藏利益等事。唯愿世尊，为现在未来一切众生，称扬地藏不思议事，令天龙八部，瞻礼获福

**讲解：**
1. **观世音菩萨的身份**：
   - 观世音菩萨摩诃萨，大慈大悲的菩萨
   - 从座而起，胡跪合掌：表示恭敬和至诚
   - 说明观世音菩萨对地藏菩萨的恭敬和赞叹
   - 体现了观世音菩萨的慈悲心

2. **观世音菩萨的观察**：
   - 地藏菩萨具大慈悲，怜愍罪苦众生
   - 于千万亿世界，化千万亿身
   - 说明地藏菩萨的慈悲心遍及一切世界
   - 说明地藏菩萨的分身无数，度化众生
   - 体现了地藏菩萨的大慈悲和大愿力

3. **诸佛的赞叹**：
   - 与十方无量诸佛，异口同音，赞叹地藏菩萨
   - 异口同音：不同的佛，说同样的话
   - 说明十方诸佛都赞叹地藏菩萨
   - 正使过去现在未来诸佛，说其功德，犹不能尽
   - 说明即使所有佛都说地藏菩萨的功德，也说不尽
   - 体现了地藏菩萨功德的不可思议

4. **提问的目的**：
   - 为现在未来一切众生，称扬地藏不思议事
   - 令天龙八部，瞻礼获福
   - 说明观世音菩萨代众生提问，希望众生能够了解地藏菩萨的功德
   - 说明瞻礼地藏菩萨能够获得福报
   - 体现了观世音菩萨的慈悲心和对众生的关心

5. **佛陀的回应**：
   - 佛告观世音菩萨：汝于娑婆世界有大因缘
   - 说明观世音菩萨与娑婆世界有因缘
   - 若天若龙、若男若女、若神若鬼、乃至六道罪苦众生，闻汝名者、见汝形者、恋慕汝者、赞叹汝者。是诸众生，于无上道，必不退转。常生人天，具受妙乐。因果将熟，遇佛授记
   - 说明观世音菩萨的功德和利益
   - 体现了佛陀对观世音菩萨的认可

### 三、天人见闻的利益

**经文要点：**
- 佛告观世音菩萨：未来现在诸世界中，有天人受天福尽，有五衰相现，或有堕于恶道之者。如是天人，若男若女，当现相时，或见地藏菩萨形像、或闻地藏菩萨名，一瞻一礼。是诸天人，转增天福，受大快乐，永不堕三恶道报。何况见闻菩萨，以诸香华，衣服，饮食，宝贝，璎珞，布施供养，所获功德福利，无量无边

**讲解：**
1. **天人的情况**：
   - 有天人受天福尽，有五衰相现
   - 五衰相：衣服垢秽、头上华萎、腋下汗流、身体臭秽、不乐本座
   - 或有堕于恶道之者：有的会堕入恶道
   - 说明天人也有福尽的时候，也会堕入恶道
   - 体现了轮回的可怕和因果的公平性

2. **见闻的方法**：
   - 当现相时，或见地藏菩萨形像、或闻地藏菩萨名，一瞻一礼
   - 一瞻一礼：看一眼，拜一拜
   - 说明见闻地藏菩萨很容易，只要一瞻一礼
   - 体现了地藏菩萨的大慈悲，不要求众生做很多事

3. **见闻的利益**：
   - 是诸天人，转增天福，受大快乐，永不堕三恶道报
   - 转增天福：天福增加，不会减少
   - 受大快乐：享受大快乐
   - 永不堕三恶道报：永远不会堕入三恶道
   - 说明见闻地藏菩萨的利益很大，能够避免堕入恶道
   - 体现了地藏菩萨的威神力

4. **供养的利益**：
   - 何况见闻菩萨，以诸香华，衣服，饮食，宝贝，璎珞，布施供养
   - 所获功德福利，无量无边
   - 说明供养地藏菩萨的利益更大，无量无边
   - 体现了供养的重要性

### 四、临命终时见闻的利益

**经文要点：**
- 复次观世音，若未来现在诸世界中，六道众生临命终时，得闻地藏菩萨名，一声历耳根者。是诸众生，永不历三恶道苦。何况临命终时，父母眷属，将是命终人舍宅、财物、宝贝、衣服，塑画地藏形像。或使病人未终之时，眼耳见闻，知道眷属将舍宅、宝贝等为其自身塑画地藏菩萨形像。是人若是业报合受重病者，承斯功德，寻即除愈，寿命增益。是人若是业报命尽，应有一切罪障业障，合堕恶趣者。承斯功德，命终之后，即生人天，受胜妙乐，一切罪障，皆悉消灭

**讲解：**
1. **临命终时的重要性**：
   - 六道众生临命终时，得闻地藏菩萨名，一声历耳根者
   - 临命终时是关键时刻，决定来世的去向
   - 一声历耳根：只要听到一声地藏菩萨的名号
   - 说明见闻地藏菩萨很容易，只要听到一声名号
   - 体现了地藏菩萨的大慈悲

2. **一声闻名的利益**：
   - 是诸众生，永不历三恶道苦
   - 永不历三恶道苦：永远不会经历三恶道的苦
   - 说明即使只听到一声地藏菩萨的名号，也能避免堕入三恶道
   - 体现了地藏菩萨的威神力不可思议

3. **塑画形像的利益**：
   - 何况临命终时，父母眷属，将是命终人舍宅、财物、宝贝、衣服，塑画地藏形像
   - 用命终人的财物塑画地藏形像
   - 或使病人未终之时，眼耳见闻，知道眷属将舍宅、宝贝等为其自身塑画地藏菩萨形像
   - 让病人在未死之前，看到或听到眷属为他塑画地藏形像
   - 说明塑画地藏形像的利益很大

4. **重病者的利益**：
   - 是人若是业报合受重病者，承斯功德，寻即除愈，寿命增益
   - 如果是业报该受重病，承此功德，病立即痊愈，寿命增加
   - 说明塑画地藏形像能够消除病苦，延长寿命
   - 体现了地藏菩萨的威神力

5. **命尽者的利益**：
   - 是人若是业报命尽，应有一切罪障业障，合堕恶趣者。承斯功德，命终之后，即生人天，受胜妙乐，一切罪障，皆悉消灭
   - 如果业报该命尽，该堕恶趣，承此功德，命终后即生人天，受胜妙乐，一切罪障都消灭
   - 说明塑画地藏形像能够消除罪障，避免堕入恶道
   - 体现了地藏菩萨的威神力

### 五、为亡者见闻的利益

**经文要点：**
- 复次观世音菩萨，若未来世，有男子女人，或乳哺时、或三岁、五岁、十岁以下，亡失父母，乃及亡失兄弟姐妹，是人年既长大，思忆父母及诸眷属，不知落在何趣、生何世界、生何天中？是人若能塑画地藏菩萨形像，乃至闻名，一瞻一礼，一日至七日，莫退初心，闻名见形，瞻礼供养。是人眷属，假因业故，堕恶趣者，计当劫数，承斯男女，兄弟姊妹，塑画地藏形像，瞻礼功德，寻即解脱，生人天中，受胜妙乐。是人眷属，如有福力，已生人天，受胜妙乐者，即承斯功德，转增圣因，受无量乐
- 是人更能三七日中，一心瞻礼地藏形像，念其名字，满于万遍。当得菩萨现无边身，具告是人，眷属生界。或于梦中，菩萨现大神力，亲领是人，于诸世界，见诸眷属。更能每日念菩萨名千遍，至于千日。是人当得菩萨遣所在土地鬼神，终身卫护，现世衣食丰溢，无诸疾苦。乃至横事不入其门，何况及身。是人毕竟得菩萨摩顶授记

**讲解：**
1. **亡失亲人的情况**：
   - 或乳哺时、或三岁、五岁、十岁以下，亡失父母，乃及亡失兄弟姐妹
   - 说明从小失去亲人，不知道亲人在哪里
   - 是人年既长大，思忆父母及诸眷属，不知落在何趣、生何世界、生何天中
   - 长大后思念亲人，不知道他们在哪里
   - 体现了对亲人的思念和关心

2. **见闻的方法**：
   - 是人若能塑画地藏菩萨形像，乃至闻名，一瞻一礼，一日至七日，莫退初心，闻名见形，瞻礼供养
   - 塑画地藏形像，或闻名，一瞻一礼
   - 一日至七日：坚持一到七天
   - 莫退初心：不要退失初心
   - 说明见闻地藏菩萨的方法很简单，但要坚持

3. **亡者得解脱的利益**：
   - 是人眷属，假因业故，堕恶趣者，计当劫数，承斯男女，兄弟姊妹，塑画地藏形像，瞻礼功德，寻即解脱，生人天中，受胜妙乐
   - 如果亡者因业堕恶趣，该受劫数的苦，承此功德，立即解脱，生人天中，受胜妙乐
   - 说明塑画地藏形像能够救度亡者，让他们脱离恶道
   - 体现了地藏菩萨的威神力

4. **已生人天的利益**：
   - 是人眷属，如有福力，已生人天，受胜妙乐者，即承斯功德，转增圣因，受无量乐
   - 如果亡者已经生人天，受胜妙乐，承此功德，转增圣因，受无量乐
   - 说明塑画地藏形像能够增加亡者的福报
   - 体现了地藏菩萨的威神力

5. **三七日瞻礼的利益**：
   - 是人更能三七日中，一心瞻礼地藏形像，念其名字，满于万遍
   - 三七日：二十一天
   - 一心瞻礼：专心瞻礼
   - 念其名字，满于万遍：念地藏菩萨的名字，满一万遍
   - 当得菩萨现无边身，具告是人，眷属生界
   - 地藏菩萨会现无边身，告诉这个人，亡者生在哪里
   - 说明至诚瞻礼能够感通地藏菩萨

6. **梦中见眷属的利益**：
   - 或于梦中，菩萨现大神力，亲领是人，于诸世界，见诸眷属
   - 或者在梦中，地藏菩萨现大神力，亲自带领这个人，到各个世界，见到亡者
   - 说明至诚瞻礼能够在梦中见到亡者
   - 体现了地藏菩萨的威神力

7. **千日念名的利益**：
   - 更能每日念菩萨名千遍，至于千日
   - 每天念地藏菩萨的名字一千遍，坚持一千天
   - 是人当得菩萨遣所在土地鬼神，终身卫护，现世衣食丰溢，无诸疾苦。乃至横事不入其门，何况及身
   - 地藏菩萨会派遣土地鬼神，终身卫护这个人，现世衣食丰溢，没有疾病痛苦，横事不入其门
   - 说明坚持念名能够获得现世的利益和护持
   - 体现了地藏菩萨的威神力

8. **摩顶授记的利益**：
   - 是人毕竟得菩萨摩顶授记
   - 这个人最终会得到地藏菩萨的摩顶授记
   - 说明坚持见闻地藏菩萨，最终能够成佛
   - 体现了地藏菩萨的威神力

### 六、发愿修行者的利益

**经文要点：**
- 复次观世音菩萨，若未来世有善男子善女人，欲发广大慈心，救度一切众生者，欲修无上菩提者，欲出离三界者。是诸人等，见地藏形像，及闻名者，至心归依，或以香华、衣服、宝贝、饮食，供养瞻礼。是善男女等，所愿速成，永无障碍
- 复次观世音，若未来世，有善男子善女人，欲求现在未来百千万亿等愿，百千万亿等事。但当归依瞻礼、供养赞叹地藏菩萨形像。如是所愿所求，悉皆成就。复愿地藏菩萨具大慈悲，永拥护我。是人于睡梦中，即得菩萨摩顶授记

**讲解：**
1. **发愿的内容**：
   - 欲发广大慈心，救度一切众生者
   - 欲修无上菩提者
   - 欲出离三界者
   - 说明发愿的内容包括度众生、成佛、出离三界
   - 体现了大乘菩萨的愿力

2. **见闻的方法**：
   - 见地藏形像，及闻名者，至心归依
   - 或以香华、衣服、宝贝、饮食，供养瞻礼
   - 至心归依：至诚心归依
   - 供养瞻礼：供养和瞻礼
   - 说明见闻地藏菩萨的方法

3. **所愿速成的利益**：
   - 是善男女等，所愿速成，永无障碍
   - 所愿速成：愿望快速成就
   - 永无障碍：永远没有障碍
   - 说明见闻地藏菩萨能够帮助实现愿望
