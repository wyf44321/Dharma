# 校量布施功德缘品第十讲解

## 章节概述

本品通过地藏菩萨的提问和佛陀的回答，详细说明了布施的功德大小，以及如何通过布施来积累功德。特别强调了回向的重要性，以及如何通过回向来获得更大的功德。通过比较不同布施对象的功德大小，说明了慈悲心的重要性，以及回向法界与回向自家眷属的功德差别。

## 主要内容讲解

### 一、地藏菩萨的提问

**经文要点：**
- 地藏菩萨摩诃萨，承佛威神，从座而起，胡跪合掌白佛言：世尊，我观业道众生，校量布施，有轻有重，有一生受福，有十生受福，有百生千生受大福利者
- 是事云何，唯愿世尊为我说之

**讲解：**
1. **地藏菩萨的观察**：
   - 地藏菩萨观察业道众生，发现布施的功德有轻有重
   - 有一生受福：布施的功德只能一生受福
   - 有十生受福：布施的功德能十生受福
   - 有百生千生受大福利者：布施的功德能百生千生受大福利
   - 说明布施的功德不同，有大小差别
   - 体现了地藏菩萨对众生的关心，想了解布施功德的差别

2. **承佛威神**：
   - 地藏菩萨承佛威神，从座而起
   - 说明地藏菩萨的谦逊，不居功自傲
   - 体现了地藏菩萨对佛陀的恭敬和感恩

3. **胡跪合掌**：
   - 胡跪合掌：表示对佛陀的恭敬
   - 说明地藏菩萨的恭敬心，即使是大菩萨，也要恭敬佛陀
   - 体现了地藏菩萨的谦逊和恭敬

4. **提问的目的**：
   - 让众生知道布施的功德大小，从而知道如何布施
   - 让众生知道如何获得更大的功德，从而精进布施
   - 体现了地藏菩萨对众生的慈悲心，想帮助众生了解布施的功德

### 二、布施贫穷的功德

**经文要点：**
- 南阎浮提，有诸国王、宰辅大臣、大长者、大刹利、大婆罗门等，若遇最下贫穷，乃至癃残喑哑，聋痴无目，如是种种不完具者
- 是大国王等欲布施时，若能具大慈悲，下心含笑，亲手遍布施，或使人施，软言慰喻
- 是国王等所获福利，如布施百恒河沙佛功德之利
- 何以故？缘是国王等，于是最贫贱辈及不完具者，发大慈心，是故福利有如此报
- 百千生中，常得七宝具足，何况衣食受用

**讲解：**
1. **布施的对象**：
   - 最下贫穷：最贫穷的人，生活最困难的人
   - 乃至癃残喑哑，聋痴无目：乃至身体残疾、哑巴、聋子、痴呆、盲人
   - 如是种种不完具者：这样种种身体不完整的人
   - 说明布施的对象是最需要帮助的人
   - 这些人是社会最底层，最需要帮助
   - 体现了佛陀对弱势群体的关注和慈悲

2. **布施者的身份**：
   - 有诸国王、宰辅大臣、大长者、大刹利、大婆罗门等
   - 说明布施者是社会上层，有地位、有财富的人
   - 这些人有能力布施，也有责任帮助弱势群体
   - 体现了社会上层对弱势群体的责任

3. **布施的方法**：
   - 具大慈悲：要有大慈悲心，不是施舍，而是慈悲
   - 下心含笑：放下身份，含笑布施，不是高高在上
   - 亲手遍布施：亲手布施，不是让别人代劳
   - 或使人施：或者让人代施，但要亲自安排
   - 软言慰喻：用温和的语言安慰，不是命令或施舍
   - 说明布施要有慈悲心，要尊重受施者
   - 体现了布施的正确方法和心态

4. **布施的功德**：
   - 如布施百恒河沙佛功德之利：功德如布施百恒河沙数佛的功德
   - 百千生中，常得七宝具足：百千生中，常得七宝具足
   - 何况衣食受用：何况衣食受用
   - 说明布施贫穷的功德极大，不可思议
   - 体现了布施贫穷的殊胜功德

5. **功德的原因**：
   - 缘是国王等，于是最贫贱辈及不完具者，发大慈心
   - 是故福利有如此报：因为对最贫贱的人发大慈心，所以有如此大的功德
   - 说明功德的大小取决于慈悲心的大小
   - 对最需要帮助的人发慈悲心，功德最大
   - 体现了慈悲心的重要性

6. **慈悲心的意义**：
   - 大慈心：大慈悲心，不是小慈悲心
   - 对最贫贱的人发大慈心，说明没有分别心
   - 没有分别心，功德才大
   - 体现了无分别心的重要性

### 三、布施三宝的功德

**经文要点：**
- 复次地藏，若未来世，有诸国王，至婆罗门等，遇佛塔寺，或佛形像，乃至菩萨声闻辟支佛像，躬自营办供养布施
- 是国王等，当得三劫为帝释身，受胜妙乐
- 若能以此布施福利，回向法界，是大国王等，于十劫中，常为大梵天王

**讲解：**
1. **布施的对象**：
   - 佛塔寺：佛塔和寺院，是三宝的象征
   - 或佛形像：佛像，代表佛宝
   - 乃至菩萨声闻辟支佛像：菩萨像、声闻像、辟支佛像
   - 说明布施的对象是三宝，是佛法僧三宝
   - 布施三宝是供养三宝，功德很大
   - 体现了三宝的重要性

2. **布施的方法**：
   - 躬自营办供养布施：亲自办理供养布施
   - 躬自：亲自，不是让别人代劳
   - 营办：办理，准备供养的物品
   - 说明布施要亲自办理，要有恭敬心
   - 体现了布施的恭敬和至诚

3. **布施的功德**：
   - 当得三劫为帝释身，受胜妙乐：三劫中为帝释天，享受胜妙乐
   - 帝释天：忉利天主，是欲界第二层天的天主
   - 三劫：三个大劫，时间极长
   - 说明布施三宝的功德很大，能生天享受胜妙乐
   - 体现了布施三宝的殊胜功德

4. **回向的功德**：
   - 若能以此布施福利，回向法界：如果能将布施的福利回向法界
   - 是大国王等，于十劫中，常为大梵天王：这些国王等，在十劫中，常为大梵天王
   - 大梵天王：色界初禅天的天主，比帝释天更高
   - 十劫：十个大劫，比三劫更长
   - 说明回向法界的功德更大，比不回向的功德大得多
   - 体现了回向的重要性

5. **回向的意义**：
   - 回向法界：将功德回向给一切众生
   - 回向法界，心量广大，功德也大
   - 不回向，心量小，功德也小
   - 体现了心量对功德的影响

### 四、修补塔庙的功德

**经文要点：**
- 复次地藏，若未来世，有诸国王，至婆罗门等，遇先佛塔庙，或至经像，毁坏破落，乃能发心修补

- 是国王等，百千生中常为转轮王身
- 如是他人同布施者，百千生中常为小国王身

**讲解：**
1. **修补的对象**：
   - 先佛塔庙：过去佛的塔庙，已经毁坏破落
   - 或至经像：或者佛经和佛像，已经毁坏破落
   - 毁坏破落：已经损坏，需要修补
   - 说明修补的对象是已经毁坏的三宝之物
   - 修补三宝之物，功德很大
   - 体现了对三宝的护持和恭敬

2. **修补的方法**：
   - 或自营办：或者自己办理修补
   - 或劝他人：或者劝他人一起修补
   - 乃至百千人等布施结缘：乃至百千人等一起布施结缘
   - 说明修补可以自己办，也可以劝他人一起办
   - 劝他人一起修补，功德更大
   - 体现了共修的重要性

3. **修补的功德**：
   - 是国王等，百千生中常为转轮王身：这些国王等，百千生中常为转轮王
   - 转轮王：有七宝，能统一四天下的王，福德极大
   - 如是他人同布施者，百千生中常为小国王身：一起布施的人，百千生中常为小国王
   - 说明修补的功德很大，能得王位
   - 体现了修补三宝之物的殊胜功德

4. **回向的功德**：
   - 更能于塔庙前，发回向心：如果能在塔庙前发回向心
   - 如是国王乃及诸人，尽成佛道：这些国王和所有人，都能成佛
   - 以此果报无量无边：这个果报无量无边
   - 说明回向的功德更大，能成佛道
   - 体现了回向的重要性，回向能成佛

5. **修补的意义**：
   - 修补三宝之物，是护持三宝
   - 护持三宝，功德很大
   - 体现了护持三宝的重要性

### 十、修补经典的功德

**经文要点：**
- 复次地藏，若未来世中，有善男子善女人，遇佛塔寺，大乘经典。新者布施供养，瞻礼赞叹，恭敬合掌

- 如是等辈，三十生中常为诸小国王，檀越之人，常为轮王，还以善法教化诸小国王

**讲解：**
1. **布施新经典**：
   - 新者布施供养：新的经典，布施供养
   - 瞻礼赞叹：瞻礼赞叹，恭敬经典
   - 恭敬合掌：恭敬合掌，表示恭敬
   - 说明布施新经典的方法，要恭敬
   - 体现了对经典的恭敬和供养

2. **修补旧经典**：
   - 若遇故者，或毁坏者：如果遇到旧的，或者毁坏的
   - 修补营理：修补整理
   - 或独发心：或者独自发心
   - 或劝多人同共发心：或者劝多人一起发心
   - 说明修补旧经典的方法，可以独自办，也可以劝他人一起办

### 十一、回向的差别

**经文要点：**
- 复次地藏，未来世中，若有善男子善女人，于佛法中所种善根，或布施供养，或修补塔寺，或装理经典，乃至一毛一尘，一沙一渧
- 如是善事，但能回向法界，是人功德，百千生中受上妙乐
- 如但回向自家眷属，或自身利益，如是之果，即三生受乐，舍一得万报
- 是故地藏，布施因缘，其事如是

**讲解：**
1. **善根的范围**：
   - 于佛法中所种善根：在佛法中所种的善根
   - 或布施供养：或者布施供养
   - 或修补塔寺：或者修补塔寺
   - 或装理经典：或者整理经典
   - 乃至一毛一尘，一沙一渧：乃至一毛一尘，一沙一渧那么少
   - 说明善根的范围很广，即使很少也有功德
   - 体现了善根的殊胜功德

2. **回向法界**：
   - 但能回向法界：只要能回向法界
   - 是人功德，百千生中受上妙乐：这个人的功德，百千生中受上妙乐
   - 说明回向法界的功德很大，百千生中受上妙乐
   - 体现了回向法界的殊胜功德

3. **回向自家眷属**：
   - 如但回向自家眷属，或自身利益：如果只回向自家眷属，或者自身利益
   - 如是之果，即三生受乐，舍一得万报：这样的果报，就是三生受乐，舍一得万报
   - 说明回向自家眷属的功德较小，只有三生受乐
   - 体现了回向的差别

4. **回向的差别**：
   - 回向法界，百千生中受上妙乐：回向法界，百千生中受上妙乐
   - 回向自家眷属，三生受乐：回向自家眷属，三生受乐
   - 说明回向的差别很大，回向法界功德大得多
   - 体现了回向法界的重要性

5. **回向的意义**：
   - 回向法界，心量广大，功德也大
   - 回向自家眷属，心量小，功德也小
   - 体现了心量对功德的影响

6. **布施因缘**：
   - 是故地藏，布施因缘，其事如是：所以地藏，布施的因缘，就是这样
   - 说明布施的因缘和果报，就是这样
   - 体现了因果的对应性

## 教义要点

1. **布施的功德**：

   - 布施的对象不同，功德不同：布施贫穷功德最大，如布施百恒河沙佛功德
   - 布施的方法不同，功德不同：具大慈悲、下心含笑、亲手布施，功德更大

   - 布施三宝，功德很大：三劫为帝释身，回向法界十劫为大梵天王

   - 所受福利，不可为喻：所受的福利，无法比喻
   - 说明即使善根很少，功德也很大
