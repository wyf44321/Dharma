# 观众生业缘品第三讲解

## 章节概述

本品通过摩耶夫人与地藏菩萨的问答，详细说明了阎浮提众生造业差别及所受报应，特别是五无间地狱的状况。通过描述地狱的恐怖景象，警示众生要断恶修善，避免堕入恶道。

## 主要内容讲解

### 一、摩耶夫人的提问

**经文要点：**
- 佛母摩耶夫人恭敬合掌问地藏菩萨
- 阎浮众生，造业差别，所受报应，其事云何？

**讲解：**
1. **摩耶夫人**：
   - 佛陀的生母
   - 在忉利天宫
   - 关心众生，代众生提问

2. **提问的意义**：
   - 了解众生造业的差别
   - 了解所受报应的情况
   - 帮助众生认识因果

### 二、地藏菩萨的回答

**经文要点：**
- 千万世界，乃及国土，或有地狱、或无地狱
- 或有女人、或无女人
- 或有佛法、或无佛法
- 非但地狱罪报一等

**讲解：**
1. **世界的差别**：
   - 不同世界有不同的情况
   - 有的有地狱，有的没有
   - 说明业报的复杂性

2. **摩耶夫人的追问**：
   - 且愿闻于阎浮罪报所感恶趣
   - 专门问阎浮提的罪报

### 三、五无间罪

**经文要点：**
地藏菩萨说明五种会堕无间地狱的罪业：

1. **不孝父母，或至杀害**
2. **出佛身血，毁谤三宝，不敬尊经**
3. **侵损常住，玷污僧尼，或伽蓝内恣行淫欲，或杀或害**
4. **伪作沙门，心非沙门，破用常住，欺诳白衣，违背戒律**
5. **偷窃常住财物谷米，饮食衣服，乃至一物不与取者**

**讲解：**
1. **不孝父母**：
   - 不孝顺父母是重罪
   - 杀害父母更是五逆重罪
   - 说明孝道的重要性

2. **出佛身血，毁谤三宝**：
   - 伤害佛的身体
   - 毁谤佛、法、僧三宝
   - 不敬尊经
   - 这些都是极重的罪业

3. **侵损常住**：
   - 侵损寺院的财物
   - 玷污僧尼
   - 在寺院内行淫欲、杀生等
   - 这些都是对三宝的严重伤害

4. **伪作沙门**：
   - 假扮出家人
   - 心不是出家人
   - 破用常住，欺诳白衣
   - 违背戒律
   - 这是破坏佛教形象的重罪

5. **偷窃常住**：
   - 偷窃寺院的财物
   - 即使是一物不与取也是重罪
   - 说明对三宝财物的尊重

### 四、无间地狱的详细描述

**经文要点：**
- 无间地狱在大铁围山之内
- 其狱城周匝八万余里，其城纯铁，高一万里
- 城上火聚，少有空缺
- 其狱周匝万八千里，狱墙高一千里
- 上火彻下，下火彻上
- 铁蛇铁狗，吐火驰逐

**讲解：**
1. **地狱的位置**：
   - 在大铁围山之内
   - 大地狱有一十八所
   - 次有五百，名号各别
   - 次有千百，名字亦别

2. **无间地狱的规模**：
   - 狱城周匝八万余里
   - 其城纯铁，高一万里
   - 城上火聚，少有空缺
   - 说明地狱的恐怖

3. **地狱的景象**：
   - 上火彻下，下火彻上：上下都是火
   - 铁蛇铁狗，吐火驰逐：各种刑具
   - 狱墙之上，东西而走：无处可逃

### 五、地狱中的痛苦

**经文要点：**
- 狱中有床，遍满万里
- 一人受罪，自见其身遍卧满床
- 千万人受罪，亦各自见身满床上
- 众业所感获报如是

**讲解：**
1. **床的比喻**：
   - 狱中有床，遍满万里
   - 一人受罪，自见其身遍卧满床
   - 说明业力的不可思议
   - 每个人都会感受到完整的痛苦

2. **各种刑罚**：
   - 千百夜叉及以恶鬼，口牙如剑，眼如电光
   - 手复铜爪，拖拽罪人
   - 复有夜叉执大铁戟，中罪人身
   - 复有铁鹰啖罪人目
   - 复有铁蛇绞罪人颈
   - 百肢节内，悉下长钉
   - 拔舌耕犁，抽肠锉斩
   - 烊铜灌口，热铁缠身

3. **万死千生**：
   - 万死千生，业感如是
   - 动经亿劫，求出无期
   - 说明地狱之苦的持续

### 六、五事业感

**经文要点：**
地藏菩萨说明为什么叫"无间"地狱，有五种原因：

1. **日夜受罪，以至劫数，无时间绝**
2. **一人亦满，多人亦满**
3. **罪器叉棒，鹰蛇狼犬，更无间断**
4. **不问男子女人，羌胡夷狄，老幼贵贱，或龙或神，或天或鬼，罪行业感，悉同受之**
5. **若堕此狱，从初入时，至百千劫，一日一夜，万死万生，求一念间暂住不得**

**讲解：**
1. **无时间绝**：
   - 日夜受罪，无时间绝
   - 说明痛苦不间断

2. **一人亦满，多人亦满**：
   - 无论多少人，每个人都会感受到完整的痛苦
   - 说明业力的不可思议

3. **更无间断**：
   - 各种刑具，更无间断
   - 说明痛苦持续不断

4. **悉同受之**：
   - 无论什么人，造了同样的罪业，都会受同样的报应
   - 说明因果的公平

5. **求一念间暂住不得**：
   - 连一刹那的休息都没有
   - 说明痛苦至极

## 教义要点

1. **因果报应**：
   - 造什么业，受什么报
   - 因果丝毫不爽
   - 五无间罪必堕无间地狱

2. **地狱的真实性**：
   - 地狱确实存在
   - 不是吓唬人的
   - 是业力所感

3. **业力的不可思议**：
   - 一人亦满，多人亦满
   - 说明业力的力量
   - 每个人都会感受到完整的痛苦

4. **无间的含义**：
   - 无时间绝
   - 无空间绝
   - 无间断
   - 无差别
   - 无暂停

## 修行启示

1. **断恶修善**：
   - 要避免造五无间罪
   - 要孝顺父母
   - 要恭敬三宝
   - 要持戒修行

2. **深信因果**：
   - 要深信因果报应
   - 知道造恶业的果报
   - 从而断恶修善

3. **敬畏之心**：
   - 要对因果有敬畏之心
   - 不要以为造恶业没有报应
   - 要时刻警惕

4. **精进修行**：
   - 要精进修行
   - 避免堕入恶道
   - 要修善积德

5. **忏悔罪业**：
   - 如果造了恶业，要忏悔
   - 要改过自新
   - 要修善补过

## 现代意义

1. **因果教育**：
   - 要教育人们相信因果
   - 知道善恶有报
   - 从而规范行为

2. **孝道教育**：
   - 要重视孝道
   - 孝顺父母是做人的根本
   - 不孝是重罪

3. **敬畏之心**：
   - 要有敬畏之心
   - 不要为所欲为
   - 要知道行为的后果

4. **道德规范**：
   - 要遵守道德规范
   - 不要伤害他人
   - 要尊重三宝

5. **警示作用**：
   - 地狱的描述有警示作用
   - 让人们知道造恶业的后果
   - 从而断恶修善
