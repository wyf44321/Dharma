# 观众生业缘品第三讲解

## 章节概述

本品通过摩耶夫人与地藏菩萨的问答，详细说明了阎浮提众生造业差别及所受报应，特别是五无间地狱的状况。通过描述地狱的恐怖景象，警示众生要断恶修善，避免堕入恶道。

## 主要内容讲解

### 一、摩耶夫人的提问

**经文要点：**
- 尔时佛母摩耶夫人，恭敬合掌问地藏菩萨言：圣者，阎浮众生，造业差别，所受报应，其事云何？

**讲解：**
1. **摩耶夫人**：
   - 佛陀的生母，在忉利天宫
   - 摩耶夫人是佛陀的生母，生下佛陀七天后就去世了，往生到忉利天
   - 在忉利天宫，关心众生，代众生提问
   - 说明摩耶夫人虽然在天上，但仍然关心娑婆世界的众生
   - 体现了摩耶夫人的慈悲心

2. **恭敬合掌**：
   - 恭敬合掌问地藏菩萨
   - 说明摩耶夫人对地藏菩萨的恭敬
   - 说明即使是在天宫，也要恭敬地藏菩萨
   - 体现了摩耶夫人的谦逊和恭敬

3. **提问的内容**：
   - 阎浮众生，造业差别，所受报应，其事云何？
   - 阎浮提是娑婆世界的四大部洲之一，这里指娑婆世界的众生
   - 造业差别：众生造的业不同，有善业、恶业等
   - 所受报应：造什么业，受什么报
   - 其事云何：具体情况是怎样的
   - 说明摩耶夫人想了解众生造业的差别和所受报应的情况

4. **提问的意义**：
   - 了解众生造业的差别，帮助众生认识因果
   - 了解所受报应的情况，警示众生要断恶修善
   - 帮助众生认识因果，知道造恶业的果报
   - 体现了摩耶夫人对众生的关心和慈悲

### 二、地藏菩萨的回答

**经文要点：**
- 地藏答言：千万世界，乃及国土，或有地狱、或无地狱；或有女人、或无女人；或有佛法、或无佛法；乃至声闻、辟支佛，亦复如是，非但地狱罪报一等
- 摩耶夫人重白菩萨：且愿闻于阎浮罪报所感恶趣
- 地藏答言：圣母，唯愿听受，我粗说之
- 佛母白言：愿圣者说

**讲解：**
1. **世界的差别**：
   - 千万世界，乃及国土，或有地狱、或无地狱
   - 不同世界有不同的情况，有的有地狱，有的没有
   - 说明业报的复杂性，不是所有世界都有地狱
   - 说明地狱是业力所感，不是所有世界都有

2. **其他差别**：
   - 或有女人、或无女人：有的世界有女人，有的没有
   - 或有佛法、或无佛法：有的世界有佛法，有的没有
   - 乃至声闻、辟支佛，亦复如是：声闻、辟支佛的情况也是如此
   - 说明不同世界的情况不同，不是统一的

3. **非但地狱罪报一等**：
   - 非但地狱罪报一等：不只是地狱的罪报是一样的
   - 说明不同世界的罪报不同
   - 说明业报的复杂性，不是所有世界的罪报都一样

4. **摩耶夫人的追问**：
   - 摩耶夫人重白菩萨：且愿闻于阎浮罪报所感恶趣
   - 专门问阎浮提的罪报，想了解娑婆世界众生的罪报
   - 说明摩耶夫人关心娑婆世界的众生
   - 体现了摩耶夫人的慈悲心

5. **地藏菩萨的回应**：
   - 地藏答言：圣母，唯愿听受，我粗说之
   - 地藏菩萨答应说明，但只是粗略地说
   - 说明地狱的罪报很多，无法完全说尽
   - 佛母白言：愿圣者说，摩耶夫人请地藏菩萨说明

### 三、五无间罪

**经文要点：**
- 尔时地藏菩萨白圣母言：南阎浮提，罪报名号如是
- 若有众生不孝父母，或至杀害，当堕无间地狱，千万亿劫求出无期
- 若有众生出佛身血，毁谤三宝，不敬尊经，亦当堕于无间地狱，千万亿劫求出无期
- 若有众生侵损常住，玷污僧尼，或伽蓝内恣行淫欲，或杀或害，如是等辈，当堕无间地狱，千万亿劫求出无期
- 若有众生，伪作沙门，心非沙门，破用常住，欺诳白衣，违背戒律，种种造恶，如是等辈，当堕无间地狱，千万亿劫求出无期
- 若有众生，偷窃常住财物谷米，饮食衣服，乃至一物不与取者，当堕无间地狱，千万亿劫求出无期
- 地藏白言：圣母，若有众生，作如是罪，当堕五无间地狱，求暂停苦一念不得

**讲解：**
1. **不孝父母，或至杀害**：
   - 不孝顺父母是重罪，杀害父母更是五逆重罪
   - 当堕无间地狱，千万亿劫求出无期
   - 说明孝道的重要性，不孝是极重的罪业
   - 杀害父母是五逆重罪之一，必堕无间地狱
   - 说明父母恩重如山，不孝是极大的罪过
   - 体现了孝道在佛法中的重要性

2. **出佛身血，毁谤三宝，不敬尊经**：
   - 出佛身血：伤害佛的身体，这是极重的罪业
   - 毁谤三宝：毁谤佛、法、僧三宝，破坏佛教
   - 不敬尊经：不恭敬佛经，轻慢佛法
   - 亦当堕于无间地狱，千万亿劫求出无期
   - 说明对三宝的伤害是极重的罪业
   - 体现了三宝的重要性，伤害三宝必受重报

3. **侵损常住，玷污僧尼**：
   - 侵损常住：侵损寺院的财物，这是对三宝的伤害
   - 玷污僧尼：玷污出家人，破坏僧团的清净
   - 或伽蓝内恣行淫欲，或杀或害：在寺院内行淫欲、杀生等
   - 如是等辈，当堕无间地狱，千万亿劫求出无期
   - 说明对三宝的伤害是极重的罪业
   - 体现了对三宝的尊重和保护的重要性

4. **伪作沙门，心非沙门**：
   - 伪作沙门：假扮出家人，欺骗众生
   - 心非沙门：心不是出家人，没有出家人的心
   - 破用常住：破坏和盗用寺院的财物
   - 欺诳白衣：欺骗在家信众
   - 违背戒律：违背出家人的戒律
   - 种种造恶：做各种恶事
   - 如是等辈，当堕无间地狱，千万亿劫求出无期
   - 说明假扮出家人是极重的罪业，破坏佛教形象
   - 体现了对出家人的尊重和保护的重要性

5. **偷窃常住财物**：
   - 偷窃常住财物谷米，饮食衣服，乃至一物不与取者
   - 即使是一物不与取也是重罪
   - 当堕无间地狱，千万亿劫求出无期
   - 说明对三宝财物的尊重，偷窃三宝财物是极重的罪业
   - 体现了对三宝财物的保护的重要性

6. **五无间地狱的果报**：
   - 地藏白言：圣母，若有众生，作如是罪，当堕五无间地狱，求暂停苦一念不得
   - 说明造这五种罪业，必堕无间地狱
   - 千万亿劫求出无期：时间极长，无法计算
   - 求暂停苦一念不得：连一刹那的休息都没有
   - 说明这五种罪业的果报极其严重
   - 体现了因果报应的严厉性

### 四、无间地狱的详细描述

**经文要点：**
- 摩耶夫人重白地藏菩萨言：云何名为无间地狱？
- 地藏白言：圣母，诸有地狱在大铁围山之内，其大地狱有一十八所，次有五百，名号各别，次有千百，名字亦别
- 无间狱者，其狱城周匝八万余里，其城纯铁，高一万里，城上火聚，少有空缺
- 其狱城中，诸狱相连，名号各别。独有一狱，名曰无间
- 其狱周匝万八千里，狱墙高一千里，悉是铁围，上火彻下，下火彻上
- 铁蛇铁狗，吐火驰逐，狱墙之上，东西而走

**讲解：**
1. **摩耶夫人的追问**：
   - 摩耶夫人重白地藏菩萨言：云何名为无间地狱？
   - 摩耶夫人想了解什么是无间地狱
   - 说明摩耶夫人对地狱的关心，想了解地狱的情况
   - 体现了摩耶夫人对众生的慈悲心

2. **地狱的位置**：
   - 诸有地狱在大铁围山之内
   - 大铁围山是围绕世界的山，地狱在大铁围山之内
   - 其大地狱有一十八所：有十八个大地狱
   - 次有五百，名号各别：其次有五百个地狱，名称不同
   - 次有千百，名字亦别：再次有千百个地狱，名称也不同
   - 说明地狱的种类很多，不是只有一个

3. **无间地狱的规模**：
   - 无间狱者，其狱城周匝八万余里：无间地狱的城周长八万多里
   - 其城纯铁，高一万里：城是纯铁的，高一万里
   - 城上火聚，少有空缺：城上都是火，很少有空缺的地方
   - 说明无间地狱的规模极大，极其恐怖
   - 体现了无间地狱的恐怖和严重

4. **无间地狱的特殊性**：
   - 其狱城中，诸狱相连，名号各别。独有一狱，名曰无间
   - 说明在众多地狱中，无间地狱是特殊的
   - 无间地狱是最苦的地狱，所以特别说明
   - 体现了无间地狱的严重性

5. **无间地狱的详细规模**：
   - 其狱周匝万八千里：无间地狱的周长一万八千里
   - 狱墙高一千里：狱墙高一千里
   - 悉是铁围：都是铁围成的
   - 说明无间地狱的规模极大，极其恐怖

6. **地狱的景象**：
   - 上火彻下，下火彻上：上下都是火，没有空隙
   - 铁蛇铁狗，吐火驰逐：铁蛇铁狗吐着火追赶罪人
   - 狱墙之上，东西而走：在狱墙上东西奔跑，无处可逃
   - 说明地狱的景象极其恐怖，罪人无处可逃
   - 体现了地狱的恐怖和罪人的痛苦

### 五、地狱中的痛苦

**经文要点：**
- 狱中有床，遍满万里。一人受罪，自见其身遍卧满床。千万人受罪，亦各自见身满床上。众业所感获报如是
- 又诸罪人，备受众苦。千百夜叉及以恶鬼，口牙如剑，眼如电光，手复铜爪，拖拽罪人
- 复有夜叉执大铁戟，中罪人身，或中口鼻，或中腹背。抛空翻接或置床上
- 复有铁鹰啖罪人目。复有铁蛇绞罪人颈。百肢节内，悉下长钉，拔舌耕犁，抽肠锉斩，烊铜灌口，热铁缠身
- 万死千生，业感如是。动经亿劫，求出无期
- 此界坏时，寄生他界，他界次坏，转寄他方；他方坏时，展转相寄。此界成后，还复而来。无间罪报，其事如是

**讲解：**
1. **床的比喻**：
   - 狱中有床，遍满万里：地狱中有床，遍满万里
   - 一人受罪，自见其身遍卧满床：一个人受罪，自己看到自己的身体遍卧满床
   - 千万人受罪，亦各自见身满床上：千万人受罪，每个人也各自看到自己的身体遍卧满床
   - 众业所感获报如是：这是业力所感的报应
   - 说明业力的不可思议，每个人都会感受到完整的痛苦
   - 体现了业力的强大和不可思议

2. **夜叉恶鬼的刑罚**：
   - 千百夜叉及以恶鬼，口牙如剑，眼如电光：千百个夜叉和恶鬼，牙齿像剑，眼睛像电光
   - 手复铜爪，拖拽罪人：手是铜爪，拖拽罪人
   - 说明夜叉恶鬼的恐怖，对罪人的折磨
   - 体现了地狱的恐怖和罪人的痛苦

3. **铁戟的刑罚**：
   - 复有夜叉执大铁戟，中罪人身，或中口鼻，或中腹背
   - 抛空翻接或置床上：把罪人抛到空中，翻接或放在床上
   - 说明刑罚的残酷，对罪人的折磨
   - 体现了地狱的恐怖和罪人的痛苦

4. **各种刑具的刑罚**：
   - 复有铁鹰啖罪人目：铁鹰啄食罪人的眼睛
   - 复有铁蛇绞罪人颈：铁蛇绞罪人的脖子
   - 百肢节内，悉下长钉：在身体的各个关节都钉下长钉
   - 拔舌耕犁：拔掉舌头，用犁耕
   - 抽肠锉斩：抽出肠子，用锉刀斩
   - 烊铜灌口：把熔化的铜灌入口中
   - 热铁缠身：用热铁缠身
   - 说明各种刑罚的残酷，对罪人的折磨
   - 体现了地狱的恐怖和罪人的痛苦

5. **万死千生**：
   - 万死千生，业感如是：万死千生，这是业力所感的
   - 动经亿劫，求出无期：经过亿劫，无法求出
   - 说明地狱之苦的持续，时间极长
   - 体现了地狱之苦的严重性和持续性

6. **地狱的持续性**：
   - 此界坏时，寄生他界：这个世界坏时，寄生到其他世界
   - 他界次坏，转寄他方：其他世界坏时，转寄到其他地方
   - 他方坏时，展转相寄：其他地方坏时，展转相寄
   - 此界成后，还复而来：这个世界形成后，又回来
   - 无间罪报，其事如是：无间地狱的罪报就是这样
   - 说明地狱之苦是持续的，即使世界坏了，也会转寄到其他世界
   - 体现了地狱之苦的严重性和持续性

### 六、五事业感

**经文要点：**
- 又五事业感，故称无间。何等为五？
- 一者、日夜受罪，以至劫数，无时间绝，故称无间
- 二者、一人亦满，多人亦满，故称无间
- 三者、罪器叉棒，鹰蛇狼犬，碓磨锯凿，锉斫镬汤，铁网铁绳，铁驴铁马，生革络首，热铁浇身，饥吞铁丸，渴饮铁汁，从年竟劫，数那由他，苦楚相连，更无间断，故称无间
- 四者、不问男子女人，羌胡夷狄，老幼贵贱，或龙或神，或天或鬼，罪行业感，悉同受之，故称无间
- 五者、若堕此狱，从初入时，至百千劫，一日一夜，万死万生，求一念间暂住不得，除非业尽，方得受生，以此连绵，故称无间
- 地藏菩萨白圣母言：无间地狱，粗说如是。若广说地狱罪器等名，及诸苦事，一劫之中，求说不尽

**讲解：**
1. **第一无间：无时间绝**：
   - 日夜受罪，以至劫数，无时间绝，故称无间
   - 日夜受罪，没有时间停止
   - 以至劫数：经过劫数的时间
   - 无时间绝：没有时间停止
   - 说明痛苦不间断，没有休息的时间
   - 体现了无间地狱的恐怖和严重

2. **第二无间：一人亦满，多人亦满**：
   - 一人亦满，多人亦满，故称无间
   - 无论是一个人还是多人，每个人都会感受到完整的痛苦
   - 说明业力的不可思议，每个人都会感受到完整的痛苦
   - 体现了业力的强大和不可思议

3. **第三无间：更无间断**：
   - 罪器叉棒，鹰蛇狼犬，碓磨锯凿，锉斫镬汤，铁网铁绳，铁驴铁马，生革络首，热铁浇身，饥吞铁丸，渴饮铁汁
   - 从年竟劫，数那由他，苦楚相连，更无间断，故称无间
   - 说明各种刑具，更无间断
   - 从年竟劫：从一年到一劫
   - 数那由他：数量极多，无法计算
   - 苦楚相连，更无间断：痛苦相连，更无间断
   - 说明痛苦持续不断，没有停止的时候
   - 体现了无间地狱的恐怖和严重

4. **第四无间：悉同受之**：
   - 不问男子女人，羌胡夷狄，老幼贵贱，或龙或神，或天或鬼，罪行业感，悉同受之，故称无间
   - 无论是什么人，造了同样的罪业，都会受同样的报应
   - 不问：不管
   - 男子女人：男人女人
   - 羌胡夷狄：各种民族
   - 老幼贵贱：老人小孩，贵贱
   - 或龙或神，或天或鬼：龙、神、天、鬼等
   - 罪行业感，悉同受之：造了同样的罪业，都受同样的报应
   - 说明因果的公平，无论是什么人，造了同样的罪业，都受同样的报应
   - 体现了因果的公平性和严厉性

5. **第五无间：求一念间暂住不得**：
   - 若堕此狱，从初入时，至百千劫，一日一夜，万死万生，求一念间暂住不得，除非业尽，方得受生，以此连绵，故称无间
   - 从初入时，至百千劫：从刚进入时，到百千劫
   - 一日一夜，万死万生：一天一夜，万死万生
   - 求一念间暂住不得：连一刹那的休息都没有
   - 除非业尽，方得受生：除非业力尽了，才能受生
   - 以此连绵，故称无间：这样连绵不断，所以叫无间
   - 说明痛苦至极，连一刹那的休息都没有
   - 体现了无间地狱的恐怖和严重

6. **地狱的说不尽**：
   - 地藏菩萨白圣母言：无间地狱，粗说如是。若广说地狱罪器等名，及诸苦事，一劫之中，求说不尽

   - 若广说：如果详细说
   - 一劫之中，求说不尽：一劫的时间也说不尽

   - 体现了地狱的恐怖和严重
