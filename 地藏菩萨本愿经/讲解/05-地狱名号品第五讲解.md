# 地狱名号品第五讲解

## 章节概述

本品通过普贤菩萨的提问，地藏菩萨详细说明了娑婆世界及阎浮提罪苦众生所受报处的地狱名号及恶报等事。通过列举各种地狱的名称和刑罚，警示众生要断恶修善，避免堕入恶道。

## 主要内容讲解

### 一、普贤菩萨的提问

**经文要点：**
- 尔时普贤菩萨摩诃萨白地藏菩萨言：仁者，愿为天龙四众，及未来现在一切众生，说娑婆世界，及阎浮提罪苦众生，所受报处，地狱名号，及恶报等事，使未来世末法众生，知是果报

**讲解：**
1. **普贤菩萨**：
   - 普贤菩萨摩诃萨，十大愿王，代表行愿
   - 关心众生，代众生提问
   - 说明普贤菩萨的慈悲心和对众生的关心

2. **提问的对象**：
   - 愿为天龙四众，及未来现在一切众生
   - 天龙四众：天、龙、比丘、比丘尼、优婆塞、优婆夷
   - 未来现在一切众生：包括未来和现在的一切众生
   - 说明普贤菩萨关心所有众生

3. **提问的内容**：
   - 说娑婆世界，及阎浮提罪苦众生，所受报处，地狱名号，及恶报等事
   - 娑婆世界：我们居住的世界
   - 阎浮提：四大部洲之一，这里指娑婆世界的众生
   - 所受报处：所受报应的地方
   - 地狱名号：地狱的名称
   - 恶报等事：恶报等情况

4. **提问的目的**：
   - 使未来世末法众生，知是果报
   - 让未来世末法众生知道地狱的名号和恶报的情况
   - 让众生知道果报的可怕，从而断恶修善
   - 体现了普贤菩萨的慈悲心和对众生的关心

### 二、地狱的位置和名称

**经文要点：**
- 地藏答言：仁者，我今承佛威神，及大士之力，略说地狱名号，及罪报恶报之事
- 仁者，阎浮提东方有山，号曰铁围，其山黑邃，无日月光
- 有大地狱，号极无间，又有地狱，名大阿鼻
- 复有地狱，名曰四角。复有地狱，名曰飞刀。复有地狱，名曰火箭。复有地狱，名曰夹山。复有地狱，名曰通枪。复有地狱，名曰铁车。复有地狱，名曰铁床。复有地狱，名曰铁牛。复有地狱，名曰铁衣。复有地狱，名曰千刃。复有地狱，名曰铁驴。复有地狱，名曰烊铜。复有地狱，名曰抱柱。复有地狱，名曰流火。复有地狱，名曰耕舌。复有地狱，名曰剉首。复有地狱，名曰烧脚。复有地狱，名曰啖眼。复有地狱，名曰铁丸。复有地狱，名曰诤论。复有地狱，名曰铁呋。复有地狱，名曰多嗔

**讲解：**
1. **地藏菩萨的回应**：
   - 地藏答言：仁者，我今承佛威神，及大士之力，略说地狱名号，及罪报恶报之事
   - 地藏菩萨承佛威神和普贤菩萨的力量，略说地狱名号
   - 说明地藏菩萨的谦逊，不居功自傲
   - 体现了地藏菩萨的感恩之心

2. **铁围山**：
   - 阎浮提东方有山，号曰铁围
   - 其山黑邃，无日月光：山是黑色的，深邃，没有日月光明
   - 说明地狱的黑暗和恐怖
   - 体现了地狱的恐怖环境

3. **极无间地狱**：
   - 有大地狱，号极无间
   - 最苦的地狱，无间地狱的极致
   - 说明其痛苦的严重，是最苦的地狱
   - 体现了无间地狱的恐怖

4. **大阿鼻地狱**：
   - 又有地狱，名大阿鼻
   - 阿鼻地狱，无间地狱的另一种称呼
   - 说明其痛苦的持续，没有间断
   - 体现了阿鼻地狱的恐怖

5. **各种地狱的名称**：
   - 四角地狱、飞刀地狱、火箭地狱、夹山地狱、通枪地狱、铁车地狱、铁床地狱、铁牛地狱、铁衣地狱、千刃地狱、铁驴地狱、烊铜地狱、抱柱地狱、流火地狱、耕舌地狱、剉首地狱、烧脚地狱、啖眼地狱、铁丸地狱、诤论地狱、铁呋地狱、多嗔地狱
   - 每个地狱的名称都对应一种刑罚
   - 说明造不同恶业，受不同果报
   - 显示因果的精确性
   - 体现了地狱的复杂性和因果的对应性

### 三、各种地狱的名称

**经文要点：**
- 地藏白言：仁者，铁围之内，有如是等地狱，其数无限
- 更有叫唤地狱、拔舌地狱、粪尿地狱、铜锁地狱、火象地狱、火狗地狱、火马地狱、火牛地狱、火山地狱、火石地狱、火床地狱、火梁地狱。如是等地狱
- 其中各各复有诸小地狱，或一、或二、或三、或四、乃至百千，其中名号，各各不同

**讲解：**
1. **地狱的数量**：
   - 地藏白言：仁者，铁围之内，有如是等地狱，其数无限
   - 说明地狱的数量是无限的，无法完全列举
   - 体现了地狱的复杂性和多样性

2. **更多的地狱名称**：
   - 更有叫唤地狱、拔舌地狱、粪尿地狱、铜锁地狱、火象地狱、火狗地狱、火马地狱、火牛地狱、火山地狱、火石地狱、火床地狱、火梁地狱
   - 说明地狱的种类很多，无法完全列举
   - 每个地狱的名称都对应一种刑罚
   - 体现了地狱的复杂性和因果的对应性

3. **小地狱**：
   - 其中各各复有诸小地狱，或一、或二、或三、或四、乃至百千，其中名号，各各不同
   - 说明每个大地狱中还有小地狱
   - 小地狱的数量也不同，或一、或二、或三、或四、乃至百千
   - 每个小地狱的名称也不同
   - 体现了地狱的复杂性和多样性

4. **地狱名称的含义**：
   - 每个地狱的名称都对应一种刑罚
   - 叫唤地狱：罪人叫唤的地狱
   - 拔舌地狱：拔舌的地狱
   - 粪尿地狱：粪尿的地狱
   - 说明造不同恶业，受不同果报
   - 显示因果的精确性

5. **地狱的种类**：
   - 有各种不同的地狱，对应各种不同的恶业
   - 说明业报的复杂性，不是所有恶业都受同样的报应
   - 体现了因果的对应性和精确性

### 四、地狱中的刑罚

**经文要点：**
- 地藏菩萨告普贤菩萨言：仁者，此者皆是南阎浮提行恶众生，业感如是
- 业力甚大，能敌须弥，能深巨海，能障圣道
- 是故众生莫轻小恶，以为无罪，死后有报，纤毫受之
- 父子至亲，歧路各别，纵然相逢，无肯代受
- 我今承佛威力，略说地狱罪报之事，唯愿仁者暂听是言
- 普贤答言：吾已久知三恶道报，望仁者说，令后世末法一切恶行众生，闻仁者说，使令归佛
- 地藏白言：仁者，地狱罪报，其事如是。或有地狱，取罪人舌，使牛耕之。或有地狱，取罪人心，夜叉食之。或有地狱，镬汤盛沸，煮罪人身。或有地狱，赤烧铜柱，使罪人抱。或有地狱，使诸火烧，趁及罪人。或有地狱，一向寒冰。或有地狱，无限粪尿。或有地狱，纯飞蒺藜。或有地狱，多攒火枪。或有地狱，唯撞胸背。或有地狱，但烧手足。或有地狱，盘缴铁蛇。或有地狱，驱逐铁狗。或有地狱，尽驾铁骡

**讲解：**
1. **业感如是**：
   - 地藏菩萨告普贤菩萨言：仁者，此者皆是南阎浮提行恶众生，业感如是
   - 说明这些地狱都是南阎浮提行恶众生，业力所感的
   - 体现了因果的对应性，造恶业必受恶报

2. **业力的强大**：
   - 业力甚大，能敌须弥，能深巨海，能障圣道
   - 能敌须弥：能对抗须弥山，说明业力的强大
   - 能深巨海：能深入大海，说明业力的深入
   - 能障圣道：能障碍成圣的道路，说明业力的障碍
   - 说明业力的不可思议和强大

3. **莫轻小恶**：
   - 是故众生莫轻小恶，以为无罪，死后有报，纤毫受之
   - 不要轻视小的恶业，以为没有罪
   - 死后有报，纤毫受之：死后有报应，即使是很小的恶业也会受报
   - 说明要谨慎行事，不能轻视小的恶业

4. **无人代受**：
   - 父子至亲，歧路各别，纵然相逢，无肯代受
   - 父子至亲，但歧路各别：父子虽然至亲，但道路不同
   - 纵然相逢，无肯代受：即使相逢，也不肯代受
   - 说明业报只能自己承受，没有人能代受
   - 体现了因果的公平性和严厉性

5. **普贤菩萨的回应**：
   - 普贤答言：吾已久知三恶道报，望仁者说，令后世末法一切恶行众生，闻仁者说，使令归佛
   - 普贤菩萨已经知道三恶道报，但希望地藏菩萨说，让后世末法一切恶行众生，听到地藏菩萨说，使他们归依佛
   - 说明普贤菩萨的慈悲心，希望众生能够归依佛
   - 体现了普贤菩萨的大慈悲心

6. **各种刑罚**：
   - 或有地狱，取罪人舌，使牛耕之：拔掉罪人的舌头，用牛耕
   - 或有地狱，取罪人心，夜叉食之：取出罪人的心，夜叉吃
   - 或有地狱，镬汤盛沸，煮罪人身：用沸汤煮罪人的身体
   - 或有地狱，赤烧铜柱，使罪人抱：用烧红的铜柱，让罪人抱
   - 或有地狱，使诸火烧，趁及罪人：用各种火烧罪人
   - 或有地狱，一向寒冰：一直是寒冰
   - 或有地狱，无限粪尿：无限的粪尿
   - 或有地狱，纯飞蒺藜：纯是飞蒺藜
   - 或有地狱，多攒火枪：很多火枪
   - 或有地狱，唯撞胸背：只撞胸背
   - 或有地狱，但烧手足：只烧手足
   - 或有地狱，盘缴铁蛇：盘绕铁蛇
   - 或有地狱，驱逐铁狗：驱逐铁狗
   - 或有地狱，尽驾铁骡：尽驾铁骡
   - 说明每种地狱都有不同的刑罚
   - 说明造不同恶业，受不同果报
   - 显示因果的对应性

7. **刑罚的恐怖**：
   - 各种刑罚都非常恐怖，说明地狱之苦的严重
   - 警示众生要断恶修善，避免堕入恶道
   - 体现了地狱的恐怖和因果的严厉性

### 六、地狱的复杂性

**经文要点：**
- 仁者，如是等报，各各狱中，有百千种业道之器，无非是铜、是铁、是石、是火。此四种物，众业行感
- 若广说地狱罪报等事，一一狱中，更有百千种苦楚，何况多狱
- 我今承佛威神及仁者问，略说如是。若广解说，穷劫不尽

**讲解：**
1. **业道之器**：
   - 如是等报，各各狱中，有百千种业道之器
   - 每个地狱都有百千种业道之器
   - 说明地狱的复杂，业报的复杂
   - 体现了地狱的复杂性和多样性

2. **刑罚的材料**：
   - 无非是铜、是铁、是石、是火。此四种物，众业行感
   - 刑罚的材料无非是铜、铁、石、火这四种
   - 这四种物，都是众业行感：这四种物，都是众生的业力所感的
   - 说明业力的力量，能够感召这些物
   - 体现了业力的强大

3. **苦楚的众多**：
   - 若广说地狱罪报等事，一一狱中，更有百千种苦楚，何况多狱
   - 如果详细说地狱罪报等事，每个地狱中，更有百千种苦楚
   - 何况多狱：何况有很多地狱
   - 说明地狱之苦的众多，无法完全说尽
   - 体现了地狱的复杂性和恐怖

4. **说不尽**：
   - 我今承佛威神及仁者问，略说如是。若广解说，穷劫不尽
   - 地藏菩萨承佛威神和普贤菩萨的问，略说如是
   - 若广解说，穷劫不尽：如果详细解说，穷劫也说不尽
   - 说明地狱之苦的说不尽，业报的复杂
   - 体现了地狱的复杂性和因果的复杂性
