# 利益存亡品第七讲解

## 章节概述

本品通过地藏菩萨的说明和大辩长者的提问，详细说明了为亡者修福的方法和功德，特别是临终时的注意事项和七七日内的超度方法。强调了为亡者修福的重要性，以及如何正确地为亡者超度。通过对比自修和他人代修的功德分配，强调了生前修行的必要性。

## 主要内容讲解

### 一、地藏菩萨的观察

**经文要点：**
- 地藏菩萨白佛言：我观是阎浮众生，举心动念，无非是罪
- 脱获善利，多退初心
- 若遇恶缘，念念增益
- 是等辈人，如履泥涂，负于重石，渐困渐重，足步深邃
- 若得遇知识，替与减负，或全与负。是知识有大力故，复相扶助，劝令牢脚。若达平地，须省恶路，无再经历

**讲解：**
1. **众生的状况**：
   - 举心动念，无非是罪：每一个念头都是罪业
   - 说明阎浮提众生的业障深重，心念不清净
   - 说明修行的困难，需要长期的努力
   - 体现了众生修行的难度和业障的深重
   - 这是地藏菩萨对众生状况的深刻观察

2. **退转的情况**：
   - 脱获善利，多退初心：好不容易得到善利，却多退初心
   - 说明众生容易退转，即使得到善利，也容易退失初心
   - 说明要坚持修行，不能因为得到一点善利就懈怠
   - 体现了众生修行的不稳定性
   - 说明修行需要持之以恒，不能半途而废

3. **恶缘的影响**：
   - 若遇恶缘，念念增益：如果遇到恶缘，恶念会不断增长
   - 说明恶缘的危害，会让恶念不断增长
   - 说明要远离恶缘，避免被恶缘影响
   - 体现了恶缘对修行的障碍
   - 说明要选择好的环境，远离恶友

4. **比喻的意义**：
   - 如履泥涂，负于重石：像在泥泞的路上行走，还背着沉重的石头
   - 渐困渐重，足步深邃：越来越困重，脚步越来越深陷
   - 说明众生的困难，业障越来越重
   - 体现了众生修行的艰难和业障的累积
   - 说明如果不修行，业障会越来越重

5. **善知识的帮助**：
   - 若得遇知识，替与减负，或全与负：如果遇到善知识，会帮助减轻负担，或者完全承担
   - 是知识有大力故，复相扶助，劝令牢脚：善知识有大力量，会互相扶助，劝令站稳脚跟
   - 若达平地，须省恶路，无再经历：如果到达平地，要反省恶路，不要再经历
   - 说明善知识的重要性，能够帮助减轻业障
   - 体现了善知识对修行的帮助和指导
   - 说明要珍惜善知识，听从善知识的教导

### 二、临终时的修福

**经文要点：**
- 习恶众生，从纤毫间，便至无量
- 是诸众生有如此习，临命终时，父母眷属，宜为设福，以资前路
- 或悬幡盖及然油灯，或转读尊经、或供养佛像及诸圣像，乃至念佛菩萨，及圣贤名号
- 一名一号，历临终人耳根，或闻在本识
- 是诸众生所造恶业，计其感果，必堕恶趣，缘是眷属为临终人修此圣因，如是众罪，悉皆消灭

**讲解：**
1. **习恶的累积**：
   - 习恶众生，从纤毫间，便至无量：习恶的众生，从很小的恶开始，就会累积到无量
   - 说明恶习的可怕，小的恶习会不断累积
   - 说明要防微杜渐，不能轻视小的恶习
   - 体现了恶习的累积性和危害性
   - 说明要断除恶习，不能让它累积

2. **临终的重要性**：
   - 临命终时，是关键时刻：临命终时是决定来生去向的关键时刻
   - 决定来生的去向：临终时的念头和因缘决定来生的去向
   - 所以要为亡者修福：因此要为临终的人修福，帮助他往生善道
   - 体现了临终关怀的重要性
   - 说明临终时的因缘对来生有重要影响

3. **修福的方法**：
   - 悬幡盖及然油灯：悬挂幡盖，点燃油灯
     - 幡盖表示庄严，油灯表示光明
     - 能够为临终的人营造庄严的环境
   - 转读尊经：转读佛经
     - 读经的功德能够利益临终的人
     - 让临终的人听到佛法的声音
   - 供养佛像及诸圣像：供养佛像和圣像
     - 供养的功德能够利益临终的人
     - 让临终的人看到佛像，生起善念
   - 念佛菩萨，及圣贤名号：称念佛菩萨和圣贤的名号
     - 称名的功德不可思议
     - 让临终的人听到佛菩萨的名号
   - 说明修福的方法很多，要根据情况选择

4. **闻名的功德**：
   - 一名一号，历临终人耳根：一个名号，经过临终人的耳根
   - 或闻在本识：或者听闻到本识（阿赖耶识）中
   - 说明即使临终人意识不清，也能听闻到名号
   - 体现了佛菩萨名号的功德不可思议
   - 说明称名的功德能够深入本识，影响来生

5. **修福的效果**：
   - 是诸众生所造恶业，计其感果，必堕恶趣：这些众生所造的恶业，按照因果，必堕恶趣
   - 缘是眷属为临终人修此圣因：因为眷属为临终人修这些圣因
   - 如是众罪，悉皆消灭：这些罪业，全部消灭
   - 说明为临终人修福的功德，能够消灭罪业
   - 体现了修福的功德不可思议
   - 说明为亡者修福的重要性

### 三、七七日内的修福

**经文要点：**
- 若能更为身死之后，七七日内，广造众善
- 能使是诸众生永离恶趣，得生人天，受胜妙乐
- 现在眷属，利益无量

**讲解：**
1. **七七日的重要性**：
   - 身死之后，七七日内：死后四十九天内
   - 是决定来生去向的关键时期：这是决定来生去向的关键时期
   - 中阴身在这期间还没有投生，还有机会改变
   - 所以要广造众善：因此要广造各种善业
   - 体现了七七日内超度的重要性
   - 说明这是为亡者修福的最佳时机

2. **修福的效果**：
   - 能使是诸众生永离恶趣：能够让这些众生永远脱离恶趣
   - 得生人天，受胜妙乐：能够生到人天，享受殊胜的快乐
   - 现在眷属，利益无量：现在的眷属，也能得到无量的利益
   - 说明为亡者修福，不仅利益亡者，也利益生者
   - 体现了修福的功德是双向的
   - 说明为亡者修福是自利利他的行为

3. **广造众善的意义**：
   - 广造众善：广泛地造各种善业
   - 包括诵经、念佛、放生、布施、供养等
   - 要回向给亡者，帮助亡者往生善道
   - 说明修福要全面，不能只做一种
   - 体现了修福的全面性和重要性

### 四、临终时的禁忌

**经文要点：**
- 是故我今对佛世尊，及天龙八部，人非人等，劝于阎浮提众生临终之日，慎勿杀害，及造恶缘，拜祭鬼神，求诸魍魉
- 何以故？尔所杀害乃至拜祭，无纤毫之力，利益亡人，但结罪缘，转增深重
- 假使来世或现在生，得获圣分，生人天中。缘是临终被诸眷属造是恶因，亦令是命终人殃累对辩，晚生善处
- 何况临命终人，在生未曾有少善根，各据本业，自受恶趣，何忍眷属更为增业
- 譬如有人从远地来，绝粮三日，所负担物，强过百斤，忽遇邻人，更附少物，以是之故，转复困重

**讲解：**
1. **禁忌的事项**：
   - 慎勿杀害：千万不要杀害生命
     - 杀害生命是重罪，会加重亡者的业障
     - 临终时杀害生命，会让亡者背负杀业
   - 及造恶缘：不要造恶缘
     - 不要做各种恶事，如偷盗、邪淫、妄语等
     - 造恶缘会加重亡者的业障
   - 拜祭鬼神，求诸魍魉：不要拜祭鬼神，求诸魍魉
     - 拜祭鬼神没有功德，反而会结恶缘
     - 求诸魍魉会与恶道结缘，不利于往生善道
   - 说明这些行为都是禁忌，不能做

2. **为什么不能做**：
   - 无纤毫之力，利益亡人：没有一点力量利益亡人
   - 但结罪缘，转增深重：只会结下罪缘，让业障更加深重
   - 说明这些行为不仅没有功德，反而有害
   - 体现了这些行为的危害性
   - 说明要为亡者修善，不能造恶

3. **对善根者的影响**：
   - 假使来世或现在生，得获圣分，生人天中：即使来世或现在生，能够获得圣分，生到人天中
   - 缘是临终被诸眷属造是恶因：因为临终时被眷属造了这些恶因
   - 亦令是命终人殃累对辩，晚生善处：也会让命终人受到殃累，对辩，晚生善处
   - 说明即使有善根，也会因为眷属造恶而受影响
   - 体现了眷属行为对亡者的影响

4. **对无善根者的影响**：
   - 何况临命终人，在生未曾有少善根：何况临命终人，在生时没有少善根
   - 各据本业，自受恶趣：根据本业，自己受恶趣
   - 何忍眷属更为增业：怎么忍心让眷属再增加业障
   - 说明没有善根的人，本来就要受恶趣
   - 如果眷属再造恶，会让业障更重
   - 体现了眷属造恶对亡者的严重危害

5. **比喻的意义**：
   - 譬如有人从远地来，绝粮三日：比如有人从远地来，断粮三天
   - 所负担物，强过百斤：所负担的东西，超过百斤
   - 忽遇邻人，更附少物：忽然遇到邻人，又加上一些东西
   - 以是之故，转复困重：因为这个缘故，更加困重
   - 说明造恶缘就像给已经负重的人再加负担
   - 体现了造恶缘的危害，会让亡者更加困难
   - 说明要为亡者减负，不能增负

### 五、大辩长者的提问

**经文要点：**
- 会中有一长者，名曰大辩，是长者久证无生，化度十方，现长者身
- 合掌恭敬，问地藏菩萨言：大士，是南阎浮提众生，命终之后，小大眷属，为修功德，乃至设斋，造众善因，是命终人，得大利益及解脱不？

**讲解：**
1. **大辩长者**：
   - 久证无生：已经证得无生法忍很久了
     - 无生法忍是菩萨的境界，说明大辩长者是大菩萨
   - 化度十方：化度十方众生
     - 说明大辩长者的慈悲和愿力
   - 现长者身：示现长者身
     - 说明菩萨会以各种身份度化众生
   - 代众生提问：代表众生提问
     - 说明大辩长者的慈悲，关心众生的问题
   - 体现了大辩长者的慈悲和智慧

2. **提问的内容**：
   - 为亡者修功德，设斋，造众善因：为亡者修功德，设斋，造各种善因
   - 亡者能否得大利益及解脱：亡者能否得到大利益和解脱
   - 这是众生普遍关心的问题
   - 体现了大辩长者对众生的关心
   - 说明这个问题的重要性

### 六、地藏菩萨的回答

**经文要点：**
- 未来现在诸众生等，临命终日，得闻一佛名、一菩萨名、一辟支佛名，不问有罪无罪，悉得解脱
- 若有男子女人，在生不修善因多造众罪。命终之后，眷属小大，为造福利一切圣事，七分之中而乃获一，六分功德，生者自利
- 以是之故，未来现在善男女等，闻健自修，分分己获
- 无常大鬼，不期而到，冥冥游神，未知罪福。七七日内，如痴如聋，或在诸司辩论业果。审定之后，据业受生。未测之间，千万愁苦，何况堕于诸恶趣等

**讲解：**
1. **闻名的功德**：
   - 临命终日，得闻一佛名、一菩萨名、一辟支佛名：临命终日，能够听到一个佛名、一个菩萨名、一个辟支佛名
   - 不问有罪无罪，悉得解脱：不管有罪无罪，都能得到解脱
   - 说明闻名的功德不可思议
   - 体现了佛菩萨名号的威神力
   - 说明临终时听闻佛菩萨名号的重要性
   - 这是最简单有效的超度方法

2. **修福的分配**：
   - 若有男子女人，在生不修善因多造众罪：如果有男子女人，在生时不修善因，多造众罪
   - 命终之后，眷属小大，为造福利一切圣事：命终之后，眷属大小，为造福利一切圣事
   - 七分之中而乃获一：七分功德中，亡者只能得到一分
   - 六分功德，生者自利：六分功德，生者自己得到利益
   - 说明为亡者修福，亡者只能得到少部分功德
   - 体现了自修的重要性，自己修行的功德全部自己得到
   - 说明不要等到死后让别人代修，要自己修行

3. **自修的重要性**：
   - 闻健自修，分分己获：在健康时自己修行，每一分功德都是自己获得
   - 说明要自己修行，不要等到临终
   - 自己修行的功德全部自己得到，别人代修的功德只能得到少部分
   - 体现了生前修行的重要性
   - 说明要趁健康时精进修行，不要等到临终

4. **中阴身的状况**：
   - 无常大鬼，不期而到：无常大鬼，不期而至
     - 说明死亡是突然的，无法预料
   - 冥冥游神，未知罪福：冥冥中游荡的神识，不知道罪福
     - 说明中阴身的状态，不知道自己的罪福
   - 七七日内，如痴如聋：七七日内，像痴聋一样
     - 说明中阴身的状态，意识不清
   - 或在诸司辩论业果：或者在各个司中辩论业果
     - 说明中阴身会到各个司中辩论业果
   - 审定之后，据业受生：审定之后，根据业力受生
     - 说明会根据业力决定投生
   - 未测之间，千万愁苦：在未测之间，千万愁苦
     - 说明中阴身的痛苦和不安
   - 何况堕于诸恶趣等：何况堕入各种恶趣
     - 说明堕入恶趣的痛苦更严重
   - 体现了中阴身的状况和重要性

### 七、七七日内的状况

**经文要点：**
- 是命终人，未得受生，在七七日内，念念之间，望诸骨肉眷属，与造福力救拔
- 过是日后，随业受报
- 若是罪人，动经千百岁中，无解脱日
- 若是五无间罪，堕大地狱，千劫万劫，永受众苦

**讲解：**
1. **七七日内的状况**：
   - 未得受生：还没有投生
     - 说明七七日内还在中阴身状态
   - 念念之间，望诸骨肉眷属，与造福力救拔：念念之间，希望骨肉眷属，为他修福力救拔
     - 说明亡者在这期间非常需要眷属的帮助
     - 体现了七七日内超度的重要性
   - 说明七七日内是救度的关键时期
   - 体现了为亡者修福的紧迫性

2. **过是日后的状况**：
   - 过是日后，随业受报：过了七七日后，随业受报
     - 说明过了七七日，就会根据业力投生
   - 若是罪人，动经千百岁中，无解脱日：如果是罪人，经过千百岁，没有解脱的日子
     - 说明造恶业的果报严重，时间很长
   - 若是五无间罪，堕大地狱，千劫万劫，永受众苦：如果是五无间罪，堕大地狱，千劫万劫，永远受各种苦
     - 说明五无间罪的果报极其严重
     - 体现了五无间罪的恐怖
   - 说明过了七七日，就很难改变了
   - 体现了七七日内超度的重要性

3. **七七日的重要性**：
   - 七七日内是救度的最佳时机
   - 过了七七日，就会根据业力投生，很难改变
   - 所以要抓紧时间，在七七日内为亡者修福
   - 体现了时间的重要性

### 八、设斋的注意事项

**经文要点：**
- 如是罪业众生，命终之后，眷属骨肉，为修营斋，资助业道未斋食竟，及营斋之次，米泔菜叶，不弃于地，乃至诸食未献佛僧，勿得先食
- 如有违食，及不精勤，是命终人，了不得力
- 如精勤护净奉献佛僧，是命终人，七分获一
- 是故长者，阎浮众生，若能为其父母乃至眷属，命终之后，设斋供养，志心勤恳。如是之人，存亡获利

**讲解：**
1. **设斋的注意事项**：
   - 米泔菜叶，不弃于地：米泔菜叶，不要丢弃在地上
     - 说明要珍惜食物，不能浪费
     - 体现了对食物的恭敬
