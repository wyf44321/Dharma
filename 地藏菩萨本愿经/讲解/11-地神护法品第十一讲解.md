# 地神护法品第十一讲解

## 章节概述

本品通过坚牢地神的说明，详细介绍了供养地藏菩萨的十种利益，以及地神护法的功德。说明了地藏菩萨与阎浮提的大因缘，以及如何通过供养地藏菩萨来获得护持。

## 主要内容讲解

### 一、坚牢地神的赞叹

**经文要点：**
- 坚牢地神白佛言：世尊，我从昔来瞻视顶礼无量菩萨摩诃萨，皆是大不可思议神通智慧，广度众生
- 是地藏菩萨摩诃萨，于诸菩萨誓愿深重，世尊，是地藏菩萨于阎浮提有大因缘
- 如文殊、普贤、观音、弥勒，亦化百千身形，度于六道，其愿尚有毕竟
- 是地藏菩萨教化六道一切众生，所发誓愿劫数，如千百亿恒河沙

**讲解：**
1. **坚牢地神的观察**：
   - 我从昔来瞻视顶礼无量菩萨摩诃萨
   - 坚牢地神是护持大地的神，长期观察和顶礼无量菩萨
   - 说明坚牢地神有丰富的经验，能够比较不同菩萨的功德
   - 体现了坚牢地神的恭敬心和观察力
   - 说明坚牢地神对菩萨的了解和认识

2. **菩萨的共同特点**：
   - 皆是大不可思议神通智慧，广度众生
   - 所有菩萨都有大不可思议的神通和智慧
   - 都以广度众生为己任
   - 说明菩萨的共同特点，都是大慈悲、大智慧、大神通
   - 体现了菩萨的功德和愿力

3. **地藏菩萨的特殊性**：
   - 于诸菩萨誓愿深重：地藏菩萨的誓愿比其他菩萨更深重
   - 于阎浮提有大因缘：地藏菩萨与娑婆世界有特别深的因缘
   - 所发誓愿劫数，如千百亿恒河沙：地藏菩萨发愿的时间极长，如千百亿恒河沙劫
   - 说明地藏菩萨的愿力特别深广，与娑婆世界的因缘特别深
   - 体现了地藏菩萨的特殊地位和重要性

4. **与其他菩萨的比较**：
   - 如文殊、普贤、观音、弥勒，亦化百千身形，度于六道，其愿尚有毕竟
   - 文殊、普贤、观音、弥勒等大菩萨，也化百千身形度化六道众生
   - 但他们的愿力有毕竟的时候，有完成的时候
   - 地藏菩萨的愿力，如千百亿恒河沙，没有毕竟的时候
   - 说明地藏菩萨的愿力是无限的，没有完成的时候
   - 体现了地藏菩萨"众生度尽，方证菩提"的大愿
   - 说明地藏菩萨把度化众生放在第一位，自己成佛放在后面

5. **大因缘的意义**：
   - 地藏菩萨于阎浮提有大因缘
   - 说明地藏菩萨与娑婆世界的众生有特别深的因缘
   - 这是地藏菩萨特别关注娑婆世界众生的原因
   - 说明地藏菩萨对娑婆世界众生的特别关怀
   - 体现了地藏菩萨的大慈悲心

### 二、供养地藏菩萨的十种利益

**经文要点：**
- 世尊，我观未来及现在众生，于所住处，于南方清洁之地，以土石竹木作其龛室，是中能塑画，乃至金银铜铁，作地藏形像，烧香供养，瞻礼赞叹
- 是人居处，即得十种利益。何等为十？

1. **土地丰壤**
2. **家宅永安**
3. **先亡生天**
4. **现存益寿**
5. **所求遂意**
6. **无水火灾**
7. **虚耗辟除**
8. **杜绝恶梦**
9. **出入神护**
10. **多遇圣因**

**讲解：**
1. **供养的对象**：
   - 未来及现在众生：包括未来和现在的一切众生
   - 说明这个法门适合所有众生，不分时间
   - 体现了地藏菩萨的大慈悲，要利益一切众生

2. **供养的地点**：
   - 于所住处，于南方清洁之地
   - 在所住的地方，选择南方清洁的地方
   - 南方代表光明、吉祥，清洁代表清净
   - 说明供养要选择清净的地方，体现恭敬心
   - 体现了供养的如法和恭敬

3. **供养的设施**：
   - 以土石竹木作其龛室：用土、石、竹、木等材料做龛室
   - 龛室是供奉佛像的小室，可以用简单的材料制作
   - 说明供养不一定要用昂贵的材料，重要的是恭敬心
   - 体现了供养的方便和实用

4. **供养的佛像**：
   - 是中能塑画，乃至金银铜铁，作地藏形像
   - 可以在龛室中塑画地藏菩萨的形像
   - 可以用各种材料，如金、银、铜、铁等
   - 说明供养的方式多样，可以根据条件选择
   - 重要的是要有地藏菩萨的形像，体现恭敬心

5. **供养的方法**：
   - 烧香供养：烧香供养，表示恭敬
   - 瞻礼赞叹：瞻仰礼拜，赞叹地藏菩萨的功德
   - 说明供养要有恭敬心，要瞻礼赞叹
   - 体现了供养的如法和恭敬

6. **十种利益详解**：
   - **土地丰壤**：土地肥沃，收成好
     - 说明供养地藏菩萨能带来物质上的利益
     - 土地丰壤代表生活富足，不愁吃穿
     - 体现了地藏菩萨对众生的护持
   
   - **家宅永安**：家宅平安，没有灾难
     - 说明供养地藏菩萨能带来平安
     - 家宅永安代表家庭和睦，没有灾难
     - 体现了地藏菩萨的护持，让家庭平安
   
   - **先亡生天**：已故的亲人能生天
     - 说明供养地藏菩萨能利益已故的亲人
     - 先亡生天代表已故的亲人能生到天上，不再受苦
     - 体现了地藏菩萨的大慈悲，不仅利益现在的人，还利益已故的亲人
   
   - **现存益寿**：现在的人能延寿
     - 说明供养地藏菩萨能延寿
     - 现存益寿代表现在的人能延长寿命，身体健康
     - 体现了地藏菩萨的护持，让现在的人健康长寿
   
   - **所求遂意**：所求的愿望都能实现
     - 说明供养地藏菩萨能满愿
     - 所求遂意代表所求的愿望都能实现，心想事成
     - 体现了地藏菩萨的大威神力，能够满众生的愿
   
   - **无水火灾**：没有水灾和火灾
     - 说明供养地藏菩萨能避免自然灾害
     - 无水火灾代表没有水灾和火灾等自然灾害
     - 体现了地藏菩萨的护持，让居住环境安全
   
   - **虚耗辟除**：虚耗消除
     - 说明供养地藏菩萨能消除虚耗
     - 虚耗指不必要的消耗和浪费
     - 体现了地藏菩萨的护持，让生活更有效率
   
   - **杜绝恶梦**：没有恶梦
     - 说明供养地藏菩萨能避免恶梦
     - 杜绝恶梦代表睡眠安稳，没有恶梦干扰
     - 体现了地藏菩萨的护持，让心理安宁
   
   - **出入神护**：出入都有神护
     - 说明供养地藏菩萨能得到神的护持
     - 出入神护代表出入都有神护，平安顺利
     - 体现了地藏菩萨的护持，让出行安全
   
   - **多遇圣因**：多遇到成圣的因缘
     - 说明供养地藏菩萨能遇到成圣的因缘
     - 多遇圣因代表多遇到成圣的因缘，修行顺利
     - 体现了地藏菩萨的护持，让修行顺利，最终成圣

7. **利益的殊胜**：
   - 这十种利益涵盖了物质、精神、现世、来世等各个方面
   - 说明供养地藏菩萨的利益是全面的，不仅利益现世，还利益来世
   - 体现了地藏菩萨的大慈悲和大威神力
   - 说明只要恭敬供养地藏菩萨，就能得到这些利益

### 三、地神的护持

**经文要点：**
- 复白佛言：世尊，未来世中，若有善男子善女人，于所住处，有此经典及菩萨像，是人更能转读经典，供养菩萨
- 我常日夜以本神力，卫护是人，乃至水火盗贼，大横小横，一切恶事，悉皆消灭

**讲解：**
1. **护持的条件**：
   - 于所住处，有此经典及菩萨像：在所住的地方，有《地藏菩萨本愿经》和地藏菩萨像
   - 是人更能转读经典，供养菩萨：这个人能够转读经典，供养地藏菩萨
   - 说明护持需要两个条件：有经典和菩萨像，还要转读经典和供养菩萨
   - 体现了护持的条件，要有经典和菩萨像，还要修行
   - 说明护持不是被动的，需要主动修行

2. **护持的时间**：
   - 我常日夜以本神力，卫护是人
   - 常日夜：白天和夜晚都护持，没有间断
   - 说明地神的护持是持续的，不会间断
   - 体现了地神的慈悲和责任心

3. **护持的力量**：
   - 以本神力：用本来的神力护持
   - 说明地神用自己本来的神力护持，不是借用其他力量
   - 体现了地神的神通力和护持的决心

4. **护持的范围**：
   - 乃至水火盗贼，大横小横，一切恶事，悉皆消灭
   - 水灾、火灾、盗贼、大横、小横等一切恶事，都会消灭
   - 大横：大的横祸，如大的灾难
   - 小横：小的横祸，如小的意外
   - 说明护持的范围很广，包括各种灾难和意外
   - 体现了地神护持的全面性

5. **护持的效果**：
   - 一切恶事，悉皆消灭：一切恶事都会消灭
   - 说明护持的效果很好，能够消除一切恶事
   - 体现了地神护持的威力和有效性
   - 说明只要符合条件，就能得到地神的护持

### 四、佛陀的赞叹

**经文要点：**
- 佛告坚牢地神：汝大神力，诸神少及
- 何以故？阎浮土地，悉蒙汝护，乃至草木沙石，稻麻竹苇，谷米宝贝，从地而有，皆因汝力
- 又当称扬地藏菩萨利益之事
- 汝之功德，及以神通，百千倍于常分地神

**讲解：**
1. **地神的大神力**：
   - 汝大神力，诸神少及：你的大神力，其他神很少能及
   - 说明坚牢地神的神力很大，其他神很少能及
   - 体现了坚牢地神的神通力和地位
   - 说明坚牢地神不是普通的地神，而是有大神力的地神

2. **地神护持的范围**：
   - 阎浮土地，悉蒙汝护：阎浮提的土地，都蒙你护持
   - 说明坚牢地神护持整个阎浮提的土地
   - 体现了坚牢地神护持的范围很广
   - 说明坚牢地神是护持整个大地的神

3. **地神护持的内容**：
   - 乃至草木沙石，稻麻竹苇，谷米宝贝，从地而有，皆因汝力
   - 草木、沙石、稻麻、竹苇、谷米、宝贝等，从地而有的，都因为你的力量
   - 说明大地的一切，都因为坚牢地神的力量而存在
   - 体现了坚牢地神的重要性，大地的一切都依赖他
   - 说明坚牢地神是大地的主宰，护持大地的一切

4. **地神的责任**：
   - 又当称扬地藏菩萨利益之事：还要称扬地藏菩萨的利益之事
   - 说明坚牢地神不仅要护持大地，还要称扬地藏菩萨
   - 体现了坚牢地神对地藏菩萨的恭敬和护持
   - 说明坚牢地神是地藏菩萨的护法神

5. **地神的功德**：
   - 汝之功德，及以神通，百千倍于常分地神
   - 你的功德和神通，是普通地神的百千倍
   - 说明坚牢地神的功德和神通很大，远超普通地神
   - 体现了坚牢地神的殊胜和重要性
   - 说明坚牢地神不是普通的地神，而是有大功德的地神

### 五、地神的护持承诺

**经文要点：**
- 若未来世中，有善男子善女人，供养菩萨，及转读是经，但依地藏本愿经一事修行者
- 汝以本神力而拥护之，勿令一切灾害及不如意事，辄闻于耳，何况令受
- 非但汝独护是人故，亦有释梵眷属，诸天眷属，拥护是人
- 何故得如是圣贤拥护，皆由瞻礼地藏形像，及转读是本愿经故，自然毕竟出离苦海，证涅槃乐
- 以是之故，得大拥护

**讲解：**
1. **护持的条件**：
   - 供养菩萨，及转读是经：供养地藏菩萨，转读《地藏菩萨本愿经》
   - 但依地藏本愿经一事修行者：只要依《地藏菩萨本愿经》一件事修行的人
   - 说明护持的条件很简单，只要供养地藏菩萨、转读经典、依经修行即可
   - 体现了护持的方便和容易
   - 说明只要依经修行，就能得到护持

2. **护持的方式**：
   - 汝以本神力而拥护之：你用本来的神力拥护他
   - 说明坚牢地神用自己本来的神力护持
   - 体现了坚牢地神的护持是主动的，用神力护持
   - 说明护持是有力量的，不是形式上的

3. **护持的效果**：
   - 勿令一切灾害及不如意事，辄闻于耳，何况令受
   - 不要让一切灾害和不如意的事，连听到都没有，何况让它们发生
   - 说明护持的效果很好，连听到都不会，更不会发生
   - 体现了护持的彻底性和有效性
   - 说明护持是全面的，能够避免一切灾害和不如意的事

4. **护持的力量**：
   - 非但汝独护是人故，亦有释梵眷属，诸天眷属，拥护是人
   - 不只是你一个人护持，还有释梵眷属、诸天眷属，都拥护这个人
   - 释梵眷属：帝释天和梵天的眷属
   - 诸天眷属：各种天的眷属
   - 说明护持的力量很大，不仅有地神护持，还有诸天护持
   - 体现了护持的殊胜和全面性
   - 说明供养地藏菩萨、转读经典的人，会得到诸天的护持

5. **护持的原因**：
   - 何故得如是圣贤拥护，皆由瞻礼地藏形像，及转读是本愿经故
   - 为什么能得到这样的圣贤拥护？都是因为瞻礼地藏形像，转读《地藏菩萨本愿经》
   - 说明护持的原因很简单，就是瞻礼地藏形像和转读经典
   - 体现了护持的因缘，只要恭敬供养地藏菩萨，就能得到护持

6. **护持的最终目标**：
   - 自然毕竟出离苦海，证涅槃乐
   - 自然：自然而然，不需要刻意追求
   - 毕竟：最终，最后
   - 出离苦海：脱离苦海，不再受苦
   - 证涅槃乐：证得涅槃的快乐，最终成佛
   - 说明护持的最终目标是让众生出离苦海，证得涅槃
   - 体现了护持的终极意义，不仅护持现世，还要护持来世
   - 说明护持是全面的，从现世到成佛，都有护持

7. **护持的意义**：
   - 以是之故，得大拥护：因为这个原因，得到大拥护
   - 说明因为瞻礼地藏形像和转读经典，所以得到大拥护
   - 体现了护持的因缘和果报
   - 说明只要恭敬供养地藏菩萨，就能得到大拥护
