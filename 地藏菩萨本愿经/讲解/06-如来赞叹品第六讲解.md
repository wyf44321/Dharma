# 如来赞叹品第六讲解

## 章节概述

本品通过佛陀放光赞叹地藏菩萨，以及普广菩萨的提问，详细说明了供养地藏菩萨、读诵本经的各种功德利益。包括消除罪业、增长福报、超度亡灵、保护家宅等方面的利益。通过列举各种功德利益，鼓励众生归依地藏菩萨、读诵本经、供养三宝，从而获得现世和来世的利益。

## 主要内容讲解

### 一、佛陀的赞叹

**经文要点：**
- 世尊举身放大光明，遍照百千万亿恒河沙等诸佛世界
- 出大音声，普告诸佛世界一切诸菩萨摩诃萨，及天龙、鬼神、人、非人等
- 称扬赞叹地藏菩萨摩诃萨，于十方世界，现大不可思议威神慈悲之力
- 救护一切罪苦之事
- 吾灭度后，汝等诸菩萨大士，及天龙鬼神等，广作方便，卫护是经，令一切众生证涅槃乐

**讲解：**
1. **放光的意义**：
   - 世尊举身放大光明，遍照百千万亿恒河沙等诸佛世界
   - 佛陀放光，显示对地藏菩萨的赞叹和重视
   - 光明遍照一切世界，说明地藏菩萨功德的无量，遍及一切处
   - 说明这次赞叹的殊胜，不是一般的赞叹
   - 体现了佛陀对地藏菩萨的认可和赞叹

2. **大音声的意义**：
   - 出大音声，普告诸佛世界一切诸菩萨摩诃萨，及天龙、鬼神、人、非人等
   - 说明佛陀对地藏菩萨的重视，用大音声普告一切
   - 说明地藏菩萨的重要性，值得一切众生了解
   - 说明这次法会的殊胜，一切众生都能听到
   - 体现了佛陀对地藏菩萨的赞叹是公开的、普遍的

3. **威神慈悲之力**：
   - 称扬赞叹地藏菩萨摩诃萨，于十方世界，现大不可思议威神慈悲之力
   - 大不可思议：说明地藏菩萨的威神力是不可思议的，无法用语言完全表达
   - 威神慈悲之力：既有威神力，又有慈悲力，两者结合
   - 于十方世界：说明地藏菩萨的功德遍及十方世界
   - 救护一切罪苦之事：说明地藏菩萨的功德是救护一切罪苦众生
   - 体现了地藏菩萨的大慈悲和大威神力

4. **佛陀的嘱托**：
   - 吾灭度后，汝等诸菩萨大士，及天龙鬼神等，广作方便，卫护是经，令一切众生证涅槃乐
   - 佛陀嘱托诸菩萨大士及天龙鬼神等，要卫护这部经
   - 广作方便：用各种善巧方便卫护这部经
   - 令一切众生证涅槃乐：让一切众生都能证得涅槃的快乐
   - 说明这部经的重要性，需要诸菩萨和天龙鬼神等共同护持
   - 体现了佛陀对这部经的重视和对众生的慈悲

### 二、闻地藏菩萨名的功德

**经文要点：**
- 未来世中，若有善男子、善女人，闻是地藏菩萨摩诃萨名者
- 或合掌者、赞叹者、作礼者、恋慕者
- 是人超越三十劫罪

**讲解：**
1. **闻名的功德**：
   - 未来世中，若有善男子、善女人，闻是地藏菩萨摩诃萨名者
   - 仅仅听到地藏菩萨的名字，就能得到大利益
   - 说明地藏菩萨的威神力不可思议
   - 说明闻名就能种下善根，得到加持
   - 体现了地藏菩萨的大慈悲和大威神力

2. **恭敬的行为**：
   - 或合掌者、赞叹者、作礼者、恋慕者
   - 合掌：双手合十，表示恭敬
   - 赞叹：称赞地藏菩萨的功德
   - 作礼：向地藏菩萨行礼
   - 恋慕：对地藏菩萨生起仰慕之心
   - 这些都是恭敬的表现，显示恭敬心的重要性
   - 说明恭敬心是得到功德的基础

3. **超越三十劫罪**：
   - 是人超越三十劫罪
   - 三十劫：时间极长，说明罪业的深重
   - 超越：超越、消除，说明功德的殊胜
   - 仅仅闻名、合掌、赞叹、作礼、恋慕，就能超越三十劫罪
   - 显示地藏菩萨的威神力不可思议
   - 说明恭敬心的重要性，恭敬心能够感通地藏菩萨的加持
   - 体现了地藏菩萨的大慈悲和大威神力

### 三、瞻礼地藏菩萨像的功德

**经文要点：**
- 若有善男子、善女人，或彩画形像，或土石胶漆，金银铜铁、作此菩萨，一瞻一礼者
- 是人百返生于三十三天，永不堕于恶道
- 假如天福尽故，下生人间，犹为国王，不失大利

**讲解：**
1. **塑像的功德**：
   - 或彩画形像，或土石胶漆，金银铜铁、作此菩萨
   - 彩画形像：用彩画的方式画地藏菩萨像
   - 土石胶漆：用土、石、胶、漆等材料塑像
   - 金银铜铁：用金、银、铜、铁等材料塑像
   - 无论用什么材料，都有功德，说明恭敬心比材料更重要
   - 显示恭敬心的重要性，材料不是关键，关键是恭敬心
   - 说明塑像的功德很大，能够种下善根

2. **瞻礼的功德**：
   - 一瞻一礼者：一瞻一礼，表示恭敬
   - 瞻：瞻仰，用眼睛看
   - 礼：礼拜，用身体行礼
   - 说明瞻礼的功德很大，一瞻一礼就能得到大利益
   - 体现了恭敬心的重要性

3. **百返生于三十三天**：
   - 是人百返生于三十三天，永不堕于恶道
   - 百返：一百次，说明次数很多
   - 三十三天：忉利天，是欲界第二层天
   - 永不堕于恶道：永远不堕入恶道
   - 说明瞻礼的功德很大，能够获得天福，永不堕恶道
   - 体现了地藏菩萨的威神力

4. **转生的利益**：
   - 假如天福尽故，下生人间，犹为国王，不失大利
   - 天福尽故：如果天福尽了
   - 下生人间：下生到人间
   - 犹为国王：还会成为国王
   - 不失大利：不会失去大利益
   - 说明功德的持续，即使天福尽了，下生人间，还会成为国王
   - 体现了瞻礼功德的殊胜和持续

### 四、女人供养地藏菩萨的功德

**经文要点：**
- 若有女人，厌女人身，尽心供养地藏菩萨画像，及土石胶漆铜铁等像
- 如是日日不退，常以华香、饮食、衣服、缯彩、幢幡、钱、宝物等供养
- 是善女人，尽此一报女身，百千万劫，更不生有女人世界，何况复受
- 除非慈愿力故，要受女身，度脱众生，承斯供养地藏力故，及功德力，百千万劫不受女身

**讲解：**
1. **厌女身的因缘**：
   - 若有女人，厌女人身
   - 有些女人厌弃女身，想要转成男身
   - 这是修行的愿望，想要更好地修行
   - 说明女身在修行上有一些障碍，所以有些女人想要转成男身
   - 体现了修行的愿望和决心

2. **供养的方法**：
   - 尽心供养地藏菩萨画像，及土石胶漆铜铁等像
   - 如是日日不退：每天都不退转，说明精进的重要性
   - 常以华香、饮食、衣服、缯彩、幢幡、钱、宝物等供养
   - 华香：花和香
   - 饮食：食物和饮料
   - 衣服：衣服
   - 缯彩：丝绸和彩带
   - 幢幡：幢和幡
   - 钱、宝物：钱财和宝物
   - 说明供养要全面，要用各种物品供养
   - 显示精进的重要性，日日不退才能得到大利益

3. **转生的利益**：
   - 是善女人，尽此一报女身，百千万劫，更不生有女人世界，何况复受
   - 尽此一报女身：这一生是女身，之后就不再是女身了
   - 百千万劫，更不生有女人世界：百千万劫中，都不会生在有女人的世界
   - 何况复受：更不会再次受女身
   - 说明供养的功德很大，能够转成男身
   - 体现了地藏菩萨的威神力

4. **慈愿受女身**：
   - 除非慈愿力故，要受女身，度脱众生，承斯供养地藏力故，及功德力，百千万劫不受女身
   - 除非：除了
   - 慈愿力故：因为慈悲的愿力
   - 要受女身，度脱众生：如果要受女身，是为了度脱众生
   - 承斯供养地藏力故，及功德力：因为供养地藏菩萨的功德力
   - 百千万劫不受女身：百千万劫中都不会受女身
   - 说明即使有慈悲愿力要受女身度脱众生，但因为供养地藏菩萨的功德力，百千万劫中都不会受女身
   - 体现了供养地藏菩萨功德的殊胜

### 五、丑陋女人瞻礼的功德

**经文要点：**
- 复次普广，若有女人，厌是丑陋，多疾病者，但于地藏像前，志心瞻礼，食顷之间
- 是人千万劫中，所受生身，相貌圆满
- 是丑陋女人，如不厌女身，即百千万亿生中，常为王女，乃及王妃，宰辅大姓，大长者女，端正受生，诸相圆满
- 由志心故，瞻礼地藏菩萨，获福如是

**讲解：**
1. **丑陋的因缘**：
   - 若有女人，厌是丑陋，多疾病者：如果有女人，厌弃丑陋，多疾病
   - 有些女人丑陋，多疾病，这是业报
   - 说明丑陋和疾病是宿世的业报，可以通过修行改变
   - 体现了因果的对应性，造什么业，受什么报

2. **瞻礼的方法**：
   - 但于地藏像前，志心瞻礼，食顷之间：只要在地藏菩萨像前，至诚心瞻礼，一顿饭的时间
   - 志心：至诚心，说明恭敬心的重要性
   - 食顷之间：一顿饭的时间，说明时间很短，但功德很大
   - 说明瞻礼的方法很简单，只要至诚心瞻礼，就能得到大利益

3. **瞻礼的功德**：
   - 是人千万劫中，所受生身，相貌圆满：这个人千万劫中，所受生的身体，相貌圆满
   - 千万劫：时间极长，说明功德的持续
   - 相貌圆满：相貌圆满，说明功德的殊胜
   - 说明瞻礼的功德很大，能够改变相貌，获得圆满的相貌
   - 体现了地藏菩萨的威神力

4. **转生的利益**：
   - 是丑陋女人，如不厌女身，即百千万亿生中，常为王女，乃及王妃，宰辅大姓，大长者女，端正受生，诸相圆满
   - 如不厌女身：如果不厌弃女身
   - 百千万亿生中：百千万亿生中
   - 常为王女，乃及王妃：常常成为王女，甚至成为王妃
   - 宰辅大姓，大长者女：宰辅大姓，大长者的女儿
   - 端正受生，诸相圆满：端正受生，各种相都圆满
   - 说明即使不厌弃女身，也会得到大利益，常为王女、王妃等，端正受生
   - 体现了瞻礼功德的殊胜

5. **志心的重要性**：
   - 由志心故，瞻礼地藏菩萨，获福如是：因为至诚心的缘故，瞻礼地藏菩萨，获得这样的福报
   - 说明至诚心的重要性，至诚心是得到功德的基础
   - 体现了恭敬心的重要性

### 六、供养赞叹的功德

**经文要点：**
- 复次普广，若有善男子、善女人，能对菩萨像前，作诸伎乐，及歌咏赞叹，香华供养，乃至劝于一人多人
- 如是等辈，现在世中及未来世，常得百千鬼神日夜卫护，不令恶事辄闻其耳，何况亲受诸横

**讲解：**
1. **供养的方法**：
   - 能对菩萨像前，作诸伎乐，及歌咏赞叹：能在菩萨像前，作各种音乐，以及歌咏赞叹
   - 作诸伎乐：作各种音乐，用音乐供养
   - 歌咏赞叹：歌咏赞叹，用歌声赞叹地藏菩萨的功德
   - 香华供养：用香和华供养
   - 说明供养的方法很多，可以用音乐、歌声、香、华等供养

2. **劝化的功德**：
   - 乃至劝于一人多人：甚至劝一个人或多人
   - 说明不仅要自己供养，还要劝别人供养
   - 劝化的功德很大，能够帮助他人种下善根
   - 体现了度化众生的慈悲心

3. **护持的利益**：
   - 如是等辈，现在世中及未来世，常得百千鬼神日夜卫护：这样的人，现在世和未来世，常常得到百千鬼神日夜卫护
   - 百千鬼神：数量很多，说明护持的力量很大
   - 日夜卫护：白天和晚上都卫护，说明护持的持续
   - 说明供养赞叹的功德很大，能够得到鬼神的护持

4. **消除灾难**：
   - 不令恶事辄闻其耳，何况亲受诸横：不让恶事传到耳朵里，更何况亲自遭受各种横祸
   - 不令恶事辄闻其耳：不让恶事传到耳朵里，说明能够避免听到恶事
   - 何况亲受诸横：更何况亲自遭受各种横祸，说明能够避免遭受横祸
   - 说明供养赞叹的功德很大，能够消除灾难，避免横祸
   - 体现了地藏菩萨的护持力

### 七、讥毁的果报

**经文要点：**
- 复次普广，未来世中，若有恶人及恶神恶鬼，见有善男子、善女人，归敬供养赞叹瞻礼地藏菩萨形像
- 或妄生讥毁，谤无功德，及利益事，或露齿笑，或背面非，或劝人共非，或一人非，或多人非，乃至一念生讥毁者
- 如是之人，贤劫千佛灭度，讥毁之报，尚在阿鼻地狱受极重罪
- 过是劫已，方受饿鬼。又经千劫，复受畜生。又经千劫，方得人身
- 纵受人身，贫穷下贱，诸根不具，多被恶业来结其心。不久之间，复堕恶道
- 是故普广，讥毁他人供养，尚获此报，何况别生恶见毁灭

**讲解：**
1. **讥毁的行为**：
   - 若有恶人及恶神恶鬼，见有善男子、善女人，归敬供养赞叹瞻礼地藏菩萨形像：如果有恶人和恶神恶鬼，看到有善男子、善女人，归敬供养赞叹瞻礼地藏菩萨形像
   - 或妄生讥毁，谤无功德，及利益事：或者妄生讥毁，毁谤没有功德，以及利益的事情
   - 或露齿笑：或者露齿笑，嘲笑
   - 或背面非：或者背后非议
   - 或劝人共非：或者劝别人一起非议
   - 或一人非，或多人非：或者一个人非议，或者多人非议
   - 乃至一念生讥毁者：甚至一个念头生起讥毁
   - 说明讥毁的行为很多，包括嘲笑、非议、劝人非议等

2. **讥毁的严重性**：
   - 讥毁他人供养地藏菩萨，谤无功德，及利益事
   - 这是极重的罪业，因为伤害了众生的善根
   - 说明要尊重他人的修行，不能讥毁
   - 体现了因果的严厉性

3. **果报的严重**：
   - 如是之人，贤劫千佛灭度，讥毁之报，尚在阿鼻地狱受极重罪：这样的人，贤劫千佛都灭度了，讥毁的报应，还在阿鼻地狱受极重罪
   - 贤劫千佛：贤劫中的一千尊佛，说明时间极长
   - 尚在阿鼻地狱：还在阿鼻地狱，说明罪业的严重
   - 受极重罪：受极重的罪，说明痛苦的严重
   - 说明讥毁的果报极其严重，时间极长

4. **三恶道的果报**：
   - 过是劫已，方受饿鬼：过了这个劫，才受饿鬼的报应
   - 又经千劫，复受畜生：又经过千劫，再受畜生的报应
   - 又经千劫，方得人身：又经过千劫，才得到人身
   - 说明要经过很长时间，才能从三恶道中出来
   - 体现了因果的严厉性和持续性

5. **人身的果报**：
   - 纵受人身，贫穷下贱，诸根不具：即使得到人身，也是贫穷下贱，各种根都不具足
   - 多被恶业来结其心：多被恶业来束缚其心
   - 不久之间，复堕恶道：不久之间，又堕入恶道
   - 说明即使得到人身，也是贫穷下贱，而且还会再堕恶道
   - 体现了因果的严厉性

6. **别生恶见毁灭**：
   - 是故普广，讥毁他人供养，尚获此报，何况别生恶见毁灭：所以普广，讥毁他人供养，尚且获得这样的报应，更何况别生恶见毁灭
   - 别生恶见毁灭：另外生起恶见，毁灭佛法
   - 说明别生恶见毁灭的果报更严重
   - 体现了要尊重佛法，不能毁谤
