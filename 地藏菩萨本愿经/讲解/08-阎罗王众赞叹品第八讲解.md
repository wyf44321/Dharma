# 阎罗王众赞叹品第八讲解

## 章节概述

本品通过阎罗天子和各种鬼王的提问，以及佛陀的回答，说明了众生为什么容易堕入恶道，以及如何避免堕入恶道。特别强调了临终时的重要性，以及如何帮助临终的人。通过恶毒鬼王和主命鬼王的说明，展现了修善的功德和临终关怀的重要性。

## 主要内容讲解

### 一、鬼王集会

**经文要点：**
- 尔时铁围山内，有无量鬼王，与阎罗天子，俱诣忉利，来到佛所
- 所谓恶毒鬼王、多恶鬼王、大诤鬼王、白虎鬼王、血虎鬼王、赤虎鬼王、散殃鬼王、飞身鬼王、电光鬼王、狼牙鬼王、千眼鬼王、啖兽鬼王、负石鬼王、主耗鬼王、主祸鬼王、主食鬼王、主财鬼王、主畜鬼王、主禽鬼王、主兽鬼王、主魅鬼王、主产鬼王、主命鬼王、主疾鬼王、主险鬼王、三目鬼王、四目鬼王、五目鬼王、祁利失王、大祁利失王、祁利叉王、大祁利叉王、阿那吒王、大阿那吒王
- 如是等大鬼王，各各与百千诸小鬼王，尽居阎浮提，各有所执，各有所主
- 是诸鬼王与阎罗天子，承佛威神，及地藏菩萨摩诃萨力，俱诣忉利，在一面立

**讲解：**
1. **鬼王的种类**：
   - 铁围山内有无量鬼王，与阎罗天子一起来到忉利天宫
   - 鬼王的种类很多，包括恶毒鬼王、多恶鬼王、大诤鬼王、白虎鬼王、血虎鬼王、赤虎鬼王、散殃鬼王、飞身鬼王、电光鬼王、狼牙鬼王、千眼鬼王、啖兽鬼王、负石鬼王、主耗鬼王、主祸鬼王、主食鬼王、主财鬼王、主畜鬼王、主禽鬼王、主兽鬼王、主魅鬼王、主产鬼王、主命鬼王、主疾鬼王、主险鬼王、三目鬼王、四目鬼王、五目鬼王、祁利失王、大祁利失王、祁利叉王、大祁利叉王、阿那吒王、大阿那吒王等
   - 说明鬼王的种类繁多，各有不同的职责
   - 体现了业报的复杂性和多样性

2. **鬼王的职责**：
   - 各各与百千诸小鬼王，尽居阎浮提，各有所执，各有所主
   - 每个大鬼王都有百千个小鬼王
   - 各有所执：各有执掌的事务
   - 各有所主：各有主管的范围
   - 说明鬼王的职责分工明确，各司其职
   - 体现了业报的执行是有组织的，不是混乱的

3. **鬼王集会的原因**：
   - 是诸鬼王与阎罗天子，承佛威神，及地藏菩萨摩诃萨力，俱诣忉利
   - 说明鬼王能够来到忉利天宫，是因为佛陀的威神力和地藏菩萨的力量
   - 说明佛力的不可思议，能够让鬼王也来听法
   - 体现了佛陀的慈悲，连鬼王也要度化

### 二、阎罗天子的提问

**经文要点：**
- 尔时阎罗天子，胡跪合掌白佛言：世尊，我等今者与诸鬼王，承佛威神，及地藏菩萨摩诃萨力，方得诣此忉利大会，亦是我等获善利故
- 我观地藏菩萨在六道中，百千方便而度罪苦众生，不辞疲倦，是大菩萨有如是不可思议神通之事
- 然诸众生获脱罪报，未久之间，又堕恶道
- 世尊，是地藏菩萨既有如是不可思议神力，云何众生而不依止善道，永取解脱？唯愿世尊为我解说

**讲解：**
1. **阎罗天子的恭敬**：
   - 胡跪合掌：胡跪是单膝跪地，合掌表示恭敬
   - 说明阎罗天子对佛陀的恭敬
   - 体现了阎罗天子的谦逊和恭敬

2. **阎罗天子的感恩**：
   - 我等今者与诸鬼王，承佛威神，及地藏菩萨摩诃萨力，方得诣此忉利大会
   - 说明阎罗天子和鬼王能够来到忉利天宫，是因为佛陀的威神力和地藏菩萨的力量
   - 亦是我等获善利故：这也是我们获得善利的原因
   - 体现了阎罗天子的感恩之心

3. **阎罗天子的观察**：
   - 我观地藏菩萨在六道中，百千方便而度罪苦众生
   - 不辞疲倦：不辞辛苦，不辞疲倦
   - 是大菩萨有如是不可思议神通之事
   - 说明阎罗天子观察到地藏菩萨的慈悲和神通
   - 体现了地藏菩萨度化众生的广度和深度

4. **阎罗天子的疑问**：
   - 然诸众生获脱罪报，未久之间，又堕恶道
   - 说明众生虽然被地藏菩萨救度，但不久又堕入恶道
   - 是地藏菩萨既有如是不可思议神力，云何众生而不依止善道，永取解脱？
   - 为什么地藏菩萨有如此不可思议的神力，众生却不能依止善道，永远得到解脱？
   - 这是阎罗天子的疑问，也是很多人的疑问
   - 体现了阎罗天子对众生的关心

### 三、佛陀的回答

**经文要点：**
- 佛告阎罗天子：南阎浮提众生，其性刚强，难调难伏
- 是大菩萨，于百千劫，头头救拔如是众生，早令解脱
- 是罪报人乃至堕大恶趣，菩萨以方便力，拔出根本业缘，而遣悟宿世之事
- 自是阎浮众生结恶习重，旋出旋入，劳斯菩萨久经劫数而作度脱

**讲解：**
1. **众生的特性**：
   - 南阎浮提众生，其性刚强，难调难伏
   - 刚强：性格刚强，不容易改变
   - 难调难伏：难以调伏，难以降伏
   - 说明娑婆世界的众生性格刚强，不容易被度化
   - 体现了众生修行的困难
   - 说明需要长期的努力才能度化众生

2. **菩萨的救拔**：
   - 是大菩萨，于百千劫，头头救拔如是众生
   - 百千劫：经过百千劫的时间
   - 头头救拔：一次又一次地救拔
   - 早令解脱：早日让众生得到解脱
   - 说明地藏菩萨不辞辛苦，一次又一次地救拔众生
   - 体现了地藏菩萨的大慈悲和大愿力
   - 说明地藏菩萨的耐心和恒心

3. **菩萨的方便力**：
   - 是罪报人乃至堕大恶趣，菩萨以方便力，拔出根本业缘，而遣悟宿世之事
   - 即使众生堕入大恶趣，地藏菩萨也会用方便力救拔
   - 拔出根本业缘：拔出根本的业缘
   - 遣悟宿世之事：让众生了悟宿世的事情
   - 说明地藏菩萨会用各种善巧方便救度众生
   - 体现了地藏菩萨的智慧和慈悲

4. **众生的状况**：
   - 自是阎浮众生结恶习重，旋出旋入
   - 结恶习重：恶的习气很重
   - 旋出旋入：刚出来又进去，反复轮回
   - 劳斯菩萨久经劫数而作度脱：让地藏菩萨经过久远的劫数来度脱
   - 说明众生的恶习很重，容易反复
   - 体现了众生修行的困难
   - 说明需要长期的努力才能改变

### 四、迷路人的比喻

**经文要点：**
- 譬如有人迷失本家，误入险道
- 其险道中，多诸夜叉、及虎狼狮子、蚖蛇蝮蝎
- 如是迷人，在险道中，须臾之间，即遭诸毒
- 有一知识，多解大术，善禁是毒，乃及夜叉诸恶毒等
- 忽逢迷人欲进险道，而语之言：咄哉男子，为何事故而入此路，有何异术，能制诸毒
- 是迷路人忽闻是语，方知险道，即便退步，求出此路
- 是善知识，提携接手，引出险道，免诸恶毒。至于好道，令得安乐
- 而语之言：咄哉迷人，自今已后，勿履是道。此路入者，卒难得出，复损性命
- 是迷路人亦生感重。临别之时，知识又言：若见亲知，及诸路人，若男若女，言于此路多诸毒恶，丧失性命。无令是众自取其死

**讲解：**
1. **比喻的意义**：
   - 迷失本家：迷失本性，迷失了本来的家
   - 误入险道：误入危险的道路，比喻堕入恶道
   - 多诸夜叉、及虎狼狮子、蚖蛇蝮蝎：各种恶缘，各种危险
   - 说明众生迷失本性，容易堕入恶道
   - 体现了恶道的危险和恐怖

2. **险道的危险**：
   - 其险道中，多诸夜叉、及虎狼狮子、蚖蛇蝮蝎
   - 如是迷人，在险道中，须臾之间，即遭诸毒
   - 说明险道中充满各种危险
   - 须臾之间，即遭诸毒：很短的时间内就会遭受各种毒害
   - 体现了恶道的危险和恐怖
   - 说明堕入恶道的后果很严重

3. **善知识的作用**：
   - 有一知识，多解大术，善禁是毒，乃及夜叉诸恶毒等
   - 多解大术：懂得很多大法
   - 善禁是毒：善于禁制这些毒害
   - 说明善知识有能力帮助众生脱离险道
   - 体现了善知识的重要性

4. **善知识的劝化**：
   - 忽逢迷人欲进险道，而语之言：咄哉男子，为何事故而入此路，有何异术，能制诸毒
   - 善知识遇到迷路人要进入险道，立即劝阻
   - 说明善知识会主动帮助众生，不让众生堕入险道
   - 体现了善知识的慈悲和智慧

5. **迷路人的觉悟**：
   - 是迷路人忽闻是语，方知险道，即便退步，求出此路
   - 迷路人听到善知识的话，才知道是险道
   - 立即退步，求出此路：立即退步，想要离开险道
   - 说明众生如果遇到善知识，能够觉悟，能够改过
   - 体现了善知识教化的效果

6. **善知识的救度**：
   - 是善知识，提携接手，引出险道，免诸恶毒。至于好道，令得安乐
   - 提携接手：提携、接手，帮助迷路人
   - 引出险道：引导迷路人走出险道
   - 免诸恶毒：免除各种恶毒
   - 至于好道，令得安乐：到达好的道路，让迷路人得到安乐
   - 说明善知识会帮助众生脱离险道，到达好的道路
   - 体现了善知识的慈悲和智慧

7. **善知识的劝告**：
   - 而语之言：咄哉迷人，自今已后，勿履是道。此路入者，卒难得出，复损性命
   - 善知识劝告迷路人，以后不要再走这条路
   - 此路入者，卒难得出，复损性命：进入这条路的人，很难出来，还会损失性命
   - 说明善知识会劝告众生，不要再堕入恶道
   - 体现了善知识的慈悲

8. **迷路人的感恩**：
   - 是迷路人亦生感重：迷路人也生起感恩之心
   - 说明迷路人对善知识的感恩
   - 体现了善知识教化的效果

9. **善知识的嘱托**：
   - 临别之时，知识又言：若见亲知，及诸路人，若男若女，言于此路多诸毒恶，丧失性命。无令是众自取其死
   - 善知识在临别时，嘱托迷路人要告诉其他人，这条路很危险
   - 无令是众自取其死：不要让这些人自取死路
   - 说明善知识希望迷路人也能帮助其他人，不要让他们堕入险道
   - 体现了善知识的慈悲和智慧

### 五、地藏菩萨的救度

**经文要点：**
- 是故地藏菩萨具大慈悲，救拔罪苦众生，生人天中，令受妙乐
- 是诸罪众，知业道苦，脱得出离，永不再历
- 如迷路人，误入险道，遇善知识引接令出，永不复入
- 逢见他人，复劝莫入
- 自言因是迷故，得解脱竟，更不复入
- 若再履践，犹尚迷误，不觉旧曾所落险道，或致失命
- 如堕恶趣，地藏菩萨方便力故，使令解脱，生人天中。旋又再入，若业结重，永处地狱，无解脱时

**讲解：**
1. **地藏菩萨的救度**：
   - 是故地藏菩萨具大慈悲，救拔罪苦众生，生人天中，令受妙乐
   - 具大慈悲：具有大慈悲心
   - 救拔罪苦众生：救拔罪苦的众生
   - 生人天中，令受妙乐：让众生生到人天中，享受妙乐
   - 说明地藏菩萨会用大慈悲心救拔罪苦众生
   - 体现了地藏菩萨的大慈悲和大愿力

2. **众生的觉悟**：
   - 是诸罪众，知业道苦，脱得出离，永不再历
   - 知业道苦：知道业道的痛苦
   - 脱得出离：脱离业道，得到出离
   - 永不再历：永远不再经历
   - 说明有些众生能够觉悟，知道业道的痛苦，能够改过
   - 体现了众生改过的可能性

3. **改过的表现**：
   - 如迷路人，误入险道，遇善知识引接令出，永不复入
   - 逢见他人，复劝莫入：遇到其他人，也劝告他们不要进入险道
   - 自言因是迷故，得解脱竟，更不复入：自己说因为迷失的缘故，得到解脱后，更不再进入
   - 说明改过的众生会劝告其他人，不要堕入恶道
   - 体现了改过的效果

4. **再次堕入的危险**：
   - 若再履践，犹尚迷误，不觉旧曾所落险道，或致失命
   - 若再履践：如果再次踏上险道
   - 犹尚迷误：还是迷误
   - 不觉旧曾所落险道：不觉察以前曾经落过的险道
   - 或致失命：可能会导致失去性命
   - 说明如果再次堕入恶道，可能会更加危险
   - 体现了再次堕入的严重性

5. **再次堕入的后果**：
   - 如堕恶趣，地藏菩萨方便力故，使令解脱，生人天中。旋又再入，若业结重，永处地狱，无解脱时
   - 如果再次堕入恶趣，地藏菩萨会用方便力救度，让众生生到人天中
   - 旋又再入：很快又再次进入
   - 若业结重：如果业力结得很重
   - 永处地狱，无解脱时：永远处在地狱中，没有解脱的时候
   - 说明如果反复堕入恶道，业力会越来越重，最终可能永远处在地狱中
   - 体现了反复堕入的严重后果

### 六、恶毒鬼王的说明

**经文要点：**
- 尔时恶毒鬼王合掌恭敬白佛言：世尊，我等诸鬼王，其数无量，在阎浮提，或利益人，或损害人，各各不同
- 然是业报，使我眷属游行世界，多恶少善
- 过人家庭，或城邑聚落，庄园房舍
- 或有男子女人，修毛发善事，乃至悬一幡一盖，少香少华，供养佛像及菩萨像
- 或转读尊经，烧香供养一句一偈
- 我等鬼王敬礼是人，如过去现在未来诸佛
- 敕诸小鬼，各有大力，及土地分，便令卫护，不令恶事横事、恶病横病，乃至不如意事，近于此舍等处，何况入门
- 佛赞鬼王：善哉，善哉！汝等及与阎罗，能如是拥护善男女等，吾亦告梵王帝释，令卫护汝

**讲解：**
1. **鬼王的作用**：
   - 我等诸鬼王，其数无量，在阎浮提，或利益人，或损害人，各各不同
   - 其数无量：数量无量，无法计算
   - 或利益人，或损害人：有的利益人，有的损害人
   - 各各不同：各不相同
   - 说明鬼王的作用不同，有的是执行善报，有的是执行恶报
   - 体现了业报的执行是有组织的

2. **业报的执行**：
   - 然是业报，使我眷属游行世界，多恶少善
   - 然是业报：这是业报的缘故
   - 使我眷属游行世界：让我的眷属游行世界
   - 多恶少善：多恶少善，说明恶业多，善业少
   - 说明鬼王是执行业报的，众生的恶业多，所以鬼王多执行恶报
   - 体现了因果的对应性

3. **修善的功德**：
   - 或有男子女人，修毛发善事，乃至悬一幡一盖，少香少华，供养佛像及菩萨像
   - 或转读尊经，烧香供养一句一偈
   - 修毛发善事：修很小的善事，像毛发一样小
   - 悬一幡一盖：悬挂一个幡、一个盖
   - 少香少华：很少的香、很少的花
   - 供养佛像及菩萨像：供养佛像和菩萨像
   - 转读尊经：转读佛经
   - 烧香供养一句一偈：烧香供养一句一偈
   - 说明即使是很小的善事，也能得到护持
   - 体现了修善的功德，即使是很小的善事也有功德
