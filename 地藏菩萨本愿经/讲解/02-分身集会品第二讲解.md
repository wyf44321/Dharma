# 分身集会品第二讲解

## 章节概述

本品讲述了地藏菩萨的分身从十方地狱处集会到忉利天宫，以及佛陀将娑婆世界众生付嘱给地藏菩萨的故事。通过佛陀的嘱托和地藏菩萨的承诺，展现了地藏菩萨度化众生的决心和愿力。

## 主要内容讲解

### 一、分身集会

**经文要点：**
- 百千万亿不可思议世界的地狱处分身地藏菩萨，俱来集在忉利天宫
- 各以方面，与诸得解脱从业道出者，共持香华，来供养佛
- 这些众生因地藏菩萨教化，永不退转于阿耨多罗三藐三菩提

**讲解：**
1. **分身的意义**：
   - 地藏菩萨能分身无数，显示其神通力不可思议
   - 每个世界都有地藏菩萨在度化众生
   - 说明地藏菩萨的愿力遍及一切处

2. **得解脱的众生**：
   - 千万亿那由他数：数量极多
   - 从业道出：从恶道中解脱
   - 因地藏菩萨教化：说明地藏菩萨度化的效果

3. **永不退转**：
   - 不退转于无上正等正觉
   - 说明地藏菩萨教化的力量
   - 这些众生已经走上成佛的道路

### 二、佛陀的嘱托

**经文要点：**
- 佛陀舒金色臂，摩地藏菩萨顶
- 佛陀在五浊恶世教化刚强众生，十有一二尚恶习在
- 佛陀分身千百亿，广设方便度脱众生
- 佛陀将娑婆世界众生付嘱给地藏菩萨

**讲解：**
1. **摩顶的意义**：
   - 表示佛陀对地藏菩萨的信任和嘱托
   - 是付嘱的重要仪式
   - 显示地藏菩萨承担的重任

2. **刚强众生**：
   - 五浊恶世的众生刚强难化
   - 佛陀教化，十有一二尚恶习在
   - 说明度化众生的困难

3. **佛陀的分身**：
   - 分身千百亿：显示佛陀的慈悲
   - 或现男子身、或现女人身：随类现身
   - 或现天龙身、或现神鬼身：以各种身份度化
   - 说明度化众生需要善巧方便

4. **付嘱的内容**：
   - 时间：至弥勒出世已来
   - 对象：娑婆世界众生
   - 要求：悉使解脱，永离诸苦，遇佛授记

### 三、地藏菩萨的承诺

**经文要点：**
- 诸世界分身地藏菩萨，共复一形
- 涕泪哀恋，白其佛言
- 从久远劫来，蒙佛接引，获不可思议神力
- 分身遍满百千万亿恒河沙世界
- 每一世界化百千万亿身，每一身度百千万亿人
- 三白佛言：唯愿世尊，不以后世恶业众生为虑

**讲解：**
1. **共复一形**：
   - 无数分身合为一体
   - 显示地藏菩萨的统一性
   - 说明愿力的一致

2. **涕泪哀恋**：
   - 表达对佛陀的感恩
   - 表达对众生的悲悯
   - 显示地藏菩萨的慈悲心

3. **度化的范围**：
   - 遍满百千万亿恒河沙世界
   - 每一世界化百千万亿身
   - 每一身度百千万亿人
   - 说明地藏菩萨度化的广度

4. **度化的方法**：
   - 令归敬三宝
   - 永离生死
   - 至涅槃乐
   - 一毛一渧，一沙一尘，渐度脱之

5. **三白佛言**：
   - 三次向佛承诺
   - 表示决心坚定
   - 请佛放心

### 四、佛陀的赞叹

**经文要点：**
- 佛赞地藏菩萨：善哉！善哉！
- 汝能成就久远劫来，发弘誓愿，广度将毕，即证菩提

**讲解：**
1. **善哉善哉**：
   - 佛陀的赞叹
   - 表示认可地藏菩萨的承诺
   - 显示地藏菩萨的功德

2. **即证菩提**：
   - 广度将毕：度化众生的事业即将完成
   - 即证菩提：即将成佛
   - 说明地藏菩萨的功德即将圆满

## 教义要点

1. **地藏菩萨的愿力**：
   - 分身无数，遍及一切世界
   - 度化众生，永不退转
   - 愿力坚固，不可思议

2. **佛陀的慈悲**：
   - 分身千百亿度化众生
   - 随类现身，善巧方便
   - 将众生付嘱给地藏菩萨

3. **度化的方法**：
   - 令归敬三宝
   - 永离生死
   - 至涅槃乐
   - 渐度脱之

4. **付嘱的意义**：
   - 佛陀将众生付嘱给地藏菩萨
   - 地藏菩萨承担度化众生的重任
   - 显示地藏菩萨的重要性

## 修行启示

1. **学习地藏菩萨的愿力**：
   - 要发大愿度化众生
   - 愿力要坚固，不能退转
   - 要遍及一切处，不遗漏

2. **学习佛陀的慈悲**：
   - 要以慈悲心度化众生
   - 要善巧方便，随类现身
   - 要不辞辛苦，广设方便

3. **归敬三宝**：
   - 要归依佛、法、僧三宝
   - 这是修行的基础
   - 是成佛的根本

4. **渐度脱之**：
   - 度化众生要循序渐进
   - 不要急于求成
   - 要有耐心和恒心

5. **承担重任**：
   - 要学习地藏菩萨，承担度化众生的重任
   - 要有责任感
   - 要有使命感

## 现代意义

1. **愿力的重要性**：
   - 无论做什么事，都要有坚定的愿力
   - 愿力是成功的根本
   - 愿力能感通一切

2. **慈悲心的培养**：
   - 要培养慈悲心
   - 要关心他人，帮助他人
   - 要有奉献精神

3. **善巧方便**：
   - 度化众生要善巧方便
   - 要根据不同情况采取不同方法
   - 要灵活变通

4. **责任感的培养**：
   - 要有责任感
   - 要承担应承担的责任
   - 要有使命感

5. **渐进的修行**：
   - 修行要循序渐进
   - 不要急于求成
   - 要有耐心和恒心
