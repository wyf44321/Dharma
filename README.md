# 佛教经典电子书合集

## 项目简介

本项目收集整理了大乘佛教的重要经典，以 Markdown 格式呈现，便于阅读、学习和传播。所有经文均采用统一的格式和结构，包含完整的经文内容、章节导航和回向文。

## 包含经典

### 1. [地藏菩萨本愿经](./地藏菩萨本愿经/)

《地藏菩萨本愿经》，又称《地藏经》，是佛教重要经典之一。本经主要记载了地藏菩萨在释迦牟尼佛入灭后、弥勒佛未出世前，发愿救度一切众生的事迹。全经共分十三品，详细阐述了地藏菩萨的大愿、地狱的种种状况、因果报应的法则，以及供养地藏菩萨的功德利益。

**核心思想**：众生度尽，方证菩提；地狱不空，誓不成佛

**章节数**：13品

**查看详情**：[地藏菩萨本愿经 README](./地藏菩萨本愿经/README.md)

---

### 2. [妙法莲华经](./妙法莲华经/)

《妙法莲华经》，简称《法华经》，是大乘佛教的重要经典之一，被誉为"经中之王"。本经由释迦牟尼佛在灵鹫山所说，全经共二十八品，主要阐述"一佛乘"的思想，说明一切众生皆可成佛，三乘（声闻、缘觉、菩萨）最终都归于一佛乘。

**核心思想**：一佛乘、开权显实、会三归一、佛性本具

**章节数**：28品

**查看详情**：[妙法莲华经 README](./妙法莲华经/README.md)

---

### 3. [金刚般若波罗蜜经](./金刚经/)

《金刚般若波罗蜜经》，简称《金刚经》，是大乘佛教的重要经典之一，属于《大般若经》的第九会。本经由释迦牟尼佛在舍卫国祇树给孤独园为须菩提等弟子所说，全经共三十二分，主要阐述般若空性的智慧，教导众生如何破除我相、人相、众生相、寿者相等执著，证得无上正等正觉。

**核心思想**：般若空性、无相无住、破四相、中道实相

**章节数**：32分

**查看详情**：[金刚经 README](./金刚经/README.md)

---

### 4. [大方广圆觉修多罗了义经](./圆觉经/)

《大方广圆觉修多罗了义经》，简称《圆觉经》，是大乘佛教的重要经典之一。本经由释迦牟尼佛在神通大光明藏三昧中，为文殊师利、普贤、普眼等十二位大菩萨所说，全经共十二章，主要阐述圆觉妙心的清净本性，以及如何修习圆觉法门，破除无明，证得圆觉。

**核心思想**：圆觉妙心、无明如幻、破四相、三种法门

**章节数**：12章

**查看详情**：[圆觉经 README](./圆觉经/README.md)

## 使用说明

### 阅读方式

1. **在线阅读**：使用任何支持 Markdown 的阅读器打开文件
2. **本地阅读**：下载后使用 Markdown 编辑器或阅读器打开
3. **电子书转换**：可以将 Markdown 文件转换为 PDF、EPUB 等格式

### 文件格式

- 所有经文文件均为 Markdown 格式（`.md`）
- 每个章节文件包含：
  - 上一章节和下一章节的导航链接
  - 完整的经文内容
  - 统一的格式和结构

### 讲解文档

- 部分经典包含详细的讲解文档，位于各经典的 `讲解/` 目录下
- 每个讲解文档包含：
  - 章节概述
  - 主要内容讲解
  - 教义要点
  - 修行启示
  - 现代意义
- 建议先阅读原经文，再对照讲解文档深入学习

### 阅读建议

1. **从开经偈开始**：每个经典都从"香赞开经偈"开始
2. **按顺序阅读**：建议按照章节顺序完整阅读
3. **对照讲解**：如有讲解文档，可对照学习，深入理解
4. **以回向文结束**：阅读完毕后念诵回向文

## 经典特点

### 地藏菩萨本愿经
- **重点**：因果报应、地狱救度、超度亡灵
- **适用**：为亡者超度、消除业障、增长福德
- **特色**：详细描述地狱状况，说明各种恶业果报

### 妙法莲华经
- **重点**：一佛乘思想、成佛因缘、佛性本具
- **适用**：修学大乘佛法、发菩提心、成就佛道
- **特色**：以譬喻和故事说明深奥义理，被誉为"经中之王"

### 金刚般若波罗蜜经
- **重点**：般若空性、破执著、无相无住
- **适用**：破除我执、开启智慧、证悟空性
- **特色**：以问答形式阐述空性智慧，文字简洁而义理深奥

### 大方广圆觉修多罗了义经
- **重点**：圆觉妙心、破除无明、破四相、三种法门
- **适用**：修习圆觉法门、破除四相、证得圆觉
- **特色**：以十二位大菩萨请问的形式，系统阐述圆觉修行的完整体系
