# 无断无灭分第二十七讲解

## 章节概述

本章佛陀开示如来不以具足相故，得阿耨多罗三藐三菩提。说明发阿耨多罗三藐三菩提心者，于法不说断灭相。这是破除对断灭的执著，说明中道的重要性，体现了《金刚经》无相修行的核心教义。

## 主要内容讲解

### 一、不以具足相得菩提

**经文要点：**
- 须菩提。汝若作是念。如来不以具足相故。得阿耨多罗三藐三菩提
- 须菩提。莫作是念
- 如来不以具足相故。得阿耨多罗三藐三菩提

**讲解：**
1. **"汝若作是念。如来不以具足相故。得阿耨多罗三藐三菩提"**：
   - 如果你有这样的想法：如来不用具足相，得到无上正等正觉
   - 说明如果认为如来不用具足相就能得菩提，这是错误的
   - 体现了无相修行的关键，要了知中道，不能落于断灭
   - 说明要了知无我，要了知中道
   - 这是修行的关键，要了知中道

2. **"莫作是念"**：
   - 不要这样想
   - 说明不要认为如来不用具足相就能得菩提
   - 体现了无相修行的关键，要了知中道，不能落于断灭
   - 说明要了知无我，要了知中道
   - 这是修行的关键，要了知中道

3. **"如来不以具足相故。得阿耨多罗三藐三菩提"**：
   - 如来不用具足相，得到无上正等正觉
   - 说明如来确实不用具足相就能得菩提，但这不是断灭
   - 体现了无相修行的关键，要了知中道，不能落于断灭
   - 说明要了知无我，要了知中道
   - 这是修行的关键，要了知中道

4. **中道的意义**：
   - 如来不用具足相就能得菩提，但这不是断灭
   - 说明要了知中道，不能落于有相，也不能落于断灭
   - 体现了无相修行的关键，要了知中道
   - 说明要了知无我，要了知中道
   - 这是修行的关键，要了知中道

### 二、不说断灭相

   - 这是修行的关键，要了知中道

2. **"莫作是念"**：

   - 说明不要认为发菩提心的人说一切法都断灭

   - 这是修行的关键，要了知中道

3. **"发阿耨多罗三藐三菩提心者。于法不说断灭相"**：
   - 发无上正等正觉心的人，对于法不说断灭相
   - 说明发菩提心的人不说断灭相，要说中道
   - 体现了无相修行的关键，要了知中道，不能落于断灭
   - 说明要了知无我，要了知中道
   - 这是修行的关键，要了知中道

4. **不说断灭相的意义**：
