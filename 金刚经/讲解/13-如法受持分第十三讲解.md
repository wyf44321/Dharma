# 如法受持分第十三讲解

## 章节概述

本章佛陀开示此经名为《金刚般若波罗蜜》，应如是受持。说明佛说般若波罗蜜，即非般若波罗蜜，是名般若波罗蜜。并说明受持此经的功德，超过用恒河沙等身命布施。这是说明如何受持此经，以及受持此经的功德。

## 主要内容讲解

### 一、经名

**经文要点：**
- 尔时须菩提白佛言。世尊。当何名此经。我等云何奉持
- 佛告须菩提。是经名为金刚般若波罗蜜
- 以是名字。汝当奉持

**讲解：**
1. **"当何名此经"**：
   - 须菩提问：这部经应该叫什么名字？
   - 说明须菩提想知道经名，以便受持
   - 体现了须菩提的恭敬，想知道如何受持
   - 说明受持需要知道经名，这是受持的基础

2. **"我等云何奉持"**：
   - 须菩提问：我们应该如何奉持这部经？
   - 说明须菩提想知道如何受持，如何实践
   - 体现了须菩提的恭敬，想知道如何实践
   - 说明受持需要知道方法，这是受持的关键

3. **"是经名为金刚般若波罗蜜"**：
   - 佛陀说：这部经的名字是《金刚般若波罗蜜》
   - "金刚"表示坚固，能够破除一切烦恼
   - "般若"表示智慧，是了知空性的智慧
   - "波罗蜜"表示到彼岸，是度脱生死的方法
   - 说明经名的含义，体现了这部经的殊胜
   - 体现了这部经的重要性，是度脱生死的法宝

4. **"以是名字。汝当奉持"**：
   - 佛陀说：以这个名字，你们应该奉持
   - 说明要以《金刚般若波罗蜜》这个名字来受持
   - 体现了受持的方法，要以经名来受持
   - 说明受持要恭敬，要如法受持

### 二、即非般若波罗蜜

**经文要点：**
- 所以者何。须菩提
- 佛说般若波罗蜜。即非般若波罗蜜
- 是名般若波罗蜜

**讲解：**
1. **"所以者何"**：
   - 佛陀问"所以者何"，就是要解释为什么
   - 说明下面要解释原因，让须菩提理解
   - 体现了佛陀的善巧方便，会解释原因

2. **"佛说般若波罗蜜。即非般若波罗蜜"**：
   - 佛陀说的般若波罗蜜，不是真正的般若波罗蜜
   - 说明般若波罗蜜只是假名，不是真实的
   - 体现了空性的智慧，一切法都是空的
   - 说明不要执著于名字，要超越语言文字

3. **"是名般若波罗蜜"**：
   - 只是名字叫般若波罗蜜
   - 说明般若波罗蜜只是假名，不是真实的
   - 体现了空性的智慧，一切法都是空的
   - 说明不要执著于名字，要超越语言文字

### 三、如来无所说

**经文要点：**

- 须菩提白佛言。世尊。如来无所说

**讲解：**
1. **"如来有所说法不"**：
   - 佛陀问：如来是否有所说法？
   - 说明佛陀要破除对说法的执著
   - 体现了佛陀的善巧方便，通过提问来破除执著

2. **"如来无所说"**：
   - 须菩提回答：如来没有说法
   - 说明如来虽然说了很多法，但实际上没有说法
   - 体现了空性的智慧，说法也是空的
   - 说明不要执著于说法，要超越语言文字

3. **说而无说**：
   - 如来虽然说了很多法，但实际上没有说法
   - 说明说而无说，这是如来的境界
   - 体现了空性的智慧，一切法都是空的
   - 说明要证得无相，要超越语言文字

### 四、微尘和世界

- 须菩提言。甚多。世尊
- 须菩提。诸微尘。如来说非微尘。是名微尘
- 如来说世界。非世界。是名世界

**讲解：**
1. **"三千大千世界所有微尘。是为多不"**：
   - 佛陀问：三千大千世界所有的微尘，是不是很多？
   - 三千大千世界是一个佛的教化范围，非常大
   - 微尘是极小的物质，数量非常多
   - 说明佛陀用微尘来比喻，让须菩提理解

2. **"甚多。世尊"**：
   - 须菩提回答：非常多
   - 说明须菩提理解了，知道微尘非常多
   - 体现了须菩提的智慧，能够理解佛陀的比喻

3. **"诸微尘。如来说非微尘。是名微尘"**：
   - 如来说的微尘，不是真正的微尘，只是名字叫微尘
   - 说明微尘只是假名，不是真实的
   - 体现了空性的智慧，一切法都是空的
   - 说明不要执著于微尘，要超越一切相

4. **"如来说世界。非世界。是名世界"**：
   - 如来说的世界，不是真正的世界，只是名字叫世界
   - 说明世界只是假名，不是真实的
   - 体现了空性的智慧，一切法都是空的
   - 说明不要执著于世界，要超越一切相

### 五、三十二相

**经文要点：**
- 须菩提。于意云何。可以三十二相见如来不
- 不也。世尊。不可以三十二相得见如来
- 何以故。如来说三十二相。即是非相。是名三十二相

**讲解：**
1. **"可以三十二相见如来不"**：
   - 佛陀问：可以用三十二相见到如来吗？
   - 三十二相是如来的三十二种相好，是如来的庄严相
   - 说明佛陀要破除对三十二相的执著

2. **"不也。世尊。不可以三十二相得见如来"**：
   - 须菩提回答：不可以用三十二相见到如来
   - 说明须菩提理解了，知道不可以用三十二相见如来
   - 体现了须菩提的智慧，能够理解佛陀的深意

3. **"如来说三十二相。即是非相。是名三十二相"**：
   - 如来说的三十二相，不是真正的相，只是名字叫三十二相
   - 说明三十二相只是假名，不是真实的
   - 体现了空性的智慧，一切相都是空的
   - 说明不要执著于三十二相，要超越一切相

### 六、受持的功德

**经文要点：**
- 须菩提。若有善男子。善女人。以恒河沙等身命布施
- 若复有人。于此经中。乃至受持四句偈等。为他人说
- 其福甚多

**讲解：**
1. **"以恒河沙等身命布施"**：
   - 用恒河沙等身命布施，就是用无数生命来布施
   - 恒河沙是极多的，无法计算
   - 身命布施是最殊胜的布施，功德很大
   - 说明佛陀用身命布施来比较，显示受持此经的功德

2. **"于此经中。乃至受持四句偈等"**：
   - 对于这部经，哪怕只受持四句偈
   - 四句偈是极少的，但功德很大
   - 说明受持此经的功德很大，哪怕只受持四句偈
   - 体现了受持此经的殊胜，功德不可思议

3. **"为他人说"**：
   - 为他人解说，就是为他人说这部经
   - 说明为他人说此经的功德更大
   - 体现了说法的功德，为他人说功德更大
   - 说明要受持此经，要为他人说

4. **"其福甚多"**：
   - 福德非常多，超过身命布施
   - 说明受持此经的功德很大，超过身命布施
   - 体现了受持此经的殊胜，功德不可思议
