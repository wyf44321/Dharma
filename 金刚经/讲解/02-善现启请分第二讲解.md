# 善现启请分第二讲解

## 章节概述

本章须菩提（善现）请问如何降伏其心、如何安住其心。这是《金刚经》的核心问题，也是所有修行者都要面对的根本问题。佛陀的回答奠定了整部经的基础。本章通过须菩提的请问，揭示了《金刚经》要解决的根本问题：如何安住其心，如何降伏其心。这是所有修行者都要面对的问题，也是成佛的关键。

## 主要内容讲解

### 一、须菩提的请问

**经文要点：**
- 时长老须菩提。在大众中。即从座起
- 偏袒右肩。右膝著地。合掌恭敬
- 而白佛言。希有世尊。如来善护念诸菩萨。善付嘱诸菩萨

**讲解：**
1. **"时长老须菩提"**：
   - 须菩提是佛陀的十大弟子之一，解空第一
   - "长老"表示须菩提德高望重，是僧团中的长者
   - 须菩提又名"善现"，因为他善于显现空性的道理
   - 须菩提是解空第一，对空性有深刻的理解
   - 说明由须菩提来请问《金刚经》是合适的，因为《金刚经》主要讲空性
   - 体现了因缘的殊胜，解空第一的弟子请问空性的经典

2. **"在大众中。即从座起"**：
   - 须菩提在大众中，从座位上站起来
   - 说明须菩提很恭敬，不敢坐着问法
   - 体现了对法的尊重和对佛陀的恭敬
   - 说明问法要有恭敬心，不能轻慢
   - 体现了修行的态度，要恭敬三宝

3. **"偏袒右肩。右膝著地。合掌恭敬"**：
   - 这是古印度最恭敬的礼仪
   - "偏袒右肩"：把右肩露出来，表示恭敬和至诚
   - "右膝著地"：右膝跪在地上，表示恭敬和谦逊
   - "合掌恭敬"：双手合掌，表示恭敬和至诚
   - 说明须菩提问法非常恭敬，体现了对法的尊重
   - 体现了修行的态度，要恭敬三宝，不能轻慢
   - 说明问法要有至诚心，才能得到佛的加持

4. **"希有世尊"**：
   - "希有"表示稀有、难得，说明佛陀的功德稀有难得
   - 须菩提赞叹佛陀的功德，说明他对佛陀有深刻的了解
   - 体现了须菩提的智慧，能够认识到佛陀的功德
   - 说明问法前要先赞叹，这是问法的礼仪
   - 体现了对佛陀的恭敬和感恩

5. **"如来善护念诸菩萨。善付嘱诸菩萨"**：
   - "善护念"：善于护持和忆念，说明佛陀善于护持菩萨，让菩萨不退转
   - "善付嘱"：善于付嘱和教导，说明佛陀善于教导菩萨，让菩萨能够度化众生
   - 须菩提先赞叹佛陀的功德，然后才问问题
   - 说明须菩提很懂礼仪，知道问法要先赞叹
   - 体现了须菩提的智慧和对佛陀的恭敬
   - 说明佛陀确实善于护念和付嘱菩萨，这是佛陀的功德

### 二、核心问题

**经文要点：**
- 世尊。善男子。善女人。发阿耨多罗三藐三菩提心
- 应云何住。云何降伏其心

**讲解：**
1. **"善男子。善女人"**：
   - 善男子、善女人是指有善根、有福德的人
   - 不是指性别，而是指有善根的人
   - 说明《金刚经》是为有善根的人说的
   - 体现了《金刚经》的殊胜，需要有善根才能听闻
   - 说明我们要培养善根，才能听闻和理解《金刚经》

2. **"发阿耨多罗三藐三菩提心"**：
   - "阿耨多罗三藐三菩提"是梵语，意思是无上正等正觉，就是成佛
   - "发阿耨多罗三藐三菩提心"就是发成佛的心，发菩提心
   - 这是修行的基础，没有菩提心就不能成佛
   - 说明《金刚经》是为发菩提心的人说的
   - 体现了《金刚经》的殊胜，需要有菩提心才能理解
   - 说明我们要发菩提心，才能修学《金刚经》

3. **"应云何住"**：
   - "应云何住"就是心应该安住在哪里
   - 这是修行的核心问题，心不安住就会散乱
   - 说明修行要找到心的安住处，不能散乱
   - 体现了修行的关键，要安住其心
   - 说明我们要找到心的安住处，才能修行

4. **"云何降伏其心"**：
   - "云何降伏其心"就是如何降伏妄心
   - 这是修行的核心问题，妄心不降伏就不能修行
   - 说明修行要降伏妄心，不能随妄心转
   - 体现了修行的关键，要降伏其心
   - 说明我们要降伏妄心，才能修行

5. **两个问题的关系**：
   - "应云何住"和"云何降伏其心"是两个问题，但实际上是相关的
   - 心不安住就会散乱，散乱就是妄心
   - 降伏妄心就是让心安住
   - 说明这两个问题实际上是一个问题的两个方面
   - 体现了修行的整体性，不能分开来看
   - 说明我们要同时解决这两个问题，才能修行

### 三、佛陀的回答

**经文要点：**
- 佛言。善哉善哉。须菩提
- 如汝所说。如来善护念诸菩萨。善付嘱诸菩萨
- 汝今谛听。当为汝说
- 善男子。善女人。发阿耨多罗三藐三菩提心
- 应如是住。如是降伏其心

**讲解：**
1. **"善哉善哉"**：
   - 佛陀连说两个"善哉"，表示非常赞叹
   - 说明须菩提的问题问得很好，问到了关键
   - 体现了佛陀的欢喜，因为须菩提问到了修行的核心问题
   - 说明这个问题很重要，是修行的关键
   - 体现了《金刚经》的重要性，要解决这个根本问题

2. **"如汝所说"**：
   - 佛陀肯定须菩提的赞叹，说"如汝所说"
   - 说明佛陀确实善于护念和付嘱菩萨
   - 体现了佛陀的谦逊，虽然功德圆满，但仍然肯定弟子的赞叹
   - 说明佛陀很慈悲，会肯定弟子的善行
   - 体现了佛陀的智慧，知道如何引导弟子

3. **"汝今谛听。当为汝说"**：
   - "谛听"就是仔细听，认真听
   - 佛陀要求须菩提仔细听，说明下面要说的内容很重要
   - 体现了佛陀的慈悲，愿意详细解答
   - 说明听闻佛法要仔细，不能散乱
   - 体现了修行的态度，要谛听，要深入理解

4. **"应如是住。如是降伏其心"**：
   - 佛陀说"应如是住。如是降伏其心"
   - "如是"就是"这样"，但具体怎么"这样"，佛陀会在后面详细说明
   - 说明佛陀的回答是总说，后面会详细展开
   - 体现了佛陀的善巧方便，先总说，后详说
   - 说明我们要仔细听闻，才能理解"如是"的含义
   - 体现了《金刚经》的深奥，需要深入理解

5. **"唯然。世尊。愿乐欲闻"**：
   - 须菩提说"唯然。世尊。愿乐欲闻"
   - "唯然"表示恭敬地答应
   - "愿乐欲闻"表示很愿意听闻，很欢喜听闻
   - 说明须菩提很恭敬，很欢喜，很想听闻
   - 体现了须菩提的至诚心，对法的渴求
   - 说明我们要有至诚心，才能听闻和理解佛法
