# 一体同观分第十八讲解

## 章节概述

本章佛陀开示如来有五眼（肉眼、天眼、慧眼、法眼、佛眼），能知一切众生的心。说明如恒河中所有沙，一恒河中所有沙有如是沙等恒河，是诸恒河所有沙数佛世界如是，如来悉知尔所国土中所有众生若干种心。说明如来说诸心，皆为非心，是名为心。过去心不可得，现在心不可得，未来心不可得。这是说明心的无相性，以及如来智慧的无量无边。

## 主要内容讲解

### 一、如来的五眼

**经文要点：**
- 须菩提。于意云何。如来有肉眼不。如是。世尊。如来有肉眼
- 须菩提。于意云何。如来有天眼不。如是。世尊。如来有天眼
- 须菩提。于意云何。如来有慧眼不。如是。世尊。如来有慧眼
- 须菩提。于意云何。如来有法眼不。如是。世尊。如来有法眼
- 须菩提。于意云何。如来有佛眼不。如是。世尊。如来有佛眼

**讲解：**
1. **"如来有肉眼"**：
   - 如来有肉眼，能见一切色相
   - 肉眼是凡夫的眼睛，能见可见的色相
   - 说明如来虽然成佛，但仍然有肉眼，能见一切色相
   - 体现了如来的圆满，具有一切眼
   - 说明如来不是离开色相，而是能见一切色相但不执著

2. **"如来有天眼"**：
   - 如来有天眼，能见一切远近、粗细、明暗
   - 天眼是天人的眼睛，能见更远的距离和更细微的东西
   - 说明如来有天眼，能见一切远近、粗细、明暗
   - 体现了如来的圆满，具有天眼
   - 说明如来能见一切，不受距离和粗细的限制

3. **"如来有慧眼"**：
   - 如来有慧眼，能见一切法的空性
   - 慧眼是声闻、缘觉的眼睛，能见一切法的空性
   - 说明如来有慧眼，能见一切法的空性
   - 体现了如来的智慧，能见一切法的真实相
   - 说明如来能见一切法的空性，不执著于一切法

4. **"如来有法眼"**：
   - 如来有法眼，能见一切法的差别相
   - 法眼是菩萨的眼睛，能见一切法的差别相，能度化众生
   - 说明如来有法眼，能见一切法的差别相
   - 体现了如来的智慧，能见一切法的差别相
   - 说明如来能见一切法的差别相，能度化众生

5. **"如来有佛眼"**：
   - 如来有佛眼，能见一切法的真实相
   - 佛眼是佛的眼睛，能见一切法的真实相，圆满无缺
   - 说明如来有佛眼，能见一切法的真实相
   - 体现了如来的圆满，具有佛眼
   - 说明如来能见一切法的真实相，圆满无缺

6. **五眼的关系**：
   - 五眼不是分开的，而是一体的
   - 佛眼包含一切眼，具有一切眼的功能
   - 说明如来具有一切眼，能见一切
   - 体现了如来的圆满，具有一切眼的功能
   - 说明如来能见一切，不受任何限制

### 二、如来悉知一切心

**经文要点：**
- 须菩提。于意云何。如恒河中所有沙。佛说是沙不
- 如是。世尊。如来说是沙
- 须菩提。于意云何。一恒河中所有沙。有如是沙等恒河
- 是诸恒河所有沙数。佛世界如是。宁为多不。甚多。世尊
- 佛告须菩提。尔所国土中。所有众生。若干种心。如来悉知

**讲解：**
1. **"如恒河中所有沙。佛说是沙不"**：
   - 如恒河中所有的沙，佛说是沙吗
   - 恒河是印度的大河，沙数极多，无法计算
   - 说明佛陀用恒河沙来比喻数量之多
   - 体现了佛陀的善巧方便，用比喻来说明
   - 说明要了知数量之多，无法计算

2. **"一恒河中所有沙。有如是沙等恒河"**：
   - 一恒河中所有的沙，有这么多沙等恒河
   - 说明数量之多，一恒河的沙数，等于这么多恒河
   - 体现了数量之多，无法想象
   - 说明要了知数量之多，无法计算
   - 这是比喻，说明数量之多

3. **"是诸恒河所有沙数。佛世界如是"**：
   - 这些恒河所有沙数，佛世界也是这样多
   - 说明佛世界的数量之多，等于这些恒河所有沙数
   - 体现了佛世界的数量之多，无法计算
   - 说明要了知佛世界的数量之多，无法计算
   - 这是比喻，说明佛世界的数量之多

4. **"尔所国土中。所有众生。若干种心。如来悉知"**：
   - 这些国土中，所有众生，各种心，如来全部知道
   - 说明如来能知一切众生的心，无论多少
   - 体现了如来的智慧，能知一切众生的心
   - 说明要了知如来的智慧，能知一切
   - 这是如来的智慧，能知一切众生的心

5. **如来智慧的无量**：
   - 如来的智慧无量无边，能知一切众生的心
   - 无论多少众生，多少种心，如来全部知道
   - 体现了如来的智慧，无量无边
   - 说明要了知如来的智慧，无量无边
   - 这是如来的智慧，能知一切

### 三、诸心皆为非心

**经文要点：**
- 何以故。如来说诸心。皆为非心。是名为心
- 所以者何。须菩提
- 过去心不可得。现在心不可得。未来心不可得

**讲解：**
1. **"如来说诸心。皆为非心。是名为心"**：
   - 如来说各种心，都不是真正的心，只是名字叫心
   - 说明心要无相，不能有心的执著
   - 体现了无相修行的关键，要破除心的执著
   - 说明要了知无我，没有真正的心
   - 这是《金刚经》的核心教义，要破除一切心的执著

2. **"过去心不可得"**：
   - 过去心不可得，过去的已经过去，不可得
   - 说明过去心是空的，不可得
   - 体现了无相修行的关键，要破除过去心的执著
   - 说明要了知无我，过去心不可得
   - 这是修行的关键，要破除过去心的执著

3. **"现在心不可得"**：
   - 现在心不可得，现在的正在过去，不可得
   - 说明现在心是空的，不可得
   - 体现了无相修行的关键，要破除现在心的执著
   - 说明要了知无我，现在心不可得
   - 这是修行的关键，要破除现在心的执著

4. **"未来心不可得"**：
   - 未来心不可得，未来的还没有来，不可得
   - 说明未来心是空的，不可得
   - 体现了无相修行的关键，要破除未来心的执著
   - 说明要了知无我，未来心不可得
   - 这是修行的关键，要破除未来心的执著

5. **三心不可得的意义**：
   - 过去、现在、未来三心都不可得
   - 说明心是空的，不可得
   - 体现了无相修行的关键，要破除一切心的执著
   - 说明要了知无我，一切心都不可得
   - 这是《金刚经》的核心教义，要破除一切心的执著

6. **心的无相性**：
   - 心要无相，不能有心的执著
   - 要了知心是空的，不可得
   - 要了知过去、现在、未来三心都不可得
   - 体现了无相修行的关键，要破除一切心的执著
   - 说明要了知无我，一切心都不可得
