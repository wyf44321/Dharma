# 知见不生分第三十一讲解

## 章节概述

本章佛陀开示发阿耨多罗三藐三菩提心者，于一切法，应如是知，如是见，如是信解，不生法相。说明世尊说我见人见众生见寿者见，即非我见人见众生见寿者见，是名我见人见众生见寿者见。说明所言法相者，如来说即非法相，是名法相。这是说明如何不生法相，以及如何正确理解知见，体现了《金刚经》无相修行的核心教义。

## 主要内容讲解

### 一、我见非我见

**经文要点：**
- 须菩提。若人言。佛说我见人见众生见寿者见
- 须菩提。于意云何。是人解我所说义不
- 不也。世尊。是人不解如来所说义
- 何以故。世尊说我见人见众生见寿者见。即非我见人见众生见寿者见
- 是名我见人见众生见寿者见

**白话文翻译：**
须菩提，如果有人说，佛说我见、人见、众生见、寿者见。须菩提，你认为如何？这个人理解我所说的义理吗？不，世尊，这个人不理解如来所说的义理。为什么呢？世尊说我见、人见、众生见、寿者见，不是真正的我见、人见、众生见、寿者见，只是名字叫我见、人见、众生见、寿者见。

**讲解：**
1. **"若人言。佛说我见人见众生见寿者见"**：
   - 如果有人说，佛说我见、人见、众生见、寿者见
   - 说明如果认为佛说我见等，这是错误的
   - 体现了无相修行的关键，要了知我见的无相性
   - 说明要了知无我，要了知我见的无相性
   - 这是修行的关键，要了知我见的无相性

2. **"是人不解如来所说义"**：
   - 这个人不理解如来所说的义理
   - 说明如果认为佛说我见等，就是不理解如来所说的义理
   - 体现了无相修行的关键，要了知我见的无相性
   - 说明要了知无我，要了知我见的无相性
   - 这是修行的关键，要了知我见的无相性

3. **"世尊说我见人见众生见寿者见。即非我见人见众生见寿者见。是名我见人见众生见寿者见"**：
   - 世尊说我见等，不是真正的我见等，只是名字叫我见等
   - 说明我见要无相，不能有我见的执著
   - 体现了无相修行的关键，要破除对我见的执著
   - 说明要了知无我，没有真正的我见
   - 这是《金刚经》的核心教义，要破除一切见的执著

### 二、不生法相

**经文要点：**
- 须菩提。发阿耨多罗三藐三菩提心者
- 于一切法。应如是知。如是见。如是信解
- 不生法相

**白话文翻译：**
须菩提，发无上正等正觉心的人，对于一切法，应该这样了知，这样见，这样信解，不生起法相。

**讲解：**
1. **"发阿耨多罗三藐三菩提心者"**：
   - 发无上正等正觉心的人
   - 说明发菩提心的人要无相，不能有相的执著
   - 体现了无相修行的关键，要破除相的执著
   - 说明要了知无我，要破除相的执著
   - 这是修行的关键，要破除相的执著

2. **"于一切法。应如是知。如是见。如是信解"**：
   - 对于一切法，应该这样了知，这样见，这样信解
   - 说明对于一切法，要无相了知，无相见，无相信解
   - 体现了无相修行的关键，要无相了知一切法
   - 说明要了知无我，要无相了知一切法
   - 这是修行的关键，要无相了知一切法

3. **"不生法相"**：
   - 不生起法相
   - 说明对于一切法，要不生起法相，不能有法相的执著
   - 体现了无相修行的关键，要破除法相的执著
   - 说明要了知无我，要破除法相的执著
   - 这是《金刚经》的核心教义，要破除一切法相的执著

4. **不生法相的意义**：
   - 对于一切法，要不生起法相，不能有法相的执著
   - 说明要无相了知一切法，要不生起法相
   - 体现了无相修行的关键，要破除法相的执著
   - 说明要了知无我，要破除法相的执著
   - 这是修行的关键，要破除法相的执著

### 三、法相非法相

**经文要点：**
- 须菩提。所言法相者。如来说即非法相
- 是名法相

**白话文翻译：**
须菩提，所说的法相，如来说不是真正的法相，只是名字叫法相。

**讲解：**
1. **"所言法相者"**：
   - 所说的法相
   - 说明法相要无相，不能有法相的执著
   - 体现了无相修行的关键，要破除对法相的执著
   - 说明要了知无我，没有真正的法相
   - 这是修行的关键，要破除对法相的执著

2. **"如来说即非法相"**：
   - 如来说不是真正的法相
   - 说明法相要无相，不能有法相的执著

   - 说明要了知无我，没有真正的法相
   - 这是《金刚经》的核心教义，要破除一切法相的执著

3. **"是名法相"**：
   - 只是名字叫法相
   - 说明法相要无相，不能有法相的执著
   - 体现了无相修行的关键，要破除对法相的执著
   - 说明要了知无我，没有真正的法相
   - 这是《金刚经》的核心教义，要破除一切法相的执著

4. **法相的无相性**：
   - 法相要无相，不能有法相的执著
   - 说明如果执著于法相，就不能了知无我
   - 体现了无相修行的关键，要不执著于法相，才能了知无我
