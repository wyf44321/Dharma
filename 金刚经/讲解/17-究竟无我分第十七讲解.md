# 究竟无我分第十七讲解

## 章节概述

本章佛陀再次回答如何降伏其心的问题，深入说明无我的究竟义。说明实无有法发阿耨多罗三藐三菩提心者，如来于然灯佛所无有法得阿耨多罗三藐三菩提。说明如来所得阿耨多罗三藐三菩提，于是中无实无虚。说明一切法皆是佛法，但一切法即非一切法。最后说明若菩萨通达无我法者，如来说名真是菩萨。这是说明无我的究竟义，是《金刚经》的核心教义之一。

## 主要内容讲解

### 一、如何降伏其心

**经文要点：**
- 尔时须菩提白佛言。世尊。善男子。善女人。发阿耨多罗三藐三菩提心
- 云何应住。云何降伏其心
- 佛告须菩提。善男子。善女人。发阿耨多罗三藐三菩提心者。当生如是心
- 我应灭度一切众生。灭度一切众生已。而无有一众生实灭度者

**讲解：**
1. **"云何应住。云何降伏其心"**：
   - 如何安住其心，如何降伏妄心
   - 这是须菩提再次提出同样的问题，说明这个问题的重要性
   - 体现了修行的核心问题，就是如何降伏妄心
   - 说明要反复思维这个问题，才能真正理解
   - 体现了《金刚经》的教义，要反复强调

2. **"当生如是心"**：
   - 应当生起这样的心
   - 说明发菩提心要有正确的心态
   - 体现了修行的关键，要有正确的心态
   - 说明要学习佛陀的教法，要有正确的心态

3. **"我应灭度一切众生"**：
   - 我应当度化一切众生
   - 说明发菩提心要度化一切众生
   - 体现了菩萨的大慈悲心，要度化一切众生
   - 说明要发大愿，要度化一切众生
   - 这是修行的目标，要度化一切众生

4. **"灭度一切众生已。而无有一众生实灭度者"**：
   - 度化一切众生后，实际上没有众生被度化
   - 说明度化众生要无相，不能有度化的相
   - 体现了无相修行的关键，要破除度化的执著
   - 说明要了知无我，没有能度的人，也没有被度的众生
   - 这是《金刚经》的核心教义，要破除一切执著

### 二、实无有法发菩提心

**经文要点：**
- 何以故。须菩提
- 若菩萨有我相人相众生相寿者相。即非菩萨
- 所以者何。须菩提。实无有法。发阿耨多罗三藐三菩提心者

**讲解：**
1. **"若菩萨有我相人相众生相寿者相。即非菩萨"**：
   - 如果菩萨有我相、人相、众生相、寿者相，就不是真正的菩萨
   - 说明真正的菩萨要破除四相
   - 体现了无相修行的关键，要破除一切相
   - 说明要了知无我，不能有我的执著
   - 这是修行的标准，要破除四相

2. **"实无有法。发阿耨多罗三藐三菩提心者"**：
   - 实际上没有法能发无上正等正觉心
   - 说明发菩提心要无相，不能有法的执著
   - 体现了无相修行的关键，要破除法的执著
   - 说明要了知无我，没有能发心的法
   - 这是《金刚经》的核心教义，要破除一切法的执著

### 三、如来于法无所得

**经文要点：**
- 须菩提。于意云何。如来于然灯佛所。有法得阿耨多罗三藐三菩提不
- 不也。世尊。如我解佛所说义。佛于然灯佛所。无有法得阿耨多罗三藐三菩提
- 佛言。如是如是。须菩提。实无有法。如来得阿耨多罗三藐三菩提
- 须菩提。若有法如来得阿耨多罗三藐三菩提者。然灯佛即不与我授记
- 汝于来世。当得作佛。号释迦牟尼
- 以实无有法。得阿耨多罗三藐三菩提。是故然灯佛与我授记

**讲解：**
1. **"如来于然灯佛所。有法得阿耨多罗三藐三菩提不"**：
   - 如来在然灯佛那里，有法得到无上正等正觉吗
   - 然灯佛是过去佛，是释迦牟尼佛的授记师
   - 说明佛陀用自己成佛的经历来说明无相的道理
   - 体现了佛陀的善巧方便，用实际例子来说明
   - 说明要了知无我，没有能得的法

2. **"无有法得阿耨多罗三藐三菩提"**：
   - 没有法能得到无上正等正觉
   - 说明成佛要无相，不能有法的执著
   - 体现了无相修行的关键，要破除法的执著
   - 说明要了知无我，没有能得的法
   - 这是《金刚经》的核心教义，要破除一切法的执著

3. **"若有法如来得阿耨多罗三藐三菩提者。然灯佛即不与我授记"**：
   - 如果有法如来能得无上正等正觉，然灯佛就不会给我授记
   - 说明如果有法的执著，就不能成佛
   - 体现了无相修行的关键，要破除法的执著才能成佛
   - 说明要了知无我，没有能得的法才能成佛
   - 这是成佛的条件，要破除一切法的执著

4. **"以实无有法。得阿耨多罗三藐三菩提。是故然灯佛与我授记"**：
   - 因为实际上没有法能得无上正等正觉，所以然灯佛给我授记
   - 说明无相才能成佛，有相不能成佛
   - 体现了无相修行的关键，要破除法的执著才能成佛
   - 说明要了知无我，没有能得的法才能成佛
   - 这是成佛的根本，要破除一切法的执著

### 四、如来者即诸法如义

**经文要点：**
- 何以故。如来者。即诸法如义
- 若有人言。如来得阿耨多罗三藐三菩提
- 须菩提。实无有法。佛得阿耨多罗三藐三菩提

**讲解：**
1. **"如来者。即诸法如义"**：
   - 如来就是诸法如义，就是一切法本来如是的真实相
   - 说明如来不是有相的，而是诸法的真实相
   - 体现了无相修行的关键，要了知诸法的真实相
   - 说明要了知无我，要了知诸法的真实相
   - 这是如来的真实义，是诸法的真实相

2. **"若有人言。如来得阿耨多罗三藐三菩提"**：
   - 如果有人说如来得到无上正等正觉
   - 说明这是错误的见解，如来没有得什么
   - 体现了无相修行的关键，要破除得的执著
   - 说明要了知无我，没有能得的如来
   - 这是错误的见解，要破除

3. **"实无有法。佛得阿耨多罗三藐三菩提"**：
   - 实际上没有法，佛得到无上正等正觉
   - 说明成佛要无相，不能有法的执著
   - 体现了无相修行的关键，要破除法的执著
   - 说明要了知无我，没有能得的法
   - 这是《金刚经》的核心教义，要破除一切法的执著

### 五、无实无虚

**经文要点：**
- 须菩提。如来所得阿耨多罗三藐三菩提
- 于是中无实无虚

**讲解：**
1. **"如来所得阿耨多罗三藐三菩提"**：
   - 如来所得的无上正等正觉
   - 说明如来确实有得，但不是有相的得
   - 体现了中道的思想，不是实也不是虚
   - 说明要了知中道，不能落于两边
   - 这是如来的真实义，是中道

2. **"于是中无实无虚"**：
   - 在这当中，不是实也不是虚
   - 说明如来所得不是实有的，也不是虚无的
   - 体现了中道的思想，要超越实和虚
   - 说明要了知中道，不能落于实，也不能落于虚
   - 这是中道的真实义，要超越两边
