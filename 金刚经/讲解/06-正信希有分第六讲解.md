# 正信希有分第六讲解

## 章节概述

本章须菩提请问是否有人能生实信，佛陀开示能生信心者，已于无量千万佛所种诸善根。说明不应取法，不应取非法，法尚应舍，何况非法。这是说明正信的重要性，以及如何正确对待佛法。

## 主要内容讲解

### 一、须菩提的疑问

**经文要点：**
- 须菩提白佛言。世尊。颇有众生。得闻如是言说章句。生实信不

**讲解：**
1. **生实信不**：能否生起真实的信心？
2. **这是对未来的担忧**：担心未来众生能否相信这部经。

### 二、能生信心者

**经文要点：**
- 佛告须菩提。莫作是说
- 如来灭后。后五百岁。有持戒修福者。于此章句。能生信心。以此为实
- 当知是人。不于一佛二佛三四五佛而种善根。已于无量千万佛所。种诸善根

**讲解：**
1. **后五百岁**：佛陀灭度后的第五个五百年。
2. **持戒修福**：持守戒律，修习福德。
3. **种诸善根**：已在无量千万佛所种下善根。

### 三、无四相

**经文要点：**
- 闻是章句。乃至一念生净信者。须菩提。如来悉知悉见
- 是诸众生。无复我相。人相。众生相。寿者相
- 无法相。亦无非法相

**讲解：**
1. **一念生净信**：哪怕只有一念生起清净的信心。
2. **无四相**：没有我相、人相、众生相、寿者相。
3. **无法相亦无非法相**：不执著于法相，也不执著于非法相。

### 四、不应取法不应取非法

**经文要点：**
- 是诸众生。若心取相。即为著我人众生寿者
- 若取法相。即著我人众生寿者
- 若取非法相。即著我人众生寿者
- 是故不应取法。不应取非法

**讲解：**
1. **取相即著**：如果取相，就是执著于四相。
2. **取法相即著**：如果取法相，也是执著。
3. **取非法相即著**：如果取非法相，也是执著。
4. **不应取法不应取非法**：不应该取法，也不应该取非法。

### 五、筏喻

**经文要点：**
- 以是义故。如来常说。汝等比丘。知我说法。如筏喻者
- 法尚应舍。何况非法

**讲解：**
1. **筏喻**：佛法如渡河的筏，过了河就要舍弃。
2. **法尚应舍**：连法都应该舍弃。
3. **何况非法**：何况不是法的东西。

## 教义要点

1. **能生信心者**：
   - 已在无量千万佛所种下善根
   - 能持戒修福
   - 能生起真实的信心

2. **无四相**：
   - 没有我相、人相、众生相、寿者相
   - 无法相，亦无非法相
   - 这是生起正信的条件

3. **不应取法不应取非法**：
   - 不应该取法，也不应该取非法
   - 取任何相都是执著
   - 要超越一切相

4. **筏喻**：
   - 佛法如筏，过了河就要舍弃
   - 法尚应舍，何况非法
   - 不要执著于法

## 修行启示

1. **种善根**：
   - 要在诸佛所种下善根
   - 要持戒修福
   - 要积累成佛的资粮

2. **生正信**：
   - 要生起真实的信心
   - 要无四相
   - 要无法相，亦无非法相

3. **不取法不取非法**：
   - 不要取法，也不要取非法
   - 要超越一切相
   - 要证得无相

4. **法尚应舍**：
   - 连法都应该舍弃
   - 不要执著于法
   - 要证得无相

## 现代意义

   - 要在诸佛所种下善根

   - 要积累成佛的资粮

2. **生正信**：

   - 要无四相
   - 要无法相，亦无非法相

3. **不执著**：
   - 不要执著于任何法
