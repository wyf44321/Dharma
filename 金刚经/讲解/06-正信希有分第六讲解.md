# 正信希有分第六讲解

## 章节概述

本章须菩提请问是否有人能生实信，佛陀开示能生信心者，已于无量千万佛所种诸善根。说明不应取法，不应取非法，法尚应舍，何况非法。这是说明正信的重要性，以及如何正确对待佛法。本章揭示了正信的条件和如何正确对待佛法，是《金刚经》的重要教义。

## 主要内容讲解

### 一、须菩提的疑问

**经文要点：**
- 须菩提白佛言。世尊。颇有众生。得闻如是言说章句。生实信不

**讲解：**
1. **"颇有众生"**：
   - "颇"表示是否、能否
   - 须菩提问：是否有众生能够听闻这部经而生起真实的信心
   - 说明须菩提担心未来众生能否相信这部经
   - 体现了须菩提的慈悲，关心未来众生的利益
   - 说明《金刚经》的教义深奥，不是所有人都能理解

2. **"得闻如是言说章句"**：
   - "言说章句"就是《金刚经》的经文
   - 说明须菩提问的是能否听闻《金刚经》而生起信心
   - 体现了《金刚经》的殊胜，需要深厚的善根才能理解
   - 说明听闻《金刚经》需要因缘，不是所有人都能听闻

3. **"生实信不"**：
   - "实信"就是真实的信心，不是虚假的信心
   - 说明须菩提问的是能否生起真实的信心
   - 体现了正信的重要性，是修行的基础
   - 说明生起正信不容易，需要深厚的善根

### 二、能生信心者

**经文要点：**
- 佛告须菩提。莫作是说
- 如来灭后。后五百岁。有持戒修福者。于此章句。能生信心。以此为实
- 当知是人。不于一佛二佛三四五佛而种善根。已于无量千万佛所。种诸善根

**讲解：**
1. **"莫作是说"**：
   - 佛陀说"莫作是说"，就是不要这样说
   - 说明佛陀要破除须菩提的疑虑，肯定有人能生起信心
   - 体现了佛陀的慈悲，要鼓励众生，不要让他们失去信心
   - 说明《金刚经》虽然深奥，但有人能够理解

2. **"如来灭后。后五百岁"**：
   - "后五百岁"就是佛陀灭度后的第五个五百年
   - 按照佛教的说法，佛陀灭度后分为五个五百年
   - 第一个五百年是正法时期，第二个五百年是像法时期，第三个五百年是末法时期
   - "后五百岁"就是末法时期，是最困难的时期
   - 说明即使在最困难的末法时期，也有人能生起信心
   - 体现了《金刚经》的殊胜，即使在末法时期也能度化众生

3. **"有持戒修福者"**：
   - "持戒修福"就是持守戒律，修习福德
   - 说明能生起信心的人，必须持戒修福
   - 体现了持戒修福的重要性，是生起正信的基础
   - 说明要生起正信，必须持戒修福，积累福德

4. **"于此章句。能生信心。以此为实"**：
   - 对于《金刚经》的章句，能生起信心，认为这是真实的
   - 说明能生起信心的人，会认为《金刚经》是真实的
   - 体现了正信的重要性，是修行的基础
   - 说明要生起正信，必须认为《金刚经》是真实的

5. **"不于一佛二佛三四五佛而种善根"**：
   - 不是在一位佛、两位佛、三四五位佛那里种下善根
   - 说明种善根不是一朝一夕的事，需要多生多世
   - 体现了善根的重要性，是生起正信的基础
   - 说明要生起正信，必须多生多世种善根

6. **"已于无量千万佛所。种诸善根"**：
   - 已经在无量千万佛那里种下善根
   - 说明能生起信心的人，善根非常深厚
   - 体现了善根的重要性，是生起正信的基础
   - 说明要生起正信，必须多生多世种善根，善根要非常深厚

### 三、无四相

**经文要点：**
- 闻是章句。乃至一念生净信者。须菩提。如来悉知悉见
- 是诸众生。无复我相。人相。众生相。寿者相
- 无法相。亦无非法相

**讲解：**
1. **"乃至一念生净信者"**：
   - "一念"就是一瞬间，非常短的时间
   - "净信"就是清净的信心，没有污染的信心
   - 说明哪怕只有一念生起清净的信心，也是很难得的
   - 体现了正信的殊胜，即使只有一念也是很难得的
   - 说明要生起正信不容易，需要深厚的善根

2. **"如来悉知悉见"**：
   - 如来完全知道，完全看见
   - 说明如来知道谁生起了净信，完全了解
   - 体现了如来的智慧，能够了知一切
   - 说明如来会护持生起净信的人，不会让他们退转

3. **"是诸众生。得如是无量福德"**：
   - 这些众生会得到无量的福德
   - 说明生起净信的功德很大，会得到无量的福德
   - 体现了正信的功德，会得到无量的福德
   - 说明要生起正信，会得到无量的福德

4. **"无复我相。人相。众生相。寿者相"**：
   - 没有我相、人相、众生相、寿者相
   - 说明生起净信的人，已经破除了四相
   - 体现了正信的条件，必须破除四相
   - 说明要生起正信，必须破除四相，不能执著于四相

5. **"无法相。亦无非法相"**：
   - 没有法相，也没有非法相
   - 说明生起净信的人，既不执著于法相，也不执著于非法相
   - 体现了正信的条件，必须超越法相和非法相
   - 说明要生起正信，必须超越一切相，不能执著于任何相

### 四、不应取法不应取非法

**经文要点：**
- 是诸众生。若心取相。即为著我人众生寿者
- 若取法相。即著我人众生寿者
- 若取非法相。即著我人众生寿者
- 是故不应取法。不应取非法

**讲解：**
1. **"若心取相。即为著我人众生寿者"**：
   - 如果心取相，就是执著于四相
   - 说明取相就是执著，就是障碍
   - 体现了取相的危害，会执著于四相
   - 说明要破除四相，必须不取相，不能执著于任何相

2. **"若取法相。即著我人众生寿者"**：
   - 如果取法相，也是执著于四相
   - 说明取法相也是执著，也是障碍
   - 体现了取法相的危害，会执著于四相
   - 说明要破除四相，必须不取法相，不能执著于法相

3. **"若取非法相。即著我人众生寿者"**：
   - 如果取非法相，也是执著于四相
   - 说明取非法相也是执著，也是障碍
   - 体现了取非法相的危害，会执著于四相
   - 说明要破除四相，必须不取非法相，不能执著于非法相

4. **"是故不应取法。不应取非法"**：
   - 所以不应该取法，也不应该取非法
   - 说明要破除四相，必须不取任何相
   - 体现了不取相的重要性，是破除四相的关键
   - 说明要生起正信，必须不取法，也不取非法，要超越一切相

### 五、筏喻

**经文要点：**
- 以是义故。如来常说。汝等比丘。知我说法。如筏喻者
- 法尚应舍。何况非法

**讲解：**
1. **"如筏喻者"**：
   - 佛法如渡河的筏，过了河就要舍弃
   - 说明佛法是工具，是用来度脱生死的，过了河就要舍弃
   - 体现了佛法的工具性，不要执著于佛法
   - 说明要证得无相，必须舍弃佛法，不能执著于佛法

2. **"法尚应舍"**：
   - 连法都应该舍弃
   - 说明佛法也要舍弃，不能执著于佛法
   - 体现了不执著的重要性，连法都要舍弃
   - 说明要证得无相，必须舍弃一切法，不能执著于任何法

3. **"何况非法"**：
   - 何况不是法的东西
   - 说明非法更要舍弃，不能执著于非法
   - 体现了不执著的重要性，连非法都要舍弃
   - 说明要证得无相，必须舍弃一切，不能执著于任何东西
