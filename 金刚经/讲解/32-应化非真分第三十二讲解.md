# 应化非真分第三十二讲解

## 章节概述

本章是《金刚经》的最后一分，佛陀总结整部经的核心教义：一切有为法，如梦幻泡影，如露亦如电，应作如是观。这是整部经的总结，说明一切现象都是虚幻的，应该这样观照。

## 主要内容讲解

### 一、受持的功德

**经文要点：**
- 须菩提。若有人以满无量阿僧祗世界七宝。持用布施
- 若有善男子。善女人。发菩萨心者。持于此经。乃至四句偈等
- 受持读诵。为人演说。其福胜彼

**讲解：**
1. **七宝布施**：用满无量世界的七宝布施。
2. **受持读诵**：受持读诵《金刚经》。
3. **为人演说**：为他人演说。
4. **其福胜彼**：福德超过七宝布施。

### 二、如何为人演说

**经文要点：**
- 云何为人演说。不取于相。如如不动
- 何以故

**讲解：**
1. **不取于相**：不执著于任何相。
2. **如如不动**：如如不动，不随境转。
3. **这是演说的方法**：要无相演说。

### 三、一切有为法

**经文要点：**
- 一切有为法。如梦幻泡影
- 如露亦如电。应作如是观

**讲解：**
1. **一切有为法**：一切有造作的法。
2. **如梦幻泡影**：如梦境、幻化、水泡、影子。
3. **如露亦如电**：如露水、如闪电。
4. **应作如是观**：应该这样观照。

### 四、法会圆满

**经文要点：**
- 佛说是经已。长老须菩提。及诸比丘。比丘尼
- 优婆塞。优婆夷。一切世间天人阿修罗
- 闻佛所说。皆大欢喜。信受奉行

**讲解：**
1. **法会圆满**：佛陀说完这部经。
2. **皆大欢喜**：大家都非常欢喜。
3. **信受奉行**：相信、接受、奉行。

## 教义要点

1. **受持的功德**：
   - 受持读诵《金刚经》的功德
   - 超过用七宝布施
   - 要受持读诵，为人演说

2. **如何演说**：
   - 要不取于相
   - 要如如不动
   - 要无相演说

3. **一切有为法**：
   - 一切有为法都是虚幻的
   - 如梦幻泡影，如露亦如电
   - 应该这样观照

4. **法会圆满**：
   - 法会圆满结束
   - 大家都欢喜信受
   - 要信受奉行
