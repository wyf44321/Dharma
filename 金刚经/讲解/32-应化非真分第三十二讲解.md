# 应化非真分第三十二讲解

## 章节概述

本章是《金刚经》的最后一分，佛陀总结整部经的核心教义。说明受持读诵此经的功德，超过用满无量阿僧祗世界七宝布施的功德。说明如何为人演说：不取于相，如如不动。最后总结：一切有为法，如梦幻泡影，如露亦如电，应作如是观。这是整部经的总结，说明一切现象都是虚幻的，应该这样观照，体现了《金刚经》无相修行的核心教义。

## 主要内容讲解

### 一、受持的功德

**经文要点：**
- 须菩提。若有人以满无量阿僧祗世界七宝。持用布施
- 若有善男子。善女人。发菩萨心者。持于此经。乃至四句偈等
- 受持读诵。为人演说。其福胜彼

**白话文翻译：**
须菩提，如果有人用满无量阿僧祗世界的七宝，用来布施。如果有善男子、善女人，发菩萨心的人，受持这部经，甚至四句偈等，受持读诵，为他人演说，他的福德超过前面用七宝布施的福德。

**讲解：**
1. **"若有人以满无量阿僧祗世界七宝。持用布施"**：
   - 如果有人用满无量阿僧祗世界的七宝，用来布施
   - 阿僧祗是极大的数字，无法计算
   - 说明布施的数量极大，无法想象
   - 体现了布施的功德很大，但还要看是否有执著
   - 说明要了知布施的功德，但不要执著于功德

2. **"发菩萨心者。持于此经。乃至四句偈等"**：
   - 发菩萨心的人，受持这部经，甚至四句偈等
   - 说明受持此经要发菩萨心，要有大愿
   - 体现了受持此经的重要性，要发菩萨心
   - 说明要了知受持此经的功德，要发菩萨心
   - 这是修行的关键，要发菩萨心

3. **"受持读诵。为人演说"**：
   - 受持读诵，为他人演说
   - 说明受持此经要读诵，要为他人演说
   - 体现了受持此经的全面性，要读诵和演说
   - 说明要了知受持此经的功德，要读诵和演说
   - 这是修行的关键，要读诵和演说

4. **"其福胜彼"**：
   - 福德超过七宝布施
   - 说明受持此经的功德，超过用满无量世界的七宝布施
   - 体现了此经的殊胜，功德不可思议
   - 说明要了知受持此经的功德，功德不可思议
   - 这是此经的殊胜，功德不可思议

### 二、如何为人演说

**经文要点：**
- 云何为人演说。不取于相。如如不动
- 何以故

**白话文翻译：**
如何为他人演说？不执著于任何相，如如不动。为什么呢？

**讲解：**
1. **"云何为人演说"**：
   - 如何为他人演说
   - 说明演说要有正确的方法
   - 体现了演说的关键，要有正确的方法
   - 说明要了知演说的方法，要有正确的方法
   - 这是演说的关键，要有正确的方法

2. **"不取于相"**：
   - 不执著于任何相
   - 说明演说要不执著于任何相，不能有相的执著
   - 体现了无相修行的关键，要破除相的执著
   - 说明要了知无我，要不执著于任何相
   - 这是演说的关键，要不执著于任何相

3. **"如如不动"**：
   - 如如不动，不随境转
   - 说明演说要如如不动，不随境转
   - 体现了无相修行的关键，要如如不动
   - 说明要了知无我，要如如不动
   - 这是演说的关键，要如如不动

4. **无相演说的意义**：
   - 演说要不执著于任何相，要如如不动
   - 说明演说要无相，不能有相的执著
   - 体现了无相修行的关键，要无相演说
   - 说明要了知无我，要无相演说
   - 这是演说的关键，要无相演说

### 三、一切有为法

**经文要点：**
- 一切有为法。如梦幻泡影
- 如露亦如电。应作如是观

**白话文翻译：**
一切有为法，如梦境、幻化、水泡、影子，如露水、如闪电，应该这样观照。

**讲解：**
1. **"一切有为法"**：
   - 一切有造作的法
   - 说明一切有为法都是虚幻的，不是真实的
   - 体现了无相修行的关键，要了知一切有为法都是虚幻的
   - 说明要了知无我，要了知一切有为法都是虚幻的
   - 这是修行的关键，要了知一切有为法都是虚幻的

2. **"如梦幻泡影"**：
   - 如梦境、幻化、水泡、影子
   - 说明一切有为法如梦境、幻化、水泡、影子，都是虚幻的
   - 体现了无相修行的关键，要了知一切有为法都是虚幻的
   - 说明要了知无我，要了知一切有为法都是虚幻的
   - 这是修行的关键，要了知一切有为法都是虚幻的

3. **"如露亦如电"**：
   - 如露水、如闪电
   - 说明一切有为法如露水、如闪电，都是短暂的
   - 体现了无相修行的关键，要了知一切有为法都是短暂的
   - 说明要了知无我，要了知一切有为法都是短暂的
   - 这是修行的关键，要了知一切有为法都是短暂的

4. **"应作如是观"**：
   - 应该这样观照
   - 说明要这样观照一切有为法，都是虚幻的、短暂的
   - 体现了无相修行的关键，要这样观照
   - 说明要了知无我，要这样观照
   - 这是修行的关键，要这样观照

5. **观照的意义**：
   - 要观照一切有为法都是虚幻的、短暂的
   - 说明要这样观照，才能了知无我
   - 体现了无相修行的关键，要这样观照
   - 说明要了知无我，要这样观照
   - 这是修行的关键，要这样观照

### 四、法会圆满

**经文要点：**
- 佛说是经已。长老须菩提。及诸比丘。比丘尼
- 优婆塞。优婆夷。一切世间天人阿修罗
- 闻佛所说。皆大欢喜。信受奉行

**白话文翻译：**
佛陀说完这部经后，长老须菩提，以及诸位比丘、比丘尼、优婆塞、优婆夷，一切世间的天、人、阿修罗，听闻佛陀所说，都非常欢喜，相信、接受、奉行。

**讲解：**
1. **"佛说是经已"**：
   - 佛陀说完这部经
   - 说明法会圆满结束，佛陀已经说完这部经
   - 体现了法会的圆满，佛陀已经说完这部经
   - 说明要了知法会的圆满，要珍惜听闻佛法的因缘
   - 这是法会的圆满，要珍惜听闻佛法的因缘

2. **"皆大欢喜"**：
   - 大家都非常欢喜
   - 说明听闻这部经后，大家都非常欢喜
   - 体现了听闻佛法的欢喜，大家都非常欢喜
   - 说明要了知听闻佛法的欢喜，要珍惜听闻佛法的因缘
   - 这是听闻佛法的欢喜，要珍惜听闻佛法的因缘

3. **"信受奉行"**：
   - 相信、接受、奉行
   - 说明听闻这部经后，要相信、接受、奉行
   - 体现了听闻佛法的重要性，要相信、接受、奉行
   - 说明要了知听闻佛法的重要性，要相信、接受、奉行
   - 这是听闻佛法的重要性，要相信、接受、奉行
