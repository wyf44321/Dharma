# 无得无说分第七讲解

## 章节概述

本章佛陀开示无有定法如来可说，无有定法名阿耨多罗三藐三菩提。说明如来所说法，皆不可取，不可说，非法非非法。一切贤圣，皆以无为法而有差别。这是说明佛法的无定性，以及如何正确理解佛法。本章揭示了佛法的本质：无有定法，不可取，不可说，这是《金刚经》的重要教义。

## 主要内容讲解

### 一、如来得法否

**经文要点：**
- 须菩提。于意云何。如来得阿耨多罗三藐三菩提耶
- 如来有所说法耶

**白话文翻译：**
须菩提，你认为如何？如来是否得到了无上正等正觉？如来是否有所说法？

**讲解：**
1. **"于意云何"**：
   - 佛陀问须菩提"于意云何"，就是问须菩提的意思如何
   - 这是佛陀的善巧方便，通过提问来引导弟子思考
   - 说明佛陀会通过提问来启发弟子，让弟子自己思考
   - 体现了佛陀的教学方法，不是直接说答案，而是引导思考

2. **"如来得阿耨多罗三藐三菩提耶"**：
   - 佛陀问：如来是否得到了无上正等正觉？
   - 一般人认为如来得到了无上正等正觉，但这是错误的
   - 说明佛陀要破除对得法的执著，让弟子理解真正的法
   - 体现了佛陀的善巧方便，通过提问来破除执著
   - 说明要破除对得法的执著，要理解真正的法

3. **"如来有所说法耶"**：
   - 佛陀问：如来是否有所说法？
   - 一般人认为如来有所说法，但这也是错误的
   - 说明佛陀要破除对说法的执著，让弟子理解真正的说法
   - 体现了佛陀的善巧方便，通过提问来破除执著
   - 说明要破除对说法的执著，要理解真正的说法

### 二、无有定法

**经文要点：**
- 须菩提言。如我解佛所说义
- 无有定法。名阿耨多罗三藐三菩提
- 亦无有定法。如来可说

**白话文翻译：**
须菩提说：如我理解佛陀所说的意思，没有固定的法，称为无上正等正觉。也没有固定的法，如来可以说。

**讲解：**
1. **"如我解佛所说义"**：
   - 须菩提说：如我理解佛陀所说的意思
   - 说明须菩提理解了佛陀的深意，知道如何回答
   - 体现了须菩提的智慧，能够理解佛陀的深意
   - 说明我们要理解佛陀的深意，不能只是表面理解

2. **"无有定法。名阿耨多罗三藐三菩提"**：
   - 没有固定的法，称为无上正等正觉
   - 说明无上正等正觉不是固定的法，没有固定的名称
   - 体现了佛法的无定性，没有固定的法
   - 说明要破除对法的执著，要理解法的无定性
   - 体现了空性的智慧，一切法都是空的，没有固定的法

3. **"亦无有定法。如来可说"**：
   - 也没有固定的法，如来可以说
   - 说明如来没有固定的法可以说，没有固定的说法
   - 体现了佛法的无定性，没有固定的法
   - 说明要破除对说法的执著，要理解说法的无定性
   - 体现了空性的智慧，说法也是空的，没有固定的说法

4. **无有定法的含义**：
   - 无有定法就是没有固定的法，法是无定的
   - 说明佛法不是固定的，要根据众生的根机来说法
   - 体现了佛法的善巧方便，要根据情况来说法
   - 说明要破除对法的执著，要理解法的无定性
   - 体现了空性的智慧，一切法都是空的，没有固定的法

### 三、不可取不可说

**经文要点：**
- 何以故。如来所说法。皆不可取。不可说
- 非法非非法

**白话文翻译：**
为什么呢？如来所说的法，都不可以执取，不可以言说。不是法，也不是非法。

**讲解：**
1. **"何以故"**：
   - 须菩提问"何以故"，就是要解释为什么"无有定法"
   - 说明下面要解释原因，让佛陀理解
   - 体现了须菩提的智慧，不仅要知其然，还要知其所以然

2. **"如来所说法。皆不可取。不可说"**：
   - 如来所说的法，都不可以执取，不可以言说
   - 说明如来所说的法，不能执取，也不能言说
   - 体现了佛法的超越性，超越语言文字
   - 说明要破除对法的执著，要超越语言文字
   - 体现了空性的智慧，一切法都是空的，不能执取，也不能言说

3. **"非法非非法"**：
   - 不是法，也不是非法
   - 说明如来所说的法，既不是法，也不是非法
   - 体现了佛法的中道，不落两边
   - 说明要破除对法的执著，要不落两边
   - 体现了空性的智慧，一切法都是空的，既不是法，也不是非法

4. **不可取不可说的含义**：
   - 不可取就是不能执取，不能执著于法
   - 不可说就是不能言说，不能完全用语言表达
   - 说明佛法超越语言文字，不能完全用语言表达
   - 体现了佛法的超越性，超越一切相
   - 说明要证得无相，要超越语言文字

### 四、无为法

**经文要点：**
- 所以者何。一切贤圣。皆以无为法而有差别

**白话文翻译：**
为什么呢？一切贤圣，都是因为无为法而有差别。

**讲解：**
1. **"所以者何"**：
   - 佛陀问"所以者何"，就是要解释为什么"非法非非法"
   - 说明下面要解释原因，让须菩提理解
   - 体现了佛陀的善巧方便，会解释原因

2. **"一切贤圣"**：
   - 所有的贤圣，包括声闻、缘觉、菩萨、佛
   - 说明所有的贤圣都是证得无为法的
   - 体现了无为法的重要性，是成佛的根本
   - 说明要成佛，必须证得无为法

3. **"皆以无为法"**：
   - 都是以无为法，都是证得无为法的
   - 说明所有的贤圣都是证得无为法的
   - 体现了无为法的重要性，是成佛的根本
   - 说明要成佛，必须证得无为法

4. **"而有差别"**：
   - 而有不同的差别，证得的程度不同
   - 说明虽然都是证得无为法，但证得的程度不同
   - 体现了证得的差别，声闻、缘觉、菩萨、佛的证得程度不同
   - 说明要成佛，必须证得最高的无为法

5. **无为法的含义**：
   - 无为法就是没有造作的法，是本来如此的
   - 说明无为法是根本，一切法都是从无为法出来的
   - 体现了无为法的重要性，是成佛的根本
   - 说明要证得无为法，要超越一切相
   - 体现了空性的智慧，无为法就是空性
