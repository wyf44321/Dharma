# 法界通化分第十九讲解

## 章节概述

本章佛陀开示若福德有实，如来不说得福德多。以福德无故，如来说得福德多。说明用七宝充满三千大千世界布施，得福甚多，但如果福德是真实的，如来就不说得福德多。因为福德不是真实的，所以如来说得福德多。这是说明福德的空性，以及如何正确理解福德，体现了《金刚经》无相修行的核心教义。

## 主要内容讲解

### 一、七宝布施的福德

**经文要点：**
- 须菩提。于意云何。若有人满三千大千世界七宝。以用布施
- 是人以是因缘。得福多不
- 如是。世尊。此人以是因缘。得福甚多

**白话文翻译：**
须菩提，你认为如何？如果有人用七宝充满三千大千世界，用来布施，这个人以这个因缘，得到的福德多吗？是的，世尊，这个人以这个因缘，得到的福德非常多。

**讲解：**
1. **"满三千大千世界七宝"**：
   - 用七宝充满三千大千世界
   - 七宝是金、银、琉璃、玻璃、砗磲、赤珠、玛瑙等七种宝物
   - 三千大千世界是一个佛所教化的世界，范围极大
   - 说明布施的数量极大，无法想象
   - 体现了布施的功德很大，但还要看是否有执著

2. **"以用布施"**：
   - 用来布施
   - 说明这是大布施，用七宝充满三千大千世界来布施
   - 体现了布施的功德很大，但还要看是否有执著
   - 说明要了知布施的功德，但不要执著于功德
   - 这是修行的关键，要布施但不执著

3. **"得福多不"**：
   - 得到的福德多吗
   - 说明佛陀问须菩提，这样的布施得到的福德多吗
   - 体现了佛陀的善巧方便，用问题来引导
   - 说明要了知布施的功德，但不要执著于功德
   - 这是修行的关键，要了知功德但不执著

4. **"得福甚多"**：
   - 得到的福德非常多
   - 说明须菩提回答，这样的布施得到的福德非常多
   - 体现了布施的功德确实很大，但还要看是否有执著
   - 说明要了知布施的功德，但不要执著于功德
   - 这是修行的关键，要了知功德但不执著

### 二、福德无实

**经文要点：**
- 须菩提。若福德有实。如来不说得福德多
- 以福德无故。如来说得福德多

**白话文翻译：**
须菩提，如果福德是真实的，如来就不说得福德多。因为福德不是真实的，所以如来说得福德多。

**讲解：**
1. **"若福德有实。如来不说得福德多"**：
   - 如果福德是真实的，如来就不说得福德多
   - 说明如果福德是实有的，如来就不说得福德多
   - 体现了无相修行的关键，要破除福德的执著
   - 说明要了知福德不是真实的，不能执著于福德
   - 这是《金刚经》的核心教义，要破除一切法的执著

2. **"以福德无故。如来说得福德多"**：
   - 因为福德不是真实的，所以如来说得福德多
   - 说明因为福德是空的，不是实有的，所以如来说得福德多
   - 体现了无相修行的关键，要了知福德的空性
   - 说明要了知福德是空的，不执著于福德，才能得到真正的福德
   - 这是《金刚经》的核心教义，要了知空性才能得到真正的福德

3. **福德无实的意义**：
   - 福德不是实有的，是空的
   - 说明如果执著于福德，福德就变成有限的
   - 体现了无相修行的关键，要不执著于福德，福德才是无限的
   - 说明要了知福德的空性，不执著于福德，才能得到真正的福德
   - 这是修行的关键，要了知空性才能得到真正的福德

4. **福德多的原因**：
   - 因为福德不是真实的，所以福德多
   - 说明因为福德是空的，不执著于福德，所以福德才是无限的
   - 体现了无相修行的关键，要不执著于福德，福德才是无限的
   - 说明要了知福德的空性，不执著于福德，才能得到真正的福德
   - 这是修行的关键，要了知空性才能得到真正的福德
