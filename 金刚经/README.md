# 金刚般若波罗蜜经

## 简介

《金刚般若波罗蜜经》，简称《金刚经》，是大乘佛教的重要经典之一，属于《大般若经》的第九会。本经由释迦牟尼佛在舍卫国祇树给孤独园为须菩提等弟子所说，全经共三十二分，主要阐述般若空性的智慧，教导众生如何破除我相、人相、众生相、寿者相等执著，证得无上正等正觉。

《金刚经》以"金刚"比喻般若智慧能断一切烦恼，以"般若"（智慧）为体，以"波罗蜜"（到彼岸）为用，是修学大乘佛法的重要经典。

### 正文（共三十二分）

**序分（第一分）**
1. **法会因由分第一** - 法会因缘，佛陀与弟子在舍卫国祇树给孤独园

**正宗分（第二分至第三十一分）**
2. **善现启请分第二** - 须菩提请问如何降伏其心、如何安住其心
3. **大乘正宗分第三** - 佛陀说明应如是降伏其心，度一切众生而无众生得度
4. **妙行无住分第四** - 菩萨应无所住而行布施
5. **如理实见分第五** - 不可以身相见如来
6. **正信希有分第六** - 能生信心者，已于无量佛所种诸善根
7. **无得无说分第七** - 无有定法如来可说，无有定法名阿耨多罗三藐三菩提
8. **依法出生分第八** - 一切诸佛及诸佛阿耨多罗三藐三菩提法，皆从此经出
9. **一相无相分第九** - 须陀洹、斯陀含、阿那含、阿罗汉四果皆不可得
10. **庄严净土分第十** - 菩萨庄严佛土者，即非庄严，是名庄严
11. **无为福胜分第十一** - 受持读诵此经，为他人说，其福胜彼
12. **尊重正教分第十二** - 当知此处，一切世间天人阿修罗，所应供养
13. **如法受持分第十三** - 是经名为《金刚般若波罗蜜》，应如是受持
14. **离相寂灭分第十四** - 离一切诸相，即名诸佛
15. **持经功德分第十五** - 受持读诵此经，其福胜彼
16. **能净业障分第十六** - 受持读诵此经，先世罪业，即为消灭
17. **究竟无我分第十七** - 实无有法，发阿耨多罗三藐三菩提心者
18. **一体同观分第十八** - 如来有肉眼、天眼、慧眼、法眼、佛眼
19. **法界通化分第十九** - 若福德有实，如来不说得福德多
20. **离色离相分第二十** - 不可以三十二相见如来
21. **非说所说分第二十一** - 说法者，无法可说，是名说法
22. **无法可得分第二十二** - 无有少法可得，是名阿耨多罗三藐三菩提
23. **净心行善分第二十三** - 是法平等，无有高下，是名阿耨多罗三藐三菩提
24. **福智无比分第二十四** - 受持读诵此经，其福胜彼
25. **化无所化分第二十五** - 实无有众生如来度者
26. **法身非相分第二十六** - 若以色见我，以音声求我，是人行邪道，不能见如来
27. **无断无灭分第二十七** - 发阿耨多罗三藐三菩提心者，于法不说断灭相
28. **不受不贪分第二十八** - 菩萨所作福德，不应贪著
29. **威仪寂净分第二十九** - 如来者，无所从来，亦无所去，故名如来
30. **一合理相分第三十** - 一合相者，即是不可说
31. **知见不生分第三十一** - 发阿耨多罗三藐三菩提心者，于一切法，应如是知，如是见，如是信解，不生法相

**流通分（第三十二分）**
32. **应化非真分第三十二** - 一切有为法，如梦幻泡影，如露亦如电，应作如是观

## 主要教义

### 核心思想

- **般若空性**：一切法皆空，无有自性
- **无相无住**：应无所住而生其心，不执著于任何相
- **破四相**：破除我相、人相、众生相、寿者相
- **中道实相**：不落空有，不著两边

### 重要概念

- **应无所住而生其心**：心不应执著于任何事物
- **一切有为法，如梦幻泡影**：世间一切现象都是虚幻的
- **若见诸相非相，即见如来**：不执著于相，才能见真实
- **实无有法可得**：没有固定的法可以执取

## 功德利益

根据经文记载，受持读诵《金刚经》可获得以下利益：

- **消除业障**：先世罪业，即为消灭
- **增长智慧**：开启般若智慧，破除无明
- **福德无量**：其福德不可思量
- **成就佛道**：能生信心，当知是人，不于一佛二佛三四五佛而种善根
- **天人供养**：当知此处，一切世间天人阿修罗，所应供养

## 修学要点

1. **信解受持**：对经文生起信心，理解其义，并依教奉行
2. **为人演说**：不仅自己受持，还要为他人解说
3. **不著相**：修学过程中不执著于功德、福德等相
4. **观照实相**：以般若智慧观照一切法的真实相
