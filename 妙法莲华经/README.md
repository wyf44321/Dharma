# 妙法莲华经

## 简介

《妙法莲华经》，简称《法华经》，是大乘佛教的重要经典之一，被誉为"经中之王"。本经由释迦牟尼佛在灵鹫山所说，全经共二十八品，主要阐述"一佛乘"的思想，说明一切众生皆可成佛，三乘（声闻、缘觉、菩萨）最终都归于一佛乘。本经以各种譬喻和故事，生动地说明了佛法的深奥义理，是修学大乘佛法的重要经典。

《法华经》以"妙法"（指一佛乘的微妙法门）为体，以"莲华"（比喻清净无染、因果同时）为喻，故名《妙法莲华经》。

### 正文（共二十八品）

**序分（第一品）**
1. **序品第一** - 法会因缘，佛陀在灵鹫山放光现瑞，文殊菩萨说明过去诸佛也曾现此瑞相

**正宗分（第二品至第十七品）**
2. **方便品第二** - 佛陀以方便力，为众生说三乘法，实则唯有一佛乘
3. **譬喻品第三** - 以火宅喻说明三界如火宅，三乘如方便，一佛乘为究竟
4. **信解品第四** - 须菩提等四大弟子以穷子喻说明对一佛乘的理解和信受
5. **药草喻品第五** - 以药草喻说明佛法的平等性，随众生根性而应机说法
6. **授记品第六** - 佛陀为四大声闻弟子授记，说明他们将来都会成佛
7. **化城喻品第七** - 以化城喻说明二乘涅槃如化城，非究竟，一佛乘才是真实
8. **五百弟子受记品第八** - 佛陀为五百阿罗汉授记成佛
9. **授学无学人记品第九** - 佛陀为有学无学人授记成佛
10. **法师品第十** - 说明受持、读诵、解说、书写本经的功德
11. **见宝塔品第十一** - 多宝佛塔从地涌出，证明《法华经》的真实不虚
12. **提婆达多品第十二** - 提婆达多因缘，说明恶人也能成佛，以及龙女成佛的故事
13. **劝持品第十三** - 药王、大乐说等菩萨发愿在恶世中护持本经
14. **安乐行品第十四** - 文殊菩萨请问如何在恶世中安稳修行，佛陀说四种安乐行
15. **从地涌出品第十五** - 无数菩萨从地涌出，都是释迦牟尼佛的弟子
16. **如来寿量品第十六** - 佛陀说明自己成佛以来，已经过无量劫，寿命无量
17. **分别功德品第十七** - 说明听闻如来寿量品所得的功德

**流通分（第十八品至第二十八品）**
18. **随喜功德品第十八** - 说明随喜听闻本经的功德
19. **法师功德品第十九** - 说明受持本经的法师所得的六根清净功德
20. **常不轻菩萨品第二十** - 说明常不轻菩萨的故事，说明恭敬一切众生的重要性
21. **如来神力品第二十一** - 佛陀现大神力，嘱累菩萨护持本经
22. **嘱累品第二十二** - 佛陀将本经嘱累给诸菩萨，令其广为流通
23. **药王菩萨本事品第二十三** - 说明药王菩萨过去世燃身供佛的故事
24. **妙音菩萨品第二十四** - 说明妙音菩萨的神通力和功德
25. **观世音菩萨普门品第二十五** - 说明观世音菩萨的慈悲和救苦救难的能力
26. **陀罗尼品第二十六** - 诸菩萨说咒护持本经
27. **妙庄严王本事品第二十七** - 说明妙庄严王因缘，说明善知识的重要性
28. **普贤菩萨劝发品第二十八** - 普贤菩萨发愿护持本经，劝发众生受持本经

## 主要教义

### 核心思想

- **一佛乘**：一切众生皆可成佛，三乘（声闻、缘觉、菩萨）最终都归于一佛乘
- **开权显实**：以方便法门（权）显示真实法门（实）
- **会三归一**：会三乘归于一佛乘
- **佛性本具**：一切众生皆有佛性，皆可成佛

### 重要概念

- **十如是**：如是相、如是性、如是体、如是力、如是作、如是因、如是缘、如是果、如是报、如是本末究竟等
- **一乘实相**：唯一真实的法门，即一佛乘
- **方便法门**：为适应众生根性而说的权宜法门
- **成佛因缘**：一切众生皆有成佛的因缘和可能性

## 功德利益

根据经文记载，受持读诵《妙法莲华经》可获得以下利益：

- **消除罪业**：能灭一切罪业
- **增长智慧**：开启佛智，了达实相
- **福德无量**：其福德不可思量
- **成就佛道**：能于一佛乘，究竟成佛
- **诸佛护念**：十方诸佛皆护念受持本经者
- **现世安乐**：现世得安乐，来世生佛国
