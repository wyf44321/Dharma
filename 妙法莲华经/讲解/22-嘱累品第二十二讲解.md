# 嘱累品第二十二讲解

## 章节概述

本品说明佛陀将《法华经》嘱累给菩萨。佛陀三次摩菩萨顶，将《法华经》嘱累给菩萨，要他们受持、读诵、广宣此法，说明了护持《法华经》的重要性和责任。这是《法华经》中非常重要的一品，体现了佛陀对《法华经》的重视和对菩萨的信任，也说明了护持《法华经》是每个菩萨的责任。

## 主要内容讲解

### 一、佛陀现大神力

**经文要点：**
- 尔时释迦牟尼佛从法座起。现大神力
- 以右手摩无量菩萨摩诃萨顶
- 而作是言。我于无量百千万亿阿僧祇劫。修习是难得阿耨多罗三藐三菩提法

**讲解：**
1. **从法座起**：
   - 佛陀从法座上站起来
   - 说明这是非常重要的时刻
   - 体现了佛陀对嘱累的重视
   - 说明这不是普通的说法，而是重要的嘱累

2. **现大神力**：
   - 显现大神通力
   - 说明佛陀的威德和力量
   - 体现了《法华经》的殊胜
   - 说明嘱累的重要性

3. **摩菩萨顶**：
   - 用右手摩无量菩萨的顶
   - 这是佛陀对菩萨的加持和嘱累
   - 体现了佛陀的慈悲和信任
   - 说明这是重要的传承仪式

4. **无量菩萨**：
   - 无量的菩萨摩诃萨
   - 说明护持《法华经》的菩萨众多
   - 体现了《法华经》的殊胜
   - 说明护持的责任重大

### 二、第一次嘱累

**经文要点：**
- 我于无量百千万亿阿僧祇劫。修习是难得阿耨多罗三藐三菩提法
- 今以付嘱汝等
- 汝等应当一心流布此法。广令增益

**讲解：**
1. **无量劫的修习**：
   - 佛陀经过无量百千万亿阿僧祇劫的修习
   - 才得到这个难得的法
   - 说明《法华经》的珍贵
   - 体现了成佛的艰难和法的殊胜

2. **难得之法**：
   - 这是难得阿耨多罗三藐三菩提法
   - 是成佛的根本法
   - 是《法华经》的核心
   - 说明这个法的珍贵和重要

3. **付嘱的意义**：
   - 现在付嘱给菩萨
   - 说明佛陀对菩萨的信任
   - 体现了佛法的传承
   - 说明护持的责任

4. **一心流布**：
   - 要一心一意地流布此法
   - 不要三心二意
   - 要有坚定的决心
   - 说明护持的态度

5. **广令增益**：
   - 要广泛地传播，让更多人受益
   - 要让佛法增长，不要减少
   - 要让众生得到利益
   - 说明护持的目标

### 三、第二次嘱累

**经文要点：**
- 如是三摩诸菩萨摩诃萨顶
- 而作是言。我于无量百千万亿阿僧祇劫。修习是难得阿耨多罗三藐三菩提法
- 今以付嘱汝等。汝等当受持读诵。广宣此法。令一切众生。普得闻知

**讲解：**
1. **三次摩顶**：
   - 三次摩菩萨顶
   - 说明嘱累的重要性
   - 体现了佛陀的重视
   - 说明这是非常重要的嘱累

2. **重复强调**：
   - 重复说明无量劫的修习
   - 重复说明法的难得
   - 说明这是非常重要的法
   - 体现了佛陀的重视

3. **受持读诵**：
   - 要受持《法华经》
   - 要读诵《法华经》
   - 这是护持的基础
   - 说明护持的方法

4. **广宣此法**：
   - 要广泛地宣说此法
   - 要让更多人知道
   - 要让众生受益
   - 说明护持的责任

5. **令一切众生普得闻知**：
   - 要让一切众生都能听到
   - 要让一切众生都能知道
   - 体现了大乘的慈悲
   - 说明护持的目标

### 四、嘱累的深意

**经文要点：**
- 所以者何。如来有大慈悲。无诸悭吝。亦无所畏
- 能与众生。佛之智慧。如来智慧。自然智慧
- 如来是一切众生之大施主

**讲解：**
1. **如来的大慈悲**：
   - 如来有大慈悲心
   - 愿意把法传给众生
   - 体现了如来的慈悲
   - 说明如来的无私

2. **无诸悭吝**：
   - 没有吝啬，愿意分享
   - 愿意把智慧传给众生
   - 体现了如来的无私
   - 说明如来的大度

3. **亦无所畏**：
   - 没有畏惧，敢于传法
   - 敢于面对困难
   - 体现了如来的勇气
   - 说明如来的无畏

4. **三种智慧**：
   - 佛之智慧：佛的智慧
   - 如来智慧：如来的智慧
   - 自然智慧：自然的智慧
   - 说明如来能给予众生智慧

5. **大施主**：
   - 如来是一切众生的大施主
   - 能给予众生一切
   - 体现了如来的功德
   - 说明如来的殊胜

### 五、菩萨应当学习

**经文要点：**
- 汝等亦应随学如来之法。勿生悭吝
- 于未来世。若有善男子。善女人。信如来智慧者。当为演说此法华经
- 使得闻知。为令其人得佛慧故

**讲解：**
1. **随学如来**：
   - 要学习如来的法
   - 要学习如来的慈悲
   - 要学习如来的无私
   - 说明菩萨应当学习

2. **勿生悭吝**：
   - 不要有吝啬心
   - 要愿意分享法
   - 要愿意帮助众生
   - 说明护持的态度

3. **为有信者说法**：
   - 对有信心的善男子、善女人
   - 要为他们演说《法华经》
   - 让他们听到和知道
   - 说明护持的对象

4. **得佛慧**：
   - 为了让众生得到佛的智慧
   - 这是护持的目标
   - 体现了大乘的慈悲
   - 说明护持的意义

### 六、对不信者的方便

**经文要点：**
- 若有众生不信受者。当于如来余深法中。示教利喜
- 汝等若能如是。则为已报诸佛之恩

**讲解：**
1. **不信受者**：
   - 如果有众生不信受《法华经》
   - 不要强迫，要用方便
   - 体现了如来的智慧
   - 说明方便法门的重要性

2. **余深法**：
   - 用其他的深法来引导
   - 根据众生的根机说法
   - 体现了如来的善巧
   - 说明方便法门的意义

3. **示教利喜**：
   - 示：开示
   - 教：教导
   - 利：利益
   - 喜：欢喜
   - 说明要用各种方法引导众生

4. **报诸佛之恩**：
   - 如果能这样做，就是报答诸佛的恩
   - 说明护持正法是报恩
   - 体现了护持的意义
   - 说明护持的责任

### 七、菩萨的承诺

**经文要点：**
- 时诸菩萨摩诃萨。闻佛作是说已。皆大欢喜。遍满其身
- 益加恭敬。曲躬低头。合掌向佛
- 俱发声言。如世尊敕。当具奉行。唯然世尊。愿不有虑
- 诸菩萨摩诃萨众。如是三反。俱发声言

**讲解：**
1. **大欢喜**：
   - 听到佛陀的嘱累，非常欢喜
   - 遍满其身：欢喜充满全身
   - 说明菩萨对嘱累的重视
   - 体现了菩萨的发心

2. **恭敬的态度**：
   - 益加恭敬：更加恭敬
   - 曲躬低头：弯腰低头
   - 合掌向佛：合掌向佛
   - 说明菩萨的恭敬和诚心

3. **三次承诺**：
   - 三次都说：如世尊敕，当具奉行
   - 唯然世尊，愿不有虑
   - 说明承诺的坚定
   - 体现了菩萨的决心

4. **当具奉行**：
   - 会完全按照佛陀的教导去做
   - 会护持《法华经》
   - 会流布正法
   - 说明菩萨的承诺

5. **愿不有虑**：
   - 希望佛陀不要担心
   - 我们会好好护持
   - 体现了菩萨的信心
   - 说明菩萨的决心

### 八、法会圆满

**经文要点：**
- 尔时释迦牟尼佛令十方来诸分身佛。各还本土
- 而作是言。诸佛各随所安
- 多宝佛塔还可如故

**讲解：**
1. **分身佛各还本土**：
   - 十方来的分身佛都回到各自的国土
   - 说明法会即将结束
   - 体现了法会的圆满
   - 说明嘱累已经完成

2. **诸佛各随所安**：
   - 诸佛各自回到安住的地方
   - 说明法会的结束
   - 体现了如来的自在
   - 说明法会的圆满

3. **多宝佛塔还可如故**：
   - 多宝佛塔回到原来的样子
   - 说明法会的结束
   - 体现了法会的圆满
   - 说明嘱累已经完成

4. **法会的意义**：
   - 这次法会非常重要
   - 说了《法华经》
   - 完成了嘱累
   - 说明了《法华经》的殊胜

   - 护持《法华经》是报答佛恩
   - 护持《法华经》是成佛的因缘

2. **《法华经》的殊胜**：
   - 是佛陀经过无量劫修习得到的难得法
   - 是成佛的根本法
   - 是《法华经》的核心
   - 说明这个法的珍贵和重要

3. **护持的责任**：
   - 要受持、读诵、广宣《法华经》
   - 要让一切众生普得闻知

   - 这是每个菩萨的责任

   - 是一切众生的大施主
   - 菩萨应当学习如来的慈悲

5. **方便法门**：
   - 对不信受者，要用方便法门

   - 体现了如来的善巧
