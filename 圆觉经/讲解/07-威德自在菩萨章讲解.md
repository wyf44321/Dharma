# 威德自在菩萨章讲解

## 章节概述

本章威德自在菩萨请问修行方便，佛陀开示三种法门：奢摩他（止）、三摩钵提（观）、禅那（禅）。说明这三种法门都是圆觉亲近随顺，十方如来因此成佛。通过深入理解三种法门的特点、修习方法和关系，我们能够选择适合自己的修行方法，证得圆觉。

## 主要内容讲解

### 一、威德自在菩萨请问

**经文要点：**
- 于是威德自在菩萨在大众中，即从座起，顶礼佛足，右绕三匝，长跪叉手而白佛言
- 大悲世尊，广为我等分别如是随顺觉性，令诸菩萨觉心光明，承佛圆音，不因修习而得善利
- 世尊，譬如大城，外有四门，随方来者，非止一路，一切菩萨庄严佛国及成菩提，非一方便
- 唯愿世尊广为我等宣说一切方便渐次，并修行人总有几种？
- 令此会菩萨及末世众生求大乘者，速得开悟，游戏如来大寂灭海
- 作是语已，五体投地，如是三请，终而复始

**讲解：**

1. **威德自在菩萨的恭敬**：
   - 即从座起：从座位上站起来，表示恭敬
   - 顶礼佛足：顶礼佛陀的脚，表示最高的恭敬
   - 右绕三匝：向右绕佛三圈，表示恭敬和供养
   - 长跪叉手：长跪合掌，表示至诚恭敬
   - 说明威德自在菩萨对佛陀的恭敬和至诚
   - 体现了修学佛法要有恭敬心，这是成就的根本

2. **威德自在菩萨的感恩**：
   - 广为我等分别如是随顺觉性：为我们广泛分别这样的随顺觉性
   - 令诸菩萨觉心光明：让各种菩萨觉心光明
   - 承佛圆音，不因修习而得善利：承受佛的圆音，不因为修习而得到善利
   - 说明威德自在菩萨对佛陀的感恩
   - 体现了佛陀教法的殊胜

3. **"譬如大城，外有四门"的比喻**：
   - **大城的含义**：
     - 大城：比喻成佛的目标，即圆觉的境界
     - 大城是众生的归宿，是修行的最终目标
     - 说明成佛是众生的共同目标，就像大城是所有人的归宿
   
   - **四门的含义**：
     - 外有四门：大城外面有四个门，比喻成佛有多种方便方法
     - 四门代表不同的修行路径，都可以进入大城
     - 说明成佛不是只有一种方法，有多种方便法门
   
   - **随方来者的含义**：
     - 随方来者：从不同方向来的人，比喻不同根性的众生
     - 非止一路：不是只有一条路，说明有多种路径可以到达
     - 不同根性的众生，可以选择不同的修行方法
     - 体现了修行方法的多样性和灵活性
   
   - **比喻的深层含义**：
     - 成佛的目标是统一的（大城），但方法可以多样（四门）
     - 不同根性的众生，可以选择适合自己的方法
     - 说明修行要因材施教，不能千篇一律
     - 体现了佛陀的善巧方便，能够适应不同众生的需要
   
   - **修行的启示**：
     - 不要执着于一种修行方法
     - 要根据自己的根性选择合适的方法
     - 不同的方法都能达到成佛的目标
     - 体现了修行方法的包容性和灵活性

4. **"一切菩萨庄严佛国及成菩提，非一方便"**：
   - 一切菩萨：所有菩萨
   - 庄严佛国：庄严佛国
   - 及成菩提：以及成就菩提
   - 非一方便：不是一种方便
   - 说明所有菩萨庄严佛国以及成就菩提，不是一种方便
   - 体现了修行方法的多样性

5. **"宣说一切方便渐次"**：
   - 宣说：宣说
   - 一切方便：所有方便
   - 渐次：渐次，次第
   - 说明要宣说所有方便的渐次
   - 这是威德自在菩萨的请求
   - 体现了了解方便渐次的重要性

6. **"并修行人总有几种？"**：
   - 并：以及
   - 修行人：修行的人
   - 总有几种：总共有几种
   - 说明以及修行的人总共有几种
   - 这是威德自在菩萨的疑问
   - 体现了了解修行人种类的重要性

7. **"速得开悟，游戏如来大寂灭海"**：
   - 速得开悟：快速得到开悟
   - 游戏：游戏，自在
   - 如来大寂灭海：如来的大寂灭海
   - 说明快速得到开悟，游戏如来的大寂灭海
   - 这是修行的目标
   - 体现了开悟和游戏的重要性

8. **"五体投地，如是三请"**：
   - 五体投地：两手、两膝、头都着地，表示最高的恭敬
   - 三请：三次请求，表示至诚和坚持
   - 说明威德自在菩萨的至诚和坚持
   - 体现了求法要有至诚心，不能轻易放弃

9. **威德自在菩萨的智慧**：
   - 威德自在菩萨是威德自在第一的菩萨
   - 他问的问题都是修行的关键问题
   - 说明要有智慧，才能问出正确的问题
   - 体现了智慧在修行中的重要性

### 二、佛陀的赞叹

**经文要点：**
- 尔时世尊告威德自在菩萨言：善哉善哉，善男子，汝等乃能为诸菩萨及末世众生，问于如来如是方便。汝今谛听，当为汝说
- 时威德自在菩萨奉教欢喜，及诸大众默然而听

**讲解：**

1. **"善哉善哉"**：
   - 这是佛陀的赞叹，表示对威德自在菩萨的认可
   - 说明威德自在菩萨问的问题很好，是修行的关键
   - 体现了佛陀的慈悲，会赞叹和鼓励弟子

2. **"问于如来如是方便"**：
   - 问于如来：请问如来
   - 如是方便：这样的方便
   - 说明请问如来这样的方便
   - 这是修行的关键问题
   - 体现了了解方便的重要性

3. **"汝今谛听，当为汝说"**：
   - 谛听：仔细听
   - 当为汝说：当为你宣说
   - 说明要仔细听，佛陀会宣说
   - 体现了听法的重要性

### 三、三种法门的总说

**经文要点：**
- 善男子，无上妙觉遍诸十方，出生如来与一切法，同体平等，于诸修行实无有二
- 方便随顺，其数无量，圆摄所归，循性差别，当有三种

**讲解：**

1. **"无上妙觉遍诸十方"**：
   - 无上妙觉：无上的妙觉
   - 遍诸十方：遍满十方
   - 说明无上的妙觉遍满十方
   - 体现了圆觉的遍满性

2. **"出生如来与一切法"**：
   - 出生：出生，产生
   - 如来：如来
   - 与一切法：以及一切法
   - 说明出生如来以及一切法
   - 圆觉是如来和一切法的根本
   - 体现了圆觉的根本性

3. **"同体平等"**：
   - 同体：同一体性
   - 平等：平等
   - 说明同一体性，平等
   - 一切法都是同体平等的
   - 体现了圆觉的平等性

4. **"于诸修行实无有二"**：
   - 于诸修行：在各种修行中
   - 实无有二：实际上没有二
   - 说明在各种修行中，实际上没有二
   - 一切修行在本质上没有差别
   - 体现了修行的统一性

5. **"方便随顺，其数无量"**：
   - 方便随顺：方便随顺
   - 其数无量：数量无量
   - 说明方便随顺，数量无量
   - 修行的方便方法有很多
   - 体现了方便方法的多样性

6. **"圆摄所归，循性差别，当有三种"**：
   - 圆摄所归：圆摄所归
   - 循性差别：随顺根性的差别
   - 当有三种：应该有三种
   - 说明圆摄所归，随顺根性的差别，应该有三种
   - 根据根性的差别，有三种主要法门
   - 体现了三种法门的分类

### 四、奢摩他（止）

**经文要点：**
- 善男子，若诸菩萨悟净圆觉，以净觉心，取静为行，由澄诸念，觉识烦动，静慧发生，身心客尘从此永灭，便能内发寂静轻安，由寂静故，十方世界诸如来心，于中显现，如镜中像
- 此方便者，名奢摩他

**讲解：**

1. **"若诸菩萨悟净圆觉"**：
   - 若诸菩萨：如果各种菩萨
   - 悟净圆觉：觉悟清净的圆觉
   - 说明如果各种菩萨觉悟清净的圆觉
   - 这是修行的基础
   - 体现了觉悟圆觉的重要性

2. **"以净觉心，取静为行"**：
   - 以净觉心：以清净的觉心
   - 取静为行：取静为修行
   - 说明以清净的觉心，取静为修行
   - 这是奢摩他的特点
   - 体现了取静为行的重要性

3. **"由澄诸念"**：
   - 由澄：因为澄清
   - 诸念：各种念头
   - 说明因为澄清各种念头
   - 澄清念头是修行的第一步
   - 体现了澄清念头的重要性

4. **"觉识烦动"**：
   - 觉识：觉识，意识
   - 烦动：烦动，躁动
   - 说明觉识烦动
   - 因为澄清念头，所以觉识烦动
   - 体现了觉识烦动的认识

5. **"静慧发生"**：
   - 静慧：静的智慧
   - 发生：发生，生起
   - 说明静的智慧发生
   - 因为澄清念头，所以静的智慧生起
   - 体现了静慧的重要性

6. **"身心客尘从此永灭"**：
   - 身心客尘：身心的客尘
   - 从此永灭：从此永远灭除
   - 说明身心的客尘从此永远灭除
   - 因为澄清念头，所以客尘灭除
   - 体现了灭除客尘的重要性

7. **"便能内发寂静轻安"**：
   - 便能：就能
   - 内发：内发，从内发出
   - 寂静轻安：寂静轻安
   - 说明就能从内发出寂静轻安
   - 这是修行的结果
   - 体现了寂静轻安的重要性

8. **"由寂静故，十方世界诸如来心，于中显现"**：
   - 由寂静故：因为寂静的缘故
   - 十方世界：十方世界
   - 诸如来心：各种如来的心
   - 于中显现：在其中显现
   - 说明因为寂静的缘故，十方世界各种如来的心在其中显现
   - 因为寂静，所以如来心显现
   - 体现了寂静的重要性

9. **"如镜中像"的比喻**：
   - **镜子的特点**：
     - 镜子本身是清净的，没有污染
     - 镜子能够清晰地映照一切事物
     - 比喻圆觉心本来清净，能够显现一切
   
   - **像的特点**：
     - 镜中的像不是真实的，只是影像
     - 像随物而现，物去像也消失
     - 比喻如来心的显现，不是实有，而是随缘显现
   
   - **比喻的含义**：
     - 镜子比喻寂静的心，像比喻如来心的显现
     - 当心达到寂静时，如来心就会显现
     - 就像镜子清净时，能够清晰地映照一切
     - 说明寂静是如来心显现的条件
   
   - **修行的启示**：
     - 要修习奢摩他，让心达到寂静
     - 心寂静了，如来心自然显现
     - 不需要刻意去求，只要心清净，自然显现
     - 体现了奢摩他修行的特点：以静为行，自然显现
   
   - **与奢摩他的关系**：
     - 奢摩他是修止，让心达到寂静
     - 心寂静了，就像镜子清净了
     - 如来心就会显现，就像镜中像一样清晰
     - 这是奢摩他修行的完整过程

10. **"此方便者，名奢摩他"**：
    - **奢摩他的含义**：
      - 奢摩他是梵语，音译为"奢摩他"，意译为"止"、"定"、"寂静"
      - "止"的意思是停止妄念，让心安定下来
      - "定"的意思是心专注于一境，不散乱
      - "寂静"的意思是内心平静，没有躁动
   
    - **奢摩他的作用**：
      - 能够停止妄念，让心安定
      - 能够澄清念头，让心清净
      - 能够生起静慧，开发智慧
      - 能够灭除客尘，消除烦恼
      - 能够证得寂静轻安，获得安乐
      - 能够显现如来心，证得圆觉
   
    - **奢摩他的特点**：
      - 以静为行：以静为修行的主要方法
      - 渐次深入：从澄清念头到显现如来心，是渐次深入的
      - 自然显现：如来心的显现是自然的，不需要刻意追求
      - 适合根性：适合喜欢静、能够安定的根性
   
    - **奢摩他在修行中的地位**：
      - 是三种法门之一，是圆觉的亲近随顺
      - 是十方如来成佛的方法之一
      - 是修行的根本方法之一
      - 体现了奢摩他的重要性和殊胜性

11. **奢摩他的特点**：
    - 以静为行：以静为修行
    - 澄清念头：澄清各种念头
    - 静慧发生：静的智慧发生
    - 客尘永灭：客尘永远灭除
    - 寂静轻安：寂静轻安
    - 如来心显现：如来心显现
    - 这是修行的完整过程

### 五、三摩钵提（观）

**经文要点：**
- 善男子，若诸菩萨悟净圆觉，以净觉心，知觉心性及与根尘，皆因幻化，即起诸幻以除幻者，变化诸幻而开幻众，由起幻故，便能内发大悲轻安，一切菩萨从此起行，渐次增进，彼观幻者非同幻故，非同幻观，皆是幻故，幻相永离，是诸菩萨所圆妙行，如土长苗
- 此方便者，名三摩钵提

**讲解：**

1. **"知觉心性及与根尘，皆因幻化"**：
   - 知觉：了知
   - 心性：心的性
   - 及与根尘：以及根和尘
   - 皆因幻化：都因为幻化
   - 说明了知心性以及根和尘，都因为幻化
   - 这是三摩钵提的基础认识
   - 体现了了知幻化的重要性

2. **"即起诸幻以除幻者"**：
   - 即起：就生起
   - 诸幻：各种幻
   - 以除幻者：来除幻的人
   - 说明就生起各种幻来除幻的人
   - 用幻来除幻
   - 体现了以幻除幻的方法

3. **"变化诸幻而开幻众"**：
   - 变化：变化
   - 诸幻：各种幻
   - 而开幻众：而开化幻的众生
   - 说明变化各种幻而开化幻的众生
   - 用幻来度化众生
   - 体现了以幻度化的方法

4. **"由起幻故，便能内发大悲轻安"**：
   - 由起幻故：因为生起幻的缘故
   - 便能：就能
   - 内发大悲轻安：从内发出大悲轻安
   - 说明因为生起幻的缘故，就能从内发出大悲轻安
   - 因为生起幻，所以大悲轻安
   - 体现了大悲轻安的重要性

5. **"一切菩萨从此起行"**：
   - 一切菩萨：所有菩萨
   - 从此起行：从这里起行
   - 说明所有菩萨从这里起行
   - 三摩钵提是菩萨起行的地方
   - 体现了三摩钵提的重要性

6. **"渐次增进"**：
   - 渐次：渐次
   - 增进：增进
   - 说明渐次增进
   - 修行要渐次增进
   - 体现了渐次增进的重要性

7. **"彼观幻者非同幻故"**：
   - 彼观幻者：那个观幻的人
   - 非同幻故：不是同幻的缘故
   - 说明那个观幻的人不是同幻的缘故
   - 观幻的人不是幻
   - 体现了观幻者的真实性

8. **"非同幻观，皆是幻故"**：
   - 非同幻观：不是同幻的观
   - 皆是幻故：都是幻的缘故
   - 说明不是同幻的观，都是幻的缘故
   - 观也是幻
   - 体现了观的虚幻性

9. **"幻相永离"**：
   - 幻相：幻的相
   - 永离：永远离开
   - 说明幻的相永远离开
   - 因为了知都是幻，所以幻相离开
   - 体现了幻相永离的重要性

10. **"是诸菩萨所圆妙行"**：
    - 是诸菩萨：这些菩萨
    - 所圆妙行：所圆满的妙行
    - 说明这些菩萨所圆满的妙行
    - 这是修行的成就
    - 体现了妙行的重要性

11. **"如土长苗"的比喻**：
    - **土地的含义**：
      - 土地是苗生长的基础，比喻圆觉心是修行的基础
      - 土地本身是清净的，能够生长万物
      - 比喻圆觉心本来清净，能够生起一切功德
   
    - **苗的含义**：
      - 苗从土地中生长出来，比喻修行从圆觉心中生起
      - 苗是渐次生长的，从种子到苗，再到成熟
      - 比喻修行是渐次增进的，不是一蹴而就的
   
    - **渐次增长的含义**：
      - 苗的生长需要时间，不能急功近利
      - 苗的生长需要条件，需要阳光、水分等
      - 比喻修行需要时间，需要各种因缘条件
      - 说明修行要循序渐进，不能急于求成
   
    - **比喻的深层含义**：
      - 三摩钵提的修行就像土地长苗一样
      - 从了知幻化开始，到以幻除幻，再到幻相永离
      - 这是一个渐次增进的过程，不是突然完成的
      - 说明三摩钵提是渐次修行的法门
   
    - **修行的启示**：
