# 贤善首菩萨章讲解

## 章节概述

本章是《圆觉经》的最后一章，贤善首菩萨请问经名及功德，佛陀开示经名及护持功德。说明此经名《大方广圆觉陀罗尼》，亦名《修多罗了义》，亦名《秘密王三昧》等。并说明受持此经的功德，以及诸金刚、天王、鬼王发愿护持。本章总结了整部经的功德和重要性，说明了此经在佛法中的地位，以及受持此经的殊胜功德和护持力量。

## 主要内容讲解

### 一、贤善首菩萨请问

**经文要点：**
- 于是贤善首菩萨在大众中，即从座起，顶礼佛足，右绕三匝，长跪叉手而白佛言：大悲世尊，广为我等及末世众生，开悟如是不思议事。
- 世尊，此大乘教，名字何等？云何奉持？众生修习得何功德？云何使我护持经人？流布此教至于何地？
- 作是语已，五体投地，如是三请，终而复始。

**讲解：**

1. **贤善首菩萨的请问**：
   - 贤善首菩萨：贤善首是"贤善之首"的意思，表示这位菩萨是贤善众生的首领，代表一切贤善的众生
   - 在大众中：贤善首菩萨在法会的大众中
   - 即从座起：立即从座位上站起来，表示对佛陀的恭敬
   - 顶礼佛足：顶礼佛陀的双足，这是最恭敬的礼节
   - 右绕三匝：向右绕佛陀三圈，表示对佛陀的恭敬和围绕
   - 长跪叉手：长跪并合掌，表示至诚恭敬
   - 说明贤善首菩萨的恭敬和至诚，体现了对佛陀和佛法的尊重

2. **请问的动机**：
   - 大悲世尊，广为我等及末世众生，开悟如是不思议事：大悲的世尊，广泛地为我们和末世的众生，开示了这样不可思议的道理
   - 说明贤善首菩萨的请问是为了利益现在和未来的众生
   - 体现了贤善首菩萨的慈悲，关心众生的利益
   - 说明这部经确实不可思议，值得请问其名字和功德

3. **请问的内容**：
   - **此大乘教，名字何等**：这部大乘教法，叫什么名字？
     - 这是第一个问题，问经名
     - 经名很重要，因为经名能够概括整部经的内容和意义
     - 知道经名，就能知道这部经的核心要义
     - 体现了贤善首菩萨想了解这部经的根本意义
   
   - **云何奉持**：如何奉持这部经？
     - 这是第二个问题，问如何奉持
     - 奉持包括读诵、理解、修行、护持等
     - 知道如何奉持，才能正确地修习这部经
     - 体现了贤善首菩萨想了解修习的方法
   
   - **众生修习得何功德**：众生修习这部经能得到什么功德？
     - 这是第三个问题，问修习的功德
     - 功德包括现世的利益和究竟的解脱
     - 知道功德，能够增强修习的信心
     - 体现了贤善首菩萨想了解修习的利益
   
   - **云何使我护持经人**：如何让我护持修习这部经的人？
     - 这是第四个问题，问如何护持
     - 护持包括保护、加持、引导等
     - 知道如何护持，才能更好地护持修习者
     - 体现了贤善首菩萨的慈悲，想护持修习者
   
   - **流布此教至于何地**：让这部教法流布到什么程度？
     - 这是第五个问题，问流布的范围
     - 流布包括传播、弘扬、护持等
     - 知道流布的范围，才能更好地弘扬这部经
     - 体现了贤善首菩萨想了解弘扬的范围

4. **三请的意义**：
   - 作是语已，五体投地，如是三请，终而复始：说完这些话后，五体投地，这样三次请问，终而复始
   - 五体投地：双手、双膝、额头都着地，这是最恭敬的礼节
   - 三请：三次请问，表示至诚恭敬
   - 终而复始：终而复始，表示持续不断地请问
   - 说明贤善首菩萨的至诚恭敬，体现了对这部经的重视
   - 体现了贤善首菩萨的决心，一定要了解这部经的名字和功德

### 二、经名

**经文要点：**
- 善男子，是经百千万亿恒河沙诸佛所说，三世如来之所守护，十方菩萨之所归依，十二部经清净眼目
- 是经名《大方广圆觉陀罗尼》，亦名《修多罗了义》，亦名《秘密王三昧》，亦名《如来决定境界》，亦名《如来藏自性差别》，汝当奉持。

**讲解：**

1. **经的殊胜地位**：
   - **百千万亿恒河沙诸佛所说**：
     - 恒河沙：恒河的沙子，比喻数量极多，无法计算
     - 百千万亿恒河沙：比喻数量极其庞大，无法想象
     - 诸佛所说：这部经是无数诸佛都宣说过的
     - 说明这部经不是只有释迦牟尼佛说过，而是无数诸佛都说过
     - 体现了这部经的普遍性和重要性
     - 说明这部经是诸佛共同宣说的，是佛法的核心
     - 就像无数老师都教过同样的课程，说明这门课程的重要性
   
   - **三世如来之所守护**：
     - 三世：过去世、现在世、未来世
     - 如来：佛的十号之一，表示如实而来，如实而去
     - 守护：保护、护持
     - 说明过去、现在、未来的所有如来都守护这部经
     - 体现了这部经的重要性，值得所有如来守护
     - 说明这部经是佛法的根本，需要守护
     - 就像珍贵的宝物，需要历代守护者来保护
   
   - **十方菩萨之所归依**：
     - 十方：东、南、西、北、东南、西南、东北、西北、上、下，表示所有方向
     - 菩萨：发菩提心，行菩萨道的人
     - 归依：依靠、皈依
     - 说明所有方向的菩萨都归依这部经
     - 体现了这部经是菩萨修行的根本依据
     - 说明这部经是菩萨道的核心，所有菩萨都要依此修行
     - 就像所有学生都要学习教科书一样，所有菩萨都要依此经修行
   
   - **十二部经清净眼目**：
     - 十二部经：佛经的十二种体裁，包括长行、重颂、孤起、因缘、本事、本生、未曾有、譬喻、论议、自说、方广、授记
     - 清净眼目：清净的眼睛，比喻能够看清真理
     - 说明这部经是十二部经的清净眼目，能够看清一切真理
     - 体现了这部经在佛法中的地位，是所有经典的眼睛
     - 说明这部经能够帮助我们看清佛法的真谛
     - 就像眼睛能够看清一切事物，这部经能够看清一切佛法

2. **经名的含义**：
   - **《大方广圆觉陀罗尼》**：
     - 大方广：大是体大，方是相大，广是用大，表示佛法的体相用都广大无边
     - 圆觉：圆满的觉性，表示本具的圆满觉性
     - 陀罗尼：总持，表示能够总持一切法
     - 说明这部经是总持一切法的圆满觉性
     - 体现了这部经的核心是圆满觉性
     - 说明这部经能够总持一切佛法，是佛法的根本
     - 就像总钥匙能够打开所有门，这部经能够总持一切法
   
   - **《修多罗了义》**：
     - 修多罗：梵语音译，指经、契经
     - 了义：了义经，表示直接说明究竟真理的经典
     - 说明这部经是了义经，直接说明究竟真理
     - 体现了这部经不是方便说，而是究竟说
     - 说明这部经是直接说明真理的，不是比喻或方便说
     - 就像直接说明答案，而不是绕弯子
   
   - **《秘密王三昧》**：
     - 秘密：深奥难懂，不是一般人能够理解的
     - 王：最尊贵的，最重要的
     - 三昧：正定，表示正确的定境
     - 说明这部经是最尊贵的秘密三昧
     - 体现了这部经的深奥和重要性
     - 说明这部经是最高深的定境，不是一般人能够理解的
     - 就像最珍贵的宝物，只有有缘人才能得到
   
   - **《如来决定境界》**：
     - 如来：佛的十号之一
     - 决定：确定的，不变的
     - 境界：所证的境界
     - 说明这部经是如来所证的确定境界
     - 体现了这部经是佛所证的境界
     - 说明这部经是佛的境界，是确定的真理
     - 就像佛的境界是确定的，这部经说明的就是这个境界
   
   - **《如来藏自性差别》**：
     - 如来藏：众生本具的佛性
     - 自性：本自具有的性质
     - 差别：不同的表现
     - 说明这部经说明如来藏的自性和差别
     - 体现了这部经说明众生本具的佛性
     - 说明这部经说明众生本具的佛性，以及如何显现
     - 就像说明宝藏在哪里，以及如何打开

3. **多个名字的意义**：
   - 这部经有五个名字，每个名字都从不同角度说明这部经的特点
   - 说明这部经的功德和意义是多方面的，无法用一个名字完全概括
   - 体现了这部经的重要性，需要用多个名字来说明
   - 说明这部经是佛法的核心，从不同角度都能看到其重要性
   - 就像一个人有多个身份，每个身份都说明他的重要性

### 三、经的功德

**经文要点：**
- 善男子，是经唯显如来境界，唯佛如来能尽宣说，若诸菩萨及末世众生，依此修行，渐次增进，至于佛地。
- 善男子，是经名为顿教大乘，顿机众生从此开悟，亦摄渐修一切群品，譬如大海，不让小流，乃至蚊虻及阿修罗，饮其水者，皆得充满。

**讲解：**

1. **如来境界**：
   - **唯显如来境界**：
     - 唯：只，仅仅
     - 显：显示，说明
     - 如来境界：佛所证的境界
     - 说明这部经只显示如来境界，不显示其他境界
     - 体现了这部经的究竟性，直接说明佛的境界
     - 说明这部经是最高深的，只说明佛的境界
     - 就像最高级的课程，只讲最高深的内容
   
   - **唯佛如来能尽宣说**：
     - 唯：只有
     - 佛如来：佛
     - 能尽宣说：能够完全宣说
     - 说明只有佛才能完全宣说这部经
     - 体现了这部经的深奥，只有佛才能完全理解
     - 说明这部经是佛的境界，只有佛才能完全说明
     - 就像只有专家才能完全理解专业知识一样

2. **修行的功德**：
   - **依此修行**：
     - 依：依靠，依据
     - 此：这部经
     - 修行：修习实践
     - 说明要依据这部经来修行
     - 体现了这部经是修行的依据
     - 说明这部经不是理论，而是实践的指导
     - 就像按照说明书来操作一样
   
   - **渐次增进**：
     - 渐次：逐渐，一步一步
     - 增进：进步，提升
     - 说明修行要逐渐进步，一步一步提升
     - 体现了修行的过程性，不是一蹴而就的
     - 说明修行要有次第，要循序渐进
     - 就像学习要循序渐进，不能跳跃
   
   - **至于佛地**：
     - 至：到达
     - 于：到
     - 佛地：佛的境界
     - 说明依此修行，最终能够到达佛的境界
     - 体现了这部经的究竟性，能够达到佛的境界
     - 说明这部经是成佛的道路，依此修行能够成佛
     - 就像按照地图走，最终能够到达目的地

3. **顿渐皆摄**：
   - **顿教大乘**：
     - 顿教：顿悟的教法，表示能够当下开悟
     - 大乘：大乘佛法
     - 说明这部经是顿悟的大乘教法
     - 体现了这部经的特点，能够当下开悟
     - 说明这部经是最高级的教法，能够当下开悟
     - 就像高级课程，能够快速理解
   
   - **顿机众生从此开悟**：
     - 顿机：顿悟根机的众生，表示能够当下开悟的众生
     - 从此开悟：从这部经开悟
     - 说明顿悟根机的众生能够从这部经开悟
     - 体现了这部经对顿机众生的作用
     - 说明这部经能够帮助顿机众生开悟
     - 就像高级学生能够快速理解高级课程
   
   - **亦摄渐修一切群品**：
     - 亦：也
     - 摄：摄受，包含
     - 渐修：渐次修行的众生
     - 一切群品：所有种类的众生
     - 说明这部经也摄受渐次修行的一切众生
     - 体现了这部经的包容性，不仅摄受顿机，也摄受渐机
     - 说明这部经适合所有根机的众生
     - 就像高级课程也适合初学者，只是理解程度不同
   
   - **譬如大海，不让小流**：
     - 譬如：比如，比喻
     - 大海：比喻这部经
     - 不让：不拒绝，不排斥
     - 小流：小河流，比喻渐修的众生
     - 说明这部经就像大海一样，不拒绝小河流
     - 体现了这部经的包容性，不排斥任何众生
     - 说明这部经能够容纳所有众生，不管根机如何
     - 就像大海能够容纳所有河流，不管大小
   
   - **乃至蚊虻及阿修罗，饮其水者，皆得充满**：
     - 乃至：甚至
     - 蚊虻：蚊子和小虫，比喻小根机的众生
     - 阿修罗：六道之一，比喻中等根机的众生
     - 饮其水者：喝海水的人，比喻修习这部经的众生
     - 皆得充满：都能得到满足
     - 说明即使是小根机的众生，修习这部经也能得到满足
     - 体现了这部经的普遍性，所有众生都能受益
     - 说明这部经适合所有根机的众生，都能得到利益
     - 就像大海的水，所有众生都能喝，都能得到满足

### 四、受持的功德

**经文要点：**
- 善男子，假使有人纯以七宝积满三千大千世界以用布施，不如有人闻此经名及一句义。
- 善男子，假使有人教百千恒河沙众生得阿罗汉果，不如有人宣说此经，分别半偈。
- 善男子，若复有人闻此经名，信心不惑，当知是人，非于一佛二佛种诸福德，如是乃至尽恒河沙一切佛所，种诸善根，闻此经教。

**讲解：**

1. **受持的殊胜功德**：
   - **七宝布施的比喻**：
     - 假使：假如，假设
     - 纯以七宝：纯粹用七种宝物
     - 七宝：金、银、琉璃、玻璃、砗磲、赤珠、玛瑙等七种宝物
     - 积满三千大千世界：堆积满三千大千世界
     - 三千大千世界：一个佛的教化范围，包括小千世界、中千世界、大千世界
     - 以用布施：用来布施
     - 说明用七宝堆积满三千大千世界来布施，功德非常大
     - 体现了布施的功德很大，但不如受持此经
     - 说明物质布施的功德虽然大，但不如法布施
     - 就像物质财富虽然重要，但不如精神财富
   
   - **不如闻此经名及一句义**：
     - 不如：比不上，不如
     - 闻此经名：听到这部经的名字
     - 一句义：一句经文的意义
     - 说明听到这部经的名字或理解一句经文的意义，功德就超过用七宝布施
     - 体现了受持此经的功德极其殊胜
     - 说明法布施的功德超过财布施
     - 就像精神财富比物质财富更珍贵
   
   - **为什么功德如此殊胜**：
     - 因为这部经是佛的境界，是成佛的道路
     - 受持此经能够开发本具的佛性，这是最根本的功德
     - 物质布施只能得到福报，而受持此经能够得到智慧和解脱
     - 体现了这部经的究竟性，能够达到究竟的解脱
     - 说明受持此经的功德是究竟的，不是暂时的

2. **宣说的殊胜功德**：
   - **度众生成阿罗汉的比喻**：
     - 假使有人教百千恒河沙众生得阿罗汉果：假如有人教导百千恒河沙的众生得到阿罗汉果
     - 百千恒河沙：数量极其庞大，无法计算
     - 阿罗汉果：声闻乘的最高果位，已经断除烦恼，不再轮回
     - 说明度众生成阿罗汉的功德非常大
     - 体现了度众生的功德很大，但不如宣说此经
     - 说明度众生成阿罗汉虽然功德大，但不如宣说此经
     - 就像帮助别人解决小问题，不如帮助别人解决根本问题
   
   - **不如宣说此经，分别半偈**：
     - 不如：比不上，不如
     - 宣说此经：宣说这部经
     - 分别半偈：解释半句偈颂
     - 说明宣说这部经，甚至只是解释半句偈颂，功德就超过度众生成阿罗汉
     - 体现了宣说此经的功德极其殊胜
     - 说明宣说此经的功德是究竟的，能够帮助众生成佛
     - 就像帮助别人解决根本问题，比解决小问题更重要
   
   - **为什么功德如此殊胜**：
     - 因为这部经是成佛的道路，宣说此经能够帮助众生成佛
     - 阿罗汉只是小乘的果位，而成佛是大乘的究竟果位
     - 宣说此经能够帮助众生开发本具的佛性，这是最根本的功德
     - 体现了这部经的究竟性，能够达到究竟的解脱
     - 说明宣说此经的功德是究竟的，能够帮助众生成佛

3. **闻经的善根**：
   - **闻此经名，信心不惑**：
