# 大方广圆觉修多罗了义经

## 简介

《大方广圆觉修多罗了义经》，简称《圆觉经》，是大乘佛教的重要经典之一。本经由释迦牟尼佛在神通大光明藏三昧中，为文殊师利、普贤、普眼等十二位大菩萨所说，全经共十二章，主要阐述圆觉妙心的清净本性，以及如何修习圆觉法门，破除无明，证得圆觉。

《圆觉经》以"圆觉"为核心，说明一切众生本来具有圆满的觉性，因无明而流转生死，若能依教修行，破除四相（我相、人相、众生相、寿命相），即可证得圆觉，成就佛道。

## 章节说明

### 开经偈
- **00-香赞开经偈.md** - 香赞与开经偈

### 正文（共十二章）

1. **文殊师利菩萨章** - 文殊师利请问如来因地法行，佛陀说明圆觉自性，无明如空华，知是空华即无轮转

2. **普贤菩萨章** - 普贤请问如何修行，佛陀说明知幻即离，不作方便，离幻即觉，亦无渐次

3. **普眼菩萨章** - 普眼请问修行渐次，佛陀说明观身如幻，四大分解，幻相永离，十方清净

4. **金刚藏菩萨章** - 金刚藏请问众生本来成佛为何还有无明，佛陀说明未出轮回而辨圆觉，彼圆觉性即同流转

5. **弥勒菩萨章** - 弥勒请问如何断轮回根本，佛陀说明爱为轮回根本，应断贪欲及除爱渴，说明五性差别

6. **清净慧菩萨章** - 清净慧请问所证所得云何差别，佛陀说明凡夫、菩萨、如来随顺觉性的不同层次

7. **威德自在菩萨章** - 威德自在请问修行方便，佛陀说明三种法门：奢摩他（止）、三摩钵提（观）、禅那（禅）

8. **辩音菩萨章** - 辩音请问有几种修习，佛陀说明二十五种清净定轮，三种法门的各种组合修习方法

9. **净诸业障菩萨章** - 净诸业障请问为何众生迷闷不入，佛陀说明四相（我相、人相、众生相、寿命相）的障碍

10. **普觉菩萨章** - 普觉请问求何等人、依何等法，佛陀说明如何求善知识，远离四病（作、任、止、灭）

11. **圆觉菩萨章** - 圆觉请问如何安居修习，佛陀说明三期安居的方法，三种净观的初首方便

12. **贤善首菩萨章** - 贤善首请问经名及功德，佛陀说明经名及护持功德，诸金刚、天王、鬼王发愿护持

## 主要教义

### 核心思想

- **圆觉妙心**：一切众生本来具有圆满的觉性
- **无明如幻**：无明如空华，非实有体，知是空华即无轮转
- **破四相**：破除我相、人相、众生相、寿命相
- **三种法门**：奢摩他（止）、三摩钵提（观）、禅那（禅）

### 重要概念

- **圆觉**：圆满的觉性，即佛性
- **无明**：根本无明，如空华，非实有体
- **四相**：我相、人相、众生相、寿命相，是修行的障碍
- **二十五轮**：三种法门的各种组合，共二十五种修习方法
- **四病**：作病、任病、止病、灭病，应远离

## 功德利益

根据经文记载，受持读诵《圆觉经》可获得以下利益：

- **成就正觉**：依此修行，渐次增进，至于佛地
- **破除无明**：了知无明如空华，即无轮转
- **证得圆觉**：破除四相，证得圆觉妙心
- **诸佛护念**：十方诸佛之所守护
- **金刚护持**：诸金刚、天王、鬼王发愿护持

## 修学要点

1. **信解受持**：对经文生起信心，理解其义，并依教奉行
2. **破除四相**：远离我相、人相、众生相、寿命相
3. **远离四病**：不著作、任、止、灭四病
4. **三种法门**：依根性选择或组合修习奢摩他、三摩钵提、禅那
5. **求善知识**：求正知见人，依善知识修行
